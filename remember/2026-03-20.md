- Items bloques >7 jours flagges [BLOQUE] par le daily-plan

## Architecture du systeme

```
vault/founder/
  daily/           <- Plans du jour (1 fichier/jour)
    2026-03-20.md
  todo.md          <- TODO persistant (toujours le meme fichier)
  journal/         <- Journal entries (existant)

Skills:
  /daily-plan  <- Genere le plan chaque matin (auto au demarrage)
  /todo        <- Ajouter/lister/completer des taches a tout moment
```

Le daily-plan lit le todo.md. Le cycle :
1. Matin : `/daily-plan` genere le plan avec tous les todos ouverts
2. Journee : `/todo add ...` quand un truc arrive
3. Soir : `/todo done ...` pour marquer les accomplissements
4. Lendemain matin : les items non faits deviennent carryover
