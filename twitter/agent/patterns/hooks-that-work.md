# Hooks qui marchent — Catalogue

## Hook type: Prediction audacieuse
- "Calling it now, [prediction]" — @arafatkatze (427 likes, 606 bookmarks)
- Variantes : "Mark my words", "In 6 months..."
- Engagement : tres fort, surtout si lie a un produit concret
- Pour Paul : "Calling it now, [prediction sur agents/Claude/M&A tech]"

## Hook type: Builder flex (avec chiffres)
- "I BUILT X THAT DOES Y" — @zostaff (168 likes, 209 bookmarks)
- "i built a dashboard for my claude code sessions: 254 sessions across 58 projects" — @poetengineer__
- Cle : CHIFFRES SPECIFIQUES, pas de vague
- Pour Paul : "mon pipeline scrape 3900 procedures judiciaires par jour" — deja dans les drafts

## Hook type: Breaking news energy (CAPS)
- "CLAUDE INSTANCES CAN NOW TALK TO EACH OTHER" — @RoundtableSpace (406 likes, 80K views)
- "BROWSER USE CLI 2.0 IS LIVE NOW" — @RoundtableSpace (118 likes, 51K views)
- Cle : etre premier sur l'info, caps premiere ligne, explication ensuite
- Pour Paul : utiliser quand une news Claude/AI sort. Etre rapide.

## Hook type: Observation + implication dark
- "Claude Cowork is blowing my mind. But..." — @buccocapital (1.1K likes, 91K views)
- Structure : excitation → "but" → consequence inattendue
- Tres fort en replies (94 replies = discussion)
- Pour Paul : "Claude Code vient de [feature]. Le truc c'est que ca veut dire que [implication]"

## Hook type: Humor/relatable
- "Si vous voulez savoir ce que ca fait de perdre votre copine, prenez Claude max" — @0xmaxou (42 likes)
- Engagement facile, fonctionne meme avec peu de followers
- Pour Paul : humor sec sur les sessions marathon, le code qui crash a 3h du mat

## Hook type: Hot take direct
- Affirmation sans hedge, pas de "I think", pas de "unpopular opinion"
- Juste : "[statement]" — point final
- Pour Paul : "Claude > GPT pour le code et c'est meme plus un debat" — deja dans les drafts

## Anti-patterns (hooks qui ne marchent PAS)
- "Here are my thoughts on..." — trop AI, trop LinkedIn
- "Thread:" en premiere ligne — engagement en baisse sur les threads
- "I've been thinking about..." — generic
- Listes numerotees sans hook — scroll past
