# Formats Ranking — Performance observee

## Tier S — Fort engagement
1. **Quote tweet + hot take** — Le format dominant. Exemples : @arafatkatze (427 likes), @Frenchiee. Bas effort, fort ROI. Cle : avoir un take original, pas juste "wow cool".
2. **Single tweet opinion forte** — @buccocapital (1.1K likes). Pas de media, juste du texte. Marche quand le take est unique et provoque la discussion.
3. **Builder demo + video/screenshot** — @swar_ja (1.7K likes), @noahzweben (88 likes). Le media natif boost le reach. Les screenshots de terminal/code sont du contenu saveable.

## Tier A — Bon engagement
4. **Breaking news react (first)** — @RoundtableSpace (406 likes). CAPS hook + explication. Cle : etre PREMIER.
5. **Builder log avec chiffres** — @poetengineer__ (254 sessions, 58 projets). Les chiffres specifiques = credibilite + engagement.
6. **One-liner humor** — @0xmaxou (42 likes pour un petit compte). Ratio engagement/followers tres bon.

## Tier B — Correct
7. **Thread technique** — Pas de thread viral observe le 26/03. Le format semble en baisse. Reserve pour du contenu vraiment dense.
8. **News relay simple** — Juste RT ou QT sans take. Faible engagement sauf si tu es premier.

## Tier C — A eviter
9. **Listicle** ("10 tools for X") — Pas vu dans le feed premium. Sature, generique.
10. **Thread recycle** — Contenu reformule. Zero credibilite dans cette niche.

## Recommandation mix pour Paul
- 40% QT + hot take (facile, fort ROI, reagir vite aux news)
- 25% single tweet opinion (ses convictions, pas besoin de news hook)
- 20% builder log avec chiffres (son pipeline, ses metriques, ses sessions)
- 10% replies strategiques (sous les tweets a fort reach)
- 5% threads (1/semaine max, quand il a un vrai sujet dense)
