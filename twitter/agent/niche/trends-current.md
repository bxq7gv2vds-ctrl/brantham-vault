# Tendances en cours — Mars 2026

## Mega-tendance : "Agentic everything"
Le mot "agent" est partout. Mais le marche se segmente :
- **Multi-agent orchestration** : Cline Kanban, claude-peers, Agent Teams (Opus). Le sujet chaud du moment. Pas juste "un agent", mais des agents qui collaborent.
- **Browser agents** : Browser Use CLI 2.0, dev-browser CLI (@sawyerhood), Browserbase /cookie-sync. Les agents qui controlent le navigateur = use case concret.
- **Autonomous companies** : NanoCorp (YC) — une company pilotee entierement par des agents AI. Debut du concept "AI-native business".
- **Claude Code comme OS** : claude-peers, /loop, auto-fix cloud, voice mode. Claude Code evolue d'un outil CLI vers un "systeme d'exploitation pour le travail".

## Produits/features chauds cette semaine
1. **Claude Code auto-fix cloud** — trending 287+ posts. Les PRs se corrigent toutes seules.
2. **Claude-peers** — instances Claude qui communiquent entre elles. Game changer pour les workflows multi-agents.

## Debats en cours
- **Claude vs GPT pour le code** : GPT-5.4 vs Opus 4.6. Benchmarks convergent mais devs restent team Claude pour le multi-file. Bon terrain pour Paul.
