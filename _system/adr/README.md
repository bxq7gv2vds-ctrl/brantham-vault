# Architecture Decision Records

ADRs system-wide qui croisent plusieurs projets ou définissent des choix transverses (stack, conventions, processus).

ADRs spécifiques à un projet vont dans `vault/<projet>/decisions/` (Brantham) ou équivalent.

## Convention

- Nom de fichier : `<NNNN>-<slug>.md` (ex : `0001-vault-unifie.md`)
- Numérotation séquentielle, jamais réutilisée même après suppression
- Frontmatter standard avec `status: proposed | accepted | superseded | deprecated`
- Section `## Related` avec wikilinks vers MOC + ADRs liées
