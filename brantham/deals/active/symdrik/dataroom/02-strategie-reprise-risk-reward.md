# Strategie de reprise SYMDRIK - maximiser le risk/reward

Date : 29/05/2026  
Dossier : SYMDRIK - redressement judiciaire - depot offres 12/06/2026 a 12h00  
Objectif : definir une idee de reprise interessante, avec risque limite et upside reel.

## 1. Position de depart

Je ne traiterais pas SYMDRIK comme une acquisition de societe. Je la traiterais comme une reprise opportuniste d'actifs en plan de cession.

La difference est fondamentale :

- on ne reprend pas la dette historique ;
- on choisit les actifs, salaries et contrats utiles ;
- on laisse les couts inutiles dans la procedure ;
- on force une restructuration que la societe n'a pas reussi a faire seule.

La logique gagnante n'est donc pas : "j'achete une entreprise".

La logique gagnante est : "j'achete a bas prix une equipe technique, un portefeuille clients residuel et une continuite de production, puis je les integre dans une structure plus solide".

## 2. Mon verdict strategique

Je ferais une offre uniquement si je peux securiser avant depot :

1. au moins 600-800 k EUR de CA client realiste sur 12 mois ;
2. 3 a 5 salaries vraiment indispensables ;
3. un accord de continuite avec Symdrik Tunisie ou un plan de remplacement ;
4. les acces techniques : code, domaines, Git, hebergement, emails, outils ;
5. la non-reprise du bail si le bail n'est pas indispensable.

Si ces points ne sont pas securises, je ferais soit une offre symbolique, soit pas d'offre.

## 3. Le bon angle : acheter le noyau, pas la structure

SYMDRIK a trois types d'actifs :

| Type d'actif | Valeur reelle | Lecture |
|---|---:|---|
| Actifs corporels | 3 680 EUR | Ne justifie presque aucun prix |
| Equipe technique | Potentiellement forte | Valeur si les clients suivent |
| Clients / contrats | Non documente | C'est le vrai sujet |
| Symdrik Tunisie | Critique | Production offshore / marge |
| Bail / frais fixes | Risque | A eviter sauf necessite |

Je chercherais a reprendre :

- le portefeuille clients et les contrats actifs ;
- le code source, domaines, repositories, documentations, droits IP ;
- 3 a 5 salaries techniques ;
- eventuellement le responsable administratif pour transition courte, mais pas comme obligation ;
- les contrats strictement indispensables a la continuite : hebergement, domaines, assurance, telecom si utile ;
- l'accord de production avec Symdrik Tunisie, mais renegocie.

Je chercherais a ne pas reprendre :

- le bail s'il n'est pas indispensable ;
- les contrats accessoires ;
- les couts de logement de fonction ;
- les outils remplaçables ;
- les dettes, naturellement ;
- les contrats dont l'objet n'est pas clair.

## 4. Pourquoi le dossier peut devenir interessant

Le dossier est mauvais comme entreprise autonome, mais peut etre bon comme bolt-on.

Il devient interessant si le repreneur a deja :

- une structure administrative ;
- une direction commerciale ;
- un DAF / back-office ;
- une capacite de delivery tech ;
- des clients a qui vendre du Drupal / dev / maintenance ;
- une capacite a absorber les salaries sans reconstruire toute une societe.

Dans ce cas, on peut supprimer une grande partie des couts fixes et ne garder que ce qui produit du CA.

## 5. Le probleme economique actuel

En 2025 :

- CA : 1,754 M EUR ;
- salaires + charges sociales : 1,325 M EUR ;
- charges externes : 700 k EUR ;
- resultat d'exploitation : -373 k EUR.

La societe ne marche plus car elle porte trop de couts pour son niveau de CA.

Le repreneur doit donc faire l'inverse :

- reprendre moins de salaries ;
- supprimer le bail si possible ;
- renegocier l'offshore ;
- ne garder que les clients rentables ;
- absorber l'administratif dans sa structure existante.

## 6. Scenario de reprise que je viserais

### Scenario recommande : "noyau technique + clients confirmes"

Perimetre :

- 4 salaries techniques repris ;
- pas de bail, ou bail repris uniquement si necessaire et renégociable ;
- accord Symdrik Tunisie limite a certains profils / projets ;
- reprise des actifs incorporels : clientele, contrats, code, domaines, repositories, documentation ;
- reprise des contrats hebergement / outils uniquement si indispensables.

Prix :

- offre cible : 25-50 k EUR ;
- plafond sans contrats clients solides : 30 k EUR ;
- plafond avec 800 k EUR+ de CA confirme : 80 k EUR ;
- au-dela de 80 k EUR, il faut une preuve forte de recurrence et de marge.

Pourquoi cette fourchette :

- actifs corporels = 3,68 k EUR ;
- societe deficitaire ;
- capitaux propres negatifs ;
- CA en chute ;
- pas de contrats clients dans la data room ;
- dependance Tunisie ;
- le tribunal valorisera aussi les emplois sauves, pas uniquement le prix.

Conclusion economique : le deal est asymetrique uniquement si le prix est bas et si l'on reprend un perimetre tres discipline.

## 8. Ce que je ferais avant depot de l'offre

Je ne deposerais pas d'offre sur la seule data room actuelle. Je ferais une mini-due diligence de combat en 5 jours.

### Jour 1 - Clients

Demander a l'AJ et au dirigeant :

- CA 2024, 2025, 2026 par client ;
- top 20 clients ;
- contrats / bons de commande ;
- factures 2026 ;
- encaissements 2026 ;
- clients partis et raisons du depart ;
- clients en litige.

Objectif : savoir s'il reste vraiment un portefeuille.

### Jour 2 - Equipe

Faire un call avec les salaries cles, si possible via AJ / dirigeant :

- qui connait les clients ?
- qui connait le code ?
- qui gere la production ?
- qui veut rester ?
- qui est indispensable ?
- qui est remplaçable ?

Objectif : ne pas reprendre par reflexe, mais reprendre ceux qui creent la valeur.

### Jour 3 - Tunisie

Obtenir un accord de principe ecrit de Symdrik Tunisie :

- ressources disponibles ;
- prix ;
- duree ;
- preavis ;
- transfert de connaissance ;
- propriete intellectuelle ;
- absence de captation clients ;
- engagement de continuite post-cession.

Objectif : eviter d'acheter une coquille vide.

### Jour 4 - Technique

Demander :

- liste des repositories ;
- acces Git ;
- noms de domaine ;
- hebergement ;
- bases de donnees ;
- outils de deploiement ;
- documentation ;
- droits IP ;
- licences tierces.

Objectif : etre certain que le repreneur peut operer jour 1.

### Jour 5 - Offre

Construire une offre courte, lisible, ferme :

- prix bas mais credible ;
- emplois sauves ;
- financement disponible ;
- perimetre clair ;
- contrats repris limites ;
- date d'entree en jouissance ;
- plan de continuation client ;
- plan social indirect : plus d'emplois sauves que liquidation.

## 9. Structure d'offre que je recommanderais

### Prix

Je mettrais un prix fixe faible :

- 30 k EUR si clients non prouves ;
- 50 k EUR si 600-800 k EUR de CA defendable ;
- 80 k EUR maximum si portefeuille solide et Symdrik Tunisie securisee.

Je ventilerais le prix ainsi :

- actifs corporels : 3 680 EUR ou arrondi 4 k EUR ;
- actifs incorporels / clientele / code / nom commercial : solde ;
- stock : zero sauf preuve contraire.

### Emploi

Je proposerais de reprendre 4 a 5 postes :

- lead developpeur Drupal ;
- developpeur back-end ;
- developpeur front-end senior ;
- expert systemes et reseaux ;
- informaticien expert si role client / architecture critique.

Le responsable administratif serait repris seulement si :

- les clients / factures / procedures internes sont difficiles a reprendre sans lui ;
- ou si cela ameliore fortement le dossier devant le tribunal pour un cout acceptable.

Sinon, je prevoirais plutot une prestation de transition courte, si juridiquement possible.

### Contrats repris

Reprendre uniquement :

- hebergement / domaines / outils indispensables ;
- assurance si necessaire ;
- SwissLife seulement pour les salaries repris ;
- Symdrik Tunisie si accord renegocie ;
- bail uniquement si indispensable.

Ne pas reprendre par defaut :

- contrats accessoires ;
- logement de fonction ;
- outils remplaçables ;
- contrats sans objet clair.

## 10. Comment maximiser les chances devant le tribunal

Le tribunal ne choisit pas toujours le prix le plus eleve. Il regarde aussi la sauvegarde de l'emploi, la credibilite du projet, la capacite de financement et la perennite.

Je presenterais donc l'offre comme :

- une reprise industrielle, pas speculative ;
- une reprise qui sauve le noyau technique ;
- un projet finance sans dette bancaire conditionnelle ;
- une integration dans une structure existante ;
- une reduction du risque d'execution car le repreneur connait le metier ;
- une continuite client immediate.

Le message : "Nous ne payons pas cher parce que l'activite est deficitaire et mal documentee, mais nous sommes les mieux places pour sauver la valeur qui reste."

## 11. Le vrai risk/reward

### Risque limite

Le risque est limite si :

- prix < 50 k EUR ;
- pas de bail lourd ;
- 4 salaries max au depart ;
- accord Tunisie signe ;
- clients confirmes ;
- BFR provisionne ;
- acces techniques garantis.

### Upside

L'upside existe si :

- 700 k EUR a 1 M EUR de CA est conserve ;
- l'equipe est integree dans une structure existante ;
- les clients peuvent etre upselles ;
- la marge est amelioree par reduction des frais fixes ;
- les ressources Tunisie sont mieux pilotees.

Le meilleur cas n'est pas de "redresser SYMDRIK" comme entite separee. Le meilleur cas est d'absorber le bon morceau dans une machine commerciale et operationnelle plus forte.

## 12. Conditions de no-go

Je n'irais pas plus loin si :

- aucun contrat client n'est fourni ;
- le top client represente une part massive du CA et refuse le transfert ;
- Symdrik Tunisie refuse de continuer ;
- le code ou les acces ne sont pas recuperables ;
- les salaries cles ne veulent pas rester ;
- le bail est impose et couteux ;
- le CA 2026 reel est inferieur a 400-500 k EUR annualise ;
- il existe des litiges clients majeurs non divulgues.

## 13. Offre type que je ferais

### Version prudente

- Prix : 30 k EUR.
- Salaries repris : 4 techniques.
- Contrats repris : hebergement / outils indispensables / assurance / Symdrik Tunisie sous accord.
- Bail : non repris, sauf occupation transitoire courte.
- Engagement : maintien emploi 12 mois selon perimetre repris.
- Investissement post-reprise : 100-150 k EUR de BFR / relance commerciale.

### Version competitive

- Prix : 50-60 k EUR.
- Salaries repris : 5, voire 6 si le responsable administratif est vraiment utile.
- Contrats repris : strictement necessaires.
- Bail : seulement si renegocie ou indispensable.
- Engagement : maintien emploi + integration dans structure existante.
- Argument tribunal : meilleure sauvegarde des emplois et continuite client.

### Version haute, seulement si data complementaire convaincante

- Prix : 80 k EUR.
- Condition mentale, pas juridique : il faut avoir vu les contrats clients, l'accord Tunisie et le CA 2026.
- Au-dessus : je considere que le risk/reward se degrade.

## 15. Conclusion

Ce que je ferais : une offre basse, tres disciplinee, orientee reprise du noyau technique et des clients prouves.

Je ne paierais pas pour l'histoire 2022. Je paierais uniquement pour le CA 2026 defendable et la capacite a produire demain.

Le deal devient interessant si :

- prix entre 30 et 60 k EUR ;
- 4 a 5 salaries repris ;
- pas de bail lourd ;
- accord Tunisie ;
- 700 k EUR+ de CA client defendable ;
- acces code / domaines / clients garantis.

Le deal n'est pas interessant si on doit reprendre trop de couts fixes sans preuve client.

Ma recommandation : viser une offre a 30-50 k EUR, montee a 60-80 k EUR uniquement si l'AJ fournit les contrats clients, les encaissements 2026 et un accord de continuite avec Symdrik Tunisie.
