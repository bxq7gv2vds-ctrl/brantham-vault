# Plan turnaround autonome - SYMDRIK

Date : 29/05/2026  
Objectif : rendre une reprise autonome interessante en reduisant fortement les couts fixes et en concentrant l'activite sur les clients rentables.

## 1. These

Une reprise autonome peut etre interessante, mais seulement si elle est pensee comme un redemarrage lean, pas comme la continuation de SYMDRIK a l'identique.

Le principe :

- acheter tres peu cher ;
- reprendre uniquement les ressources qui produisent ou encaissent du CA ;
- sortir des couts fixes historiques ;
- ne pas reprendre le bail si possible ;
- transformer l'activite en structure remote-first / delivery-first ;
- imposer une discipline de marge par client ;
- renegocier Symdrik Tunisie ;
- garder le cash pour financer le redemarrage, pas pour payer le prix.

## 2. Probleme actuel

En 2025, SYMDRIK a environ :

| Poste | Montant annuel |
|---|---:|
| CA | 1 754 k EUR |
| Salaires | 928 k EUR |
| Charges sociales | 397 k EUR |
| Salaires + charges | 1 325 k EUR |
| Charges externes | 700 k EUR |
| Charges d'exploitation totales | 2 170 k EUR |
| Resultat exploitation | -373 k EUR |

La societe perd de l'argent car elle a une structure de couts trop lourde pour son niveau de CA.

Il faut donc reconstruire un P&L cible autour du CA reel 2026, pas autour du CA historique.

## 3. Objectif economique

Objectif minimum : revenir a l'equilibre avec un CA annuel de 700-900 k EUR.

Objectif interessant : generer 120-250 k EUR d'EBITDA avec 900 k EUR - 1,2 M EUR de CA.

Pour cela, la societe doit devenir une petite structure rentable, pas une mini-ESN avec trop de frais fixes.

## 5. Stratégie de cut de couts

### 5.1 Ne pas reprendre le bail, sauf preuve de necessite

Le bail / immobilier semble couter environ 78 k EUR de location + 10 k EUR de charges par an. Pour une equipe tech, c'est trop lourd si le CA n'est pas securise.

Action :

- demander occupation transitoire courte si necessaire ;
- passer remote-first ;
- coworking ponctuel ;
- garder le siege chez un domiciliataire ;
- economiser 70-90 k EUR/an.

### 5.2 Reprendre 3-4 salaries, pas 6 par defaut

Il faut classer les salaries en trois groupes :

| Groupe | Decision |
|---|---|
| Producteurs indispensables | Reprendre |
| Support / administratif | Externaliser ou transition courte |
| Producteurs sans clients attaches | Ne pas reprendre au depart |

La reprise ideale :

- 1 lead technique / architecte ;
- 1 back-end ;
- 1 front-end ;
- 1 sysadmin / DevOps si infra critique ;
- pas de responsable administratif en CDI sauf besoin prouve.

Impact :

- 6 salaries visibles coutent environ 543 k EUR/an ;
- 4 salaries devraient couter environ 330-360 k EUR/an ;
- economie : 180-210 k EUR/an.

### 5.3 Renegocier Symdrik Tunisie

Le contrat actuel prevoit jusqu'a 70 % des factures clients reversees a Symdrik Tunisie sur TMA / forfait. C'est trop dangereux si on veut reconstruire une marge.

Nouvelle logique :

- passer de "70 % du CA client" a "TJM / forfait par ressource" ;
- plafonner le cout par projet ;
- imposer validation prealable des temps ;
- prevoir non-sollicitation clients ;
- obtenir transfert de connaissance ;
- prevoir sortie possible sous 30-60 jours.

Objectif :

- cout Tunisie cible : 20-30 % du CA, pas 40-70 % ;
- marge brute cible par projet : minimum 40 %.

### 5.4 Couper les frais non productifs

Postes a couper ou reduire fortement :

- locaux ;
- logement de fonction ;
- voiture / LLD Volvo ;
- deplacements non facturables ;
- outils SaaS non critiques ;
- abonnements inutiles ;
- honoraires non indispensables ;
- cadeaux / dons / receptions ;
- contrats telecom redondants.

Objectif economie : 120-180 k EUR/an.

### 5.5 Discipline de marge client

Chaque client doit etre classe :

| Type client | Action |
|---|---|
| Rentable et recurrent | Garder, securiser contrat |
| Rentable mais ponctuel | Transformer en TMA |
| Non rentable | Repricing immediat |
| Mauvais payeur | Stop work ou paiement avance |
| Litigieux | Ne pas baser l'offre dessus |

Regle :

- aucun projet au forfait sans marge cible ;
- aucun travail sans acompte ou paiement mensuel ;
- aucune regie sous TJM minimum ;
- aucun client avec impayes sans plan de paiement.

## 6. Offre de reprise adaptee au turnaround

### Prix

Le prix doit rester faible pour garder le cash.

| Situation avant offre | Prix max |
|---|---:|
| Pas de contrats clients fournis | 10-20 k EUR |
| CA 2026 prouve < 500 k EUR | 20-30 k EUR |
| CA 2026 prouve 700-900 k EUR | 40-60 k EUR |
| CA 2026 prouve > 1 M EUR + Tunisie securisee | 70-80 k EUR |

Ne pas payer plus : le besoin de tresorerie post-reprise est plus important que le prix.

### Salaries repris

Offre cible :

- 3 ou 4 salaries repris ;
- option de recruter les autres apres 60-90 jours si CA confirme ;
- engagement de priorite de reembauche pour les non repris si l'activite repart.

### Contrats repris

Reprendre :

- contrats clients actifs ;
- hebergement / domaines / outils critiques ;
- assurance necessaire ;
- Symdrik Tunisie sous avenant renegocie.

Ne pas reprendre :

- bail sauf necessite absolue ;
- logement fonction ;
- contrats accessoires ;
- LLD Volvo ;
- outils non critiques.

## 7. Tresorerie a prevoir

Meme avec une offre basse, il faut du cash.

Budget prudent :

| Poste | Besoin |
|---|---:|
| Prix de cession | 30-60 k EUR |
| Salaires / charges 3 mois | 80-120 k EUR |
| Sous-traitance / Tunisie 3 mois | 40-70 k EUR |
| Outils / hebergement / assurance | 10-20 k EUR |
| Juridique / compta / transition | 20-30 k EUR |
| Buffer client / BFR | 80-150 k EUR |
| Total tresorerie utile | 260-450 k EUR |

Conclusion : le vrai cheque n'est pas le prix. Le vrai cheque est le financement du redemarrage.

## 8. Plan d'execution 90 jours

### Semaine 1

- recuperer code, domaines, emails, hebergement ;
- appeler tous les clients actifs ;
- securiser les salaries repris ;
- signer l'avenant Tunisie ;
- couper les depenses non reprises ;
- mettre en place un compte bancaire et facturation propre.

### J+15

- chaque client doit avoir un statut : actif / perdu / a renegocier / litige ;
- chaque projet doit avoir une marge calculee ;
- chaque salarie doit avoir un plan de charge ;
- tous les outils doivent etre migres ou controles.

### J+30

- envoyer nouveaux contrats / avenants clients ;
- basculer les clients en paiement mensuel ou acompte ;
- sortir des projets non rentables ;
- fixer TJM minimum ;
- lancer relance commerciale sur anciens clients.

### J+60

- arbitrer les ressources : garder / remplacer / sous-traiter ;
- renegocier les contrats faibles ;
- vendre TMA et maintenance ;
- mettre en place reporting hebdo cash + marge.

### J+90

- objectif : run-rate mensuel rentable ;
- CA mensuel cible : 70-90 k EUR ;
- marge brute cible : >40 % ;
- cash burn neutralise.

## 9. Seuils de rentabilite

Hypothese lean 4 salaries :

| Poste mensuel | Montant |
|---|---:|
| Salaires + charges | 28-32 k EUR |
| Tunisie / freelances | 15-25 k EUR |
| Outils / assurance / compta | 6-8 k EUR |
| Divers / commercial | 4-6 k EUR |
| Total couts mensuels | 53-71 k EUR |

Seuil de CA mensuel :

- break-even : environ 60-70 k EUR/mois ;
- confortable : 80-100 k EUR/mois ;
- tres interessant : 100 k EUR+/mois.

Donc avant d'acheter, il faut prouver que 60-70 k EUR/mois de CA est realiste rapidement.

## 10. Conditions pour que ce soit un bon deal autonome

Le deal est bon si :

- prix <= 50 k EUR ;
- 3-4 salaries repris ;
- pas de bail ;
- pas de voiture / logement ;
- accord Tunisie renegocie ;
- CA client confirme >= 700 k EUR/an ;
- cash disponible post-reprise >= 300 k EUR ;
- acces techniques garantis ;
- marge brute cible >= 40 %.

Le deal est mauvais si :

- reprise de 6 salaries + bail + Tunisie non renegociee ;
- CA 2026 non prouve ;
- clients non transferables ;
- code / domaines non securises ;
- prix eleve qui consomme la tresorerie.

## 12. Conclusion

Oui, une reprise autonome peut devenir interessante. Mais il faut la penser comme une restructuration brutale et immediate.

Le bon plan n'est pas :

> acheter pas cher et continuer.

Le bon plan est :

> acheter pas cher, couper 250-400 k EUR de couts annuels, ne garder que les clients rentables, securiser la Tunisie et atteindre le break-even en moins de 90 jours.

La cible a viser :

- CA conserve : 750 k EUR - 1 M EUR ;
- couts fixes reduits : 550-750 k EUR ;
- EBITDA potentiel : 150-250 k EUR ;
- prix d'achat : 30-60 k EUR ;
- cash relance : 300 k EUR environ.

Dans ces conditions, le risk/reward devient interessant.
