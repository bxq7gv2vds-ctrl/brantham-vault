# Doc 30 — Noms de domaines internet

## Nature
Note interne declarant les 5 noms de domaine detenus par INGEBIME

## Liste des domaines

| # | Domaine | Usage declare |
|---|---|---|
| 1 | **ingebime.fr** | Site corporate principal — identifiant commercial |
| 2 | **mon-diagnostic-performance-energetique.fr** | Site dedie aux prestations DPE (diagnostics performance energetique) |
| 3 | **mon-mur-porteur.fr** | Site dedie aux diagnostics / etudes structurelles de murs porteurs |
| 4 | **sodiags.fr** | Site / marque "SoDiags" — diagnostics (probable filiale ou marque commerciale separee) |
| 5 | **bimmersion.com** | **Reserve, sans site actif** — projet BIM non lance |

## Points notables bruts
- **5 domaines** = 4 .fr + 1 .com
- **ingebime.fr** : site corporate principal — actif, utilise dans les documents
- **Trois sites verticaux B2C/B2B "diagnostics"** :
  - **mon-diagnostic-performance-energetique.fr** : positionnement tres long / SEO expert sur le DPE — peut-etre un site lead-gen pour capter des clients particuliers / syndics qui cherchent un diagnostiqueur
  - **mon-mur-porteur.fr** : site specialise dans les etudes de murs porteurs — domaine connu / utilise par des BET, peut-etre hyper specialise
  - **sodiags.fr** : marque commerciale derivee pour les diagnostics (DPE, amiante, plomb, termites, etc.) — **tres probablement une marque secondaire ou ancienne filiale**
- **bimmersion.com** : nom compose **BIM + immersion** — suggestion d'un **projet VR/BIM** (immersion 3D dans les maquettes BIM) qui n'a **jamais ete lance / abandonne**
- **Signal strategique** : tentative de diversification dans le **diagnostic reglementaire particulier** (B2C via SEO/Google Ads) en parallele de l'activite BET institutionnelle classique
- **Ecart avec le metier historique** : la **MOE logements sociaux** represente 41% du CA, mais il existe un **adjoint "diagnostics"** qui merite d'etre eclairci
- **Valeur patrimoniale** : les noms de domaine sont des **actifs incorporels**, a inclure dans le perimetre de reprise si souhaites — la valeur est faible sauf pour **mon-mur-porteur.fr** (SEO) et **ingebime.fr** (marque)
- **A verifier** : statut des sites (actifs ou non), trafic SEO, revenus eventuels, couts d'hebergement (OVH, Gandi, etc.)
- **Declaration AJ** : le document est simple mais atteste de la propriete des domaines au profit du futur cessionnaire
