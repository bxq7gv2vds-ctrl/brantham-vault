# Stratégie Brantham — OPEN BEE FRANCE

## Positionnement

Brantham intervient **côté repreneur** (pas mandaté par l'AJ). On capture **un repreneur** parmi le top 20, on l'accompagne sur :
1. Sourcing du dossier (déjà fait)
2. Pré-due diligence + chiffrage offre
3. Rédaction offre formelle (art. L. 642-2)
4. Dépôt au tribunal + suivi audience
5. Post-cession : accompagnement intégration premiers 90 jours (optionnel)

## Pourquoi ce deal pour Brantham

- **Fit thèse** : RJ M&A PME France ✅
- **Asset stratégique** (agrément PA) qui crée une concurrence multi-stratèges → maximise les chances de matcher avec un repreneur
- **Volume effectif gérable** : 60 salariés, CA 9 M€, valorisation 0,7-2,5 M€ → typique deal mid-market sous-radar des banques d'affaires
- **Visibilité industrie** : un succès ici positionne Brantham comme acteur tech distressed crédible (1er deal sectoriel logiciel)

## Risques pour Brantham

1. **Doxsa peut porter une offre interne** (Vincent Lemaire, asset deal) → notre repreneur peut être disqualifié → 0 fee
2. **Banques d'affaires concurrentes** (Lazard mid, ODDO BHF, Cambon, Clipperton, Crédit Mutuel Arkéa) sont sur ce type de dossier
3. **Timing** : 3 semaines = ultra-court pour convaincre, dilig'er, déposer. Magic Form 21/05 occupe une partie de l'équipe.
4. **Repreneur cold** : un industriel a 30-90 jours de cycle décisionnel, pas 3 semaines. Donc seuls ceux ayant déjà identifié Open Bee (Konica, Yooz) ont une vraie chance.

## Angle de différenciation Brantham

- **Vitesse + qualité** : pack J+1 prêt à dégainer (teaser, mémo deal, courrier AJ)
- **Connaissance procédure** : on parle le langage AJ/MJ, pas un consultant compta
- **Modèle de fees plat** vs success-only des banques → moins cher pour un small/mid deal
- **Approche industrielle** : Konica/Xerox/Toshiba ont besoin de quelqu'un qui appelle leur DG France direct, pas un slide deck froid

## Fee structure proposée

Vu CA cible 9 M€ et valeur stratégique :

**Option A — Forfait haut bas de tranche** : 15 k€ engagement + 15 k€ closing = **30 k€ HT**
**Option B — Success fee** : 25 k€ engagement + 2 % du prix d'offre (cap 80 k€) = potentiellement **40-65 k€ HT**

Recommandation : option A pour les industriels (Konica, Xerox), option B pour fonds + Tessi/HLD.
