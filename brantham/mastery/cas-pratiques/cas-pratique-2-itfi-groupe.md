# Cas pratique #2 — Groupe ITFI / ITFACTO / IT News Info

> Cas pratique sur le deal réel Brantham audité 2026-05-05. À drill avant les calls repreneurs IT/services numériques.

## Contexte
- **Cible** : Groupe ITFI / ITFACTO / IT News Info (services IT, presse spécialisée)
- **AJ** : 2M & Associés (Me Martinez)
- **CA** : 11,9 M€
- **EBITDA** : -392 k€ (négatif)
- **ETP** : 116
- **Procédure** : RJ avec plan de cession
- **DLDO** (Date Limite Dépôt Offres) : 21/05/2026

## Exercice 1 — Valorisation indicative

### Hypothèses
- CA : 11,9 M€
- EBITDA reporté : -392 k€
- Multiple sectoriel "Services IT / médias spécialisés" : 5-7x
- Décote distressed : -40 %
- Décote concentration (groupe diversifié IT + médias) : -20 %
- Décote taille moyenne (CA 10-15 M€) : -10 %

### Étape 1 : EBITDA normalisé estimé

Hypothèses retraitement (avant DD, prudent) :
- Frais procédure ré-intégrés : +200 k€
- Charges non récurrentes (litiges, sinistres) : +150 k€
- Sur-coûts groupe (sièges multiples) : +200 k€
- Réorganisation à coût intégré : +100 k€

```
EBITDA reporté          -392 k€
+ Retraitements          +650 k€
─────                   ──────
EBITDA normalisé          258 k€
```

### Étape 2 : Multiple ajusté
```
Multiple sectoriel : 5,5x médian
× (1 - 0,40)  distressed
× (1 - 0,20)  concentration
× (1 - 0,10)  taille
─────
Multiple ajusté = 5,5 × 0,60 × 0,80 × 0,90 = 2,38x
```

### Étape 3 : Valorisation
```
EV multiples = 258 × 2,38 = ~614 k€
Patrimoniale (estimée actifs réévalués) : 800-1200 k€
DCF (potentiel groupe optimisé) : 1000-1500 k€
─────
RANGE COMPOSITE : 600-1200 k€
PRIX OFFRE CIBLE : 800-1000 k€
```

→ Ratio prix/CA = ~7-8 % du CA → cohérent distressed multi-activités

## Exercice 2 — Stratégie périmètre

### Question : reprendre tout, ou cession partielle ?

**Option A : Plan de cession TOTAL**
- 116 ETP repris (ou périmètre optimisé, ex 90/116 = 78 %)
- Synergies inter-activités IT + médias
- Prix offert plus élevé pour gagner emploi

**Option B : Plan de cession PARTIEL**
- Reprendre uniquement la **branche IT** (ITFACTO + ITFI)
- Laisser la branche médias (IT News Info) au tribunal pour autre repreneur ou liquidation
- Simplifier le projet
- Prix moins élevé mais plus crédible

### Analyse
- Les deux **branches autonomes** existent (services IT vs presse spécialisée)
- Le tribunal peut **arrêter plusieurs plans** (L642-5 al. 1)
- L'option B permet à un repreneur **spécialisé** de gagner sa branche

→ Stratégie selon profil repreneur : industriel IT pur = option B, holding diversifié = option A.

## Exercice 3 — Périmètre social (CRITIQUE)

### Effectif initial : 116 ETP
Estimation typique groupe IT+médias :
- Production / développement IT : 60 ETP
- Commercial : 15 ETP
- Editorial / rédaction : 15 ETP
- Support (RH, compta, IT interne) : 10 ETP
- Direction : 6 ETP
- Marketing / autre : 10 ETP

### Périmètre optimisé (Option A)
**Cible : 95 ETP repris (82 %)**
- Production : 55/60 (cherry-pick seniors performants)
- Commercial : 13/15 (suppression doublons)
- Editorial : 12/15 (modèle plus lean)
- Support : 5/10 (intégration repreneur)
- Direction : 0/6 (équipe repreneur)
- Marketing : 7/10

→ 21 ETP non repris × ~50 k€ indemnités = ~1 M€ AGS

### Périmètre optimisé (Option B — IT pure)
**Cible : 70 ETP repris (60 % du total, mais 100 % de la branche IT)**
- Toute la branche IT reprise : ~85 ETP IT - quelques doublons = 70
- 0 ETP de la branche médias (laissée au tribunal)

## Exercice 5 — Red flags à scanner en DD

### Bilan
- Capitaux propres probablement négatifs
- Goodwill éventuellement gonflé (acquisitions historiques)
- PGE éventuel à examiner

### P&L
- CA 11,9 M€ : stable ou en chute ?
- EBITDA -392 k€ : retraitable de combien ?
- Marge brute IT vs médias

### BFR
- DSO clients corporate : possiblement > 60-90 j
- DPO fournisseurs IT : à examiner
- Stocks limités (services)

### Hors-bilan
- Engagements long terme (datacenters, hébergement) : à auditer
- Litiges éditoriaux ?
- Contrats clients avec clauses de change of control ?

### Spécifique IT/médias
- **Propriété intellectuelle** : qui possède le code, les marques, la base abonnés ?
- **Contrats freelances / pigistes** : statut, transferts ?
- **Données clients RGPD** : conformité, transmission ?
- **Cloud / SaaS** : continuité de service au transfert ?
