# J8 — Plan de cession : le mécanisme L642

## Définition en 1 phrase
> Le plan de cession est le mécanisme prévu par les articles **L642-1 à L642-17** du Code de commerce qui permet, dans le cadre d'un RJ ou d'une LJ avec maintien d'activité, le transfert de tout ou partie de l'activité à un repreneur tiers, **sans reprise du passif**, sous l'arbitrage du tribunal.

## Cadre juridique précis
- **L642-1 al. 1** : "La cession de l'entreprise a pour but d'assurer le maintien d'activités susceptibles d'exploitation autonome, de tout ou partie des emplois qui y sont attachés et d'apurer le passif."
- 3 finalités d'égale dignité : **activité**, **emploi**, **apurement passif**

## Les 3 piliers à mémoriser

```
            ┌─────────────────────────────────┐
            │       PLAN DE CESSION L642      │
            └────────────────┬────────────────┘
                             │
         ┌───────────────────┼───────────────────┐
         │                   │                   │
         ▼                   ▼                   ▼
   ① ACTIVITÉ        ② EMPLOI            ③ APUREMENT
   maintenue         maintenu             du passif
   (going concern)   (critère #1          (par le prix
                      tribunal)            de cession)
```

## Périmètre de la cession (L642-1)
Peut porter sur :
- **L'entreprise totale** : intégralité de l'activité (comme une vente classique)
- **Une branche autonome d'activité** : une division identifiable
- **Plusieurs branches**

**Critère** : il doit s'agir d'une **activité susceptible d'exploitation autonome**. Pas une simple vente d'actifs morts.

## Ce qui est repris vs ce qui reste

```
┌─────────────────────────────────────────────────────────────┐
│  REPRIS PAR LE REPRENEUR             RESTE DANS PROCÉDURE   │
├─────────────────────────────────────────────────────────────┤
│  • Actifs désignés dans l'offre       • Passif antérieur    │
│  • Contrats utiles à exploitation     • Dettes financières  │
│    (L642-7 : désignés par AJ)         • Dettes fiscales/    │
│  • Salariés repris (L1224-1)            sociales antérieures│
│  • Marques, brevets repris             • Litiges en cours   │
│  • Stocks repris                       • Sûretés purgées    │
│                                          (L642-12)          │
└─────────────────────────────────────────────────────────────┘
```

## La règle d'or — pas de reprise du passif
**C'est ce qui rend le plan de cession exceptionnel comme outil de M&A.**

Le repreneur achète :
- Les **actifs** désignés
- Les **contrats utiles à l'exploitation** (transférés par décision du tribunal)
- Les **salariés repris** (L1224-1 transfert auto sur les postes repris)

Le repreneur **n'achète pas** :
- Le passif financier antérieur (dettes bancaires, fournisseurs antérieurs)
- Le passif fiscal et social antérieur
- Les litiges en cours
- Les sûretés sont **purgées** (L642-12) sur les biens repris

→ Le repreneur démarre avec un **bilan propre**.

## L'exception : les sûretés "qui suivent le bien"
Attention nuance :
- Les sûretés sont purgées **sur le prix** mais **certaines** suivent l'actif (ex : crédit-bail, réserve de propriété, gage retenant le bien)
- Il faut **identifier** chaque sûreté et négocier avec le créancier ou prévoir le rachat de la créance

## Le périmètre social — L1224-1
- **Principe L1224-1** : si l'activité est transférée, les contrats de travail attachés à cette activité sont **automatiquement transférés**
- **Mais en plan de cession L642-5 al. 1** : le tribunal détermine le **nombre de salariés repris**, sur proposition du repreneur
- → Le repreneur peut donc proposer un **périmètre social inférieur** à l'effectif total
- → Les salariés non repris sont licenciés par le liquidateur (économie financée par AGS dans les limites)

## Calendrier type

```
J0   : Appel à candidatures publié par l'AJ
J+14 à J+42 : Délai de dépôt d'offre (fixé par juge-commissaire)
J+42 à J+50 : Audition par l'AJ + offre revisée si demandée
J+50 à J+60 : Audience tribunal de commerce
J+60 : Jugement de cession
J+60 à J+90 : Closing (purge sûretés, transfert actifs/contrats/salariés, paiement prix)
```

## Procédure : qui fait quoi

| Acteur | Action |
|--------|--------|
| **AJ** | Publie l'appel à candidatures, reçoit les offres, audite les candidats, transmet au tribunal |
| **Repreneur** | Dépose offre L642-2, négocie périmètre, présente projet à l'audience |
| **MJ** | Donne avis sur les offres (impact créanciers) |
| **Procureur** | Donne avis (intérêt général, emploi) |
| **CSE** | Auditionné, donne avis sur le projet emploi |
| **Tribunal** | Tranche selon L642-5 |

## Possibilité d'offre du dirigeant ?
**NON** (sauf exception). Article L642-3 : ni le débiteur, ni les dirigeants, ni leurs proches ne peuvent se porter acquéreurs (sauf autorisation exceptionnelle du tribunal pour les exploitations agricoles).

→ Pour le repreneur, **pas de risque de course interne** masquée.

## Effets à compter du jugement de cession (L642-9+)
1. **Transfert immédiat** des biens et contrats désignés au repreneur
2. **Purge des sûretés** sur les biens repris (L642-12)
3. **Transfert des contrats de travail** repris (L1224-1)
4. **Paiement du prix** entre les mains du MJ
5. **Inaliénabilité temporaire** : le repreneur ne peut revendre les biens essentiels à l'activité pendant la durée fixée par le tribunal (L642-9 al. 2 et 3)
6. **Substitution** possible : le repreneur peut se substituer un tiers sous conditions, mais reste solidairement responsable (L642-9 al. 4)

## Possibilité de modification post-jugement (L642-6)
- Le repreneur peut demander modification du plan en cours d'exécution
- Conditions strictes — pas de modification substantielle qui fausserait la concurrence rétroactive

## Question piège

> "Est-ce que le repreneur peut être tenu responsable des dettes antérieures au jugement de cession ?"

**Réponse modèle (75 sec)** :
> "Non, c'est tout l'intérêt du plan de cession. Le repreneur n'achète **pas** la société, il achète des **actifs** et reprend des **contrats utiles à l'exploitation**, désignés par le tribunal. Le passif antérieur — dettes bancaires, fournisseurs, fisc, URSSAF, prud'hommes — reste dans la procédure et est apuré par le prix de cession, dans l'ordre des privilèges. Il y a deux nuances importantes à connaître. Premièrement : les sûretés qui grèvent les biens repris sont **purgées** au sens de l'article L642-12, mais **certaines sûretés suivent le bien** — c'est le cas du crédit-bail, de la réserve de propriété, du gage avec dépossession. Il faut donc auditer chaque actif. Deuxièmement : sur le volet social, l'article L1224-1 prévoit le transfert automatique des contrats de travail attachés aux postes repris, mais le tribunal valide en amont le nombre de salariés repris sur proposition du repreneur. Donc le repreneur ne reprend pas tout le personnel — uniquement le périmètre social validé par le tribunal."

## À retenir absolument
- L642-1 à L642-17 = cadre du plan de cession
- 3 finalités : activité + emploi + apurement passif
- Périmètre choisi : actifs + contrats utiles + salariés repris
- Pas de reprise du passif antérieur (sauf sûretés spécifiques)
- Tribunal arbitre selon L642-5 (emploi prime)
- L642-3 : dirigeant exclu de l'offre (sauf agriculture)
- Inaliénabilité temporaire (L642-9)
