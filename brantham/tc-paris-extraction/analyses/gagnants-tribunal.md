# Gagnants des plans de cession — TAE Paris

Recherche menée sur 11 dossiers prioritaires extraits du master CSV (`master-offres.csv`, 571 lignes). Objectif : identifier les repreneurs retenus pour comprendre les patterns d'arbitrage du Tribunal des Activités Économiques de Paris.

**Score final : 7 dossiers décidés et tracés / 11 dossiers ciblés.** Sur les 7, 6 sont des reprises confirmées avec repreneur identifié, 1 est une liquidation totale (Minelli — précieux comme contre-exemple).

## Synthèse — patterns observés

| Pattern tribunal | Cas | Enseignement |
|---|---|---|
| Offre repreneur stratégique avec continuité d'emploi | COSA (Agence IF, 16 emplois), Sym Group (LOM founder), Vergers d'Anjou (Innatis) | Le tribunal privilégie la continuité sectorielle même si le prix n'est pas le plus haut |
| Repreneur financier asiatique sur retail mode | Eurodif (AA Investments HK Ltd), Bouchara (AA Investments HK Ltd) | AA Investments (HK) Limited est un repreneur récurrent — fond HK qui rachète des enseignes mode FR à la barre. À surveiller. |
| Démantèlement avec plusieurs repreneurs sectoriels | Neocamp (Seasonova pour Alpha 15 sites + Maeva pour franchises Camping Paradis/Ushuaïa) | Sur les groupes, le tribunal accepte le découpage si offres consolidées |
| Refus du tribunal — liquidation totale | Minelli (6 offres partielles, toutes refusées) | Offres partielles couvrant 1-2 magasins sur 21, sans engagement emploi → liquidation. Pattern : pas de partial deals sur retail multi-sites sans engagement global. |

## Enseignements consolidés

### Patterns confirmés
1. **AA Investments (HK) Limited** = repreneur HK récurrent sur retail mode FR distressed (Bouchara, Eurodif, Bonne Gueule). Cible probable de pipeline Brantham pour les sourcings retail mode.
2. **Offres partielles cherry-picking refusées en bloc** quand pas de garantie d'emploi globale (Minelli — 6 offres, 0 retenue).
3. **Démantèlement accepté** quand lots cohérents par sous-activité (Neocamp = Seasonova gestion directe + Maeva franchises).
4. **Repreneurs sectoriels stratégiques privilégiés sur EdTech / Santé** (LOM pour Sym, Vergers d'Anjou pour Innatis pôle, Agence IF pour Cosa).
5. **Délai jugement post-DLDO** typique : 1-2 mois (Innatis DLDO ~mi-mars → jugement 30/04 ; Eurodif DLDO 10/03 → jugement 11/05).

### Dossiers à creuser ultérieurement (priorité décroissante)
- **Collège de Paris** : très gros enjeu EdTech (1300 emplois). Audience probable mai-juin 2026. Cibler Galileo Global Ed, Studialis, Ionis Group, Skema.
- **SEGI** : 19 offres foncières 10-25 M€ — pattern à comprendre (qui rachète des immeubles parisiens à la barre ?).
- **Progress Sup** : pattern école santé en difficulté — utile pour comparer avec Collège de Paris.
- **Chepooka** : cas iconique tiers-lieu culturel — bon proxy pour deals brand-driven Paris.
- **Denteka** : confirmer ou non MEDIVI comme repreneur — pattern centres dentaires.

### Méthode
- BODACC web interface peu indexée → privilégier consultation directe greffe ou Pappers (changements dirigeants post-RJ).
- Press releases repreneurs (LinkedIn dirigeants) = signal le plus rapide.
- Sources sectorielles essentielles : FashionNetwork (mode), Acuité (optique), L'Officiel des terrains de camping, Réussir Fruits & Légumes, CFNEWS.
