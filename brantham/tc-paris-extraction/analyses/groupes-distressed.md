# Groupes en difficulté multi-entités (TC Paris)

Le dataset TC Paris révèle plusieurs **groupes corporate** placés simultanément en procédure collective, avec offres souvent **indivisibles** sur plusieurs entités. C'est un pattern récurrent dans les RJ de PME structurées en holding + filiales.

## Pourquoi c'est important
Pour Brantham :
- **Sourcing repreneur** : un repreneur intéressé par une entité du groupe peut être candidat sur les sœurs.
- **Analyse de valorisation** : les offres groupées masquent souvent les vraies valeurs par actif.
- **Risque indivisibilité** : si une offre est rejetée, toutes tombent → opportunité de surenchère ciblée.

## Pattern global

Sur 94 dossiers :
- **6 clusters multi-entités** identifiés (INNATIS, NEOCAMP, DENTEKA, SYM, COLLEGE DE PARIS, SEGI)
- **35 dossiers immobiliers SCI/SNC** isolés (Paris IDF) avec MUVICO/HOUSEBASE/LABRAX en repreneurs récurrents
- **~50 dossiers** d'exploitation standalone (PME services, restauration, retail, etc.)
