# Playbook — Rédaction d'offres de reprise (TC Paris)

Synthèse opérationnelle tirée de l'analyse de **570 offres** déposées au Tribunal des Activités Économiques de Paris (94 dossiers d'origine + 10 nouveaux). C'est le guide à appliquer pour les **clients-repreneurs Brantham**.

## 1. Architecture standard d'une offre gagnante

D'après les offres analysées (notamment celles présentées par MONCEY AVOCATS, FIDAL, LEXCAP, HADENGUE — cabinets les plus présents), la structure récurrente est :

```
1. Préambule / Identification
   - Société débitrice (RCS, AJ/MJ, n° procédure, jugement RJ daté)
   - Objet : offre L.642-1 (plan de cession) ou L.642-19 (cession actifs)

2. Le candidat repreneur
   - Identité juridique (SAS/SARL + RCS + capital)
   - Dirigeants (avec CNI annexée)
   - Groupe d'appartenance (organigramme complet jusqu'au BE)
   - Historique du groupe (CA, RN, effectif, références reprises distressed)
   - Faculté de substitution vers NewCo dédiée (très fréquent — voir §6)

3. Périmètre repris
   - Incorporels : marques, clientèle, fichier, savoir-faire, SI, certifications
   - Corporels : matériel, mobilier (inventaire annexé)
   - Stocks (séparément valorisés)
   - Contrats : baux commerciaux + crédits-baux + nantissements
   - Filiales et participations
   - EXCLUSIONS explicites : créances clients, dispos, passif, contentieux, immo financières

4. Prix de cession
   - Ventilé : incorporels / corporels / stocks
   - Préciser si NET vendeur ou TTC
   - Charges augmentatives séparées (loyers, congés payés, dépôt garantie, reprise emprunt L.642-12 al.4)

5. Reprise des salariés
   - Nombre précis (X / Y total)
   - Liste des postes repris (annexe)
   - Engagement non-licenciement (durée typique : 12-24 mois)
   - Sort des non-repris : priorité réembauche

6. Conditions suspensives
   - Idéalement : AUCUNE (offre ferme, valorisation +++)
   - Sinon : numérotées, exhaustives, datées

7. Financement
   - Mode (fonds propres / emprunt / mixte)
   - Preuves (RIB, attestation banque, lignes de crédit, levée de fonds)
   - Chèque de consignation si demandé

8. Validité de l'offre
   - Date butoir (généralement 1-2 mois après audience)

9. Déclarations
   - Art. L.642-3 (absence de lien parenté/affinité avec dirigeants)
   - Art. L.642-12 al.4 (reprise du nantissement bancaire)

10. Annexes (ordre logique attendu) — voir §3
```

## 2. Pricing — Ce que le tribunal accepte

**Distribution des 291 offres avec prix parsable :**

| Tranche | Nombre | % |
|---|---|---|
| Symbolique 1-9€ | 1 | 0% |
| 100-999€ | 1 | 0% |
| 1k-10k€ | 31 | 11% |
| **10k-100k€** | **119** | **41%** |
| 100k-1M€ | 86 | 30% |
| 1M-10M€ | 22 | 8% |
| > 10M€ | 32 | 11% |

**Médiane : 80 000 €**. La majorité des offres se situent entre 10k€ et 1M€.

### Leçons sur le prix
- **Le tribunal accepte les prix symboliques (1-3 €)** quand le repreneur reprend l'EMPLOI + le passif (charges augmentatives). Exemples : SLBC PERON sur NEOCAMP à 3€×N, IFOCOP à 1€, AAG 56 sur Île de Groix à 1€.
- **Le ratio prix/emploi médian est 10 000 €/emploi.** En dessous, justifier par charges augmentatives reprises. Au-dessus, le tribunal trouve l'offre "généreuse" mais ne valorise pas plus s'il n'y a pas d'emplois supplémentaires.
- **La ventilation prix est CRUCIALE.** Toujours préciser : `X k€ incorporels + Y k€ corporels + Z k€ stocks`. Plus la ventilation est détaillée, plus crédible.
- **Charges augmentatives** : 25 offres (4%) les mentionnent. Quand elles existent (loyers + CP acquis + reprise emprunt), elles peuvent multiplier la valeur économique totale par 5-10x.

## 3. Annexes — La checklist obligatoire

D'après les patterns observés, voici les annexes systématiquement présentes dans les offres SOLIDES :

**Obligatoires** :
1. ✅ Kbis du repreneur (et des holdings remontant au BE) — 55 mentions explicites
2. ✅ Statuts à jour
3. ✅ CNI/passeport des dirigeants (15 mentions)
4. ✅ Comptes annuels 3 derniers exercices (13 mentions)
5. ✅ Attestation fiscale et sociale à jour
6. ✅ RIB du compte qui financera (et chèque de consignation si exigé — 8 mentions)
7. ✅ Cotation Banque de France (très valorisé pour les gros tickets — vu sur DataSolution/COSA)
8. ✅ Business plan COSA-style : prévisionnel 3 ans (CA / EBITDA / CAPEX / BFR)
9. ✅ Déclaration L.642-3 signée (absence parenté)

**Renforcent la crédibilité** :
10. Organigramme groupe complet jusqu'au BE
11. Communiqué de presse ou track record de reprises distressed passées
12. Liste des contrats critiques à maintenir (clients top 5, fournisseurs stratégiques)
13. Lettre de soutien banque/investisseur (pour gros tickets)
14. Plan d'intégration social (CSE, communication aux salariés)

**Stack signature** : Docusign Envelope ID est mentionné dans **88 offres** — c'est le standard de facto. Un PDF Docusign signé électroniquement est mieux qu'un scan papier.

## 4. Conditions suspensives — Stratégie

| Pattern | Fréquence | Recommandation |
|---|---|---|
| **Aucune condition suspensive** | 9% | **STRATÉGIE GAGNANTE** : offre ferme = forte valorisation par le tribunal. À privilégier dès qu'on peut. Ex: LOM sur SYM, BLUEDIGO sur FAIRSPACE. |
| Audit / due diligence | 9% | Acceptable mais affaiblit. Préciser durée courte (15-30j). |
| Cohérence dataroom | 2% | Soft, généralement acceptée. |
| Accord bailleur / agrément | 3% | Difficile à éviter quand bail commercial = actif clé. Ajouter délai. |
| Reprise prêt L.642-12 | 9% | **Souvent OBLIGATOIRE** quand nantissement bancaire — l'offre DOIT le mentionner. |
| L.642-3 (parenté) | <1% | **DANGER** : autorisation Ministère public requise + jugement spécialement motivé. Si applicable, déclarer transparemment. |
| Autorisation administrative | 2% | Spécifique (DGFIP/préfecture pour activités réglementées). |
| Validation CA repreneur | <1% | Acceptable si délai court. |
| Financement bancaire | 1% | Affaiblit l'offre. Si possible, sécuriser AVANT dépôt. |

### Règle d'or
**Moins il y a de conditions suspensives, mieux c'est.** Le tribunal préfère une offre ferme moins chère qu'une offre plus chère avec 5 conditions. Notre métier de conseil = aider le repreneur à transformer ses conditions en pré-requis purgés AVANT le dépôt.

## 5. Reprise des emplois — Critère #1 du tribunal

C'est **le critère prépondérant** que regarde le tribunal (art. L.642-5).

- **Ratio prix/emploi médian : 10 000 €/emploi**. Le tribunal favorise les offres qui maximisent les emplois.
- **Cas FRAPPANT : MAJE/SMCP sur MINELLI à 550k€ pour 0 emploi vs Baghaira à 300k€ pour 9 emplois — le tribunal arbitrera potentiellement en faveur de Baghaira sur le partiel digital** (à confirmer par la décision).
- **Cas CONTRE-INTUITIF : ALPHA CAMPING — FEMINA STYL+MARENGO à 2 M€ avec 19 emplois retenue probable** vs offres concurrentes à 1€ avec 5-10 emplois.

### Comment formuler les emplois
- Format : `X / Y` (X repris sur Y total)
- Liste nominative en annexe (postes, ancienneté)
- Engagement de non-licenciement chiffré (durée 12-24 mois)
- Mention "priorité réembauche" pour les non-repris (obligation morale + signal positif)

### Pattern observé : améliorations augmentent les emplois
Les améliorations entre offre initiale et finale concernent **les emplois plus souvent que le prix** :
- PROGRESS SUP : J.L 13 RUE DE LA GRANGE BATELIERE : 3 → 44 emplois (offre finale)
- COSA : Agence IF : 3 → 16 emplois (offre définitive)
- LPC INNATIS : 6 → 19 emplois (consortium amélioré)
- ESCAPADE / NEOCAMP : 0 → 13 emplois pour Mauvin/Noel

**Tactique** : si concurrence, l'amélioration sur les emplois > l'amélioration sur le prix.

## 6. Faculté de substitution — Standard de marché

**70% des offres mentionnent une faculté de substitution** vers une NewCo (SAS ad hoc) à constituer entre l'offre et le closing.

### Pourquoi systématique
- Limite la responsabilité de la holding mère
- Permet de structurer le financement (CCCA, emprunt acquéreur)
- Permet de prévoir un actionnariat managérial ou financier
- Permet une intégration fiscale optimisée

### Wording standard
> "Le candidat se réserve la faculté de se substituer, pour la signature des actes définitifs, par une société qu'il constituera (ci-après 'la Société Cessionnaire'), dont il détiendra directement ou indirectement le capital. Le candidat demeurera solidairement responsable des engagements pris."

### Précisions à apporter
- Forme juridique de la NewCo
- Capital social prévu
- Actionnariat (qui détient quoi)
- Engagement solidaire du candidat initial

## 8. Améliorations & contre-offres — Comment réagir

D'après les **18 cas d'amélioration mesurés**, voici les patterns gagnants :

| Mouvement | Effet sur acceptation |
|---|---|
| **+ Emplois repris** (le plus puissant) | Souvent décisif |
| **+ Prix sur incorporels** | Acceptable mais marginal |
| **+ Reprise charges augmentatives** | Très valorisé (signal capacité financière) |
| **- Conditions suspensives** | Très valorisé |
| **+ Périmètre géographique** | Valorisé si reprise multi-sites |
| **Substitution clarifiée** | Renforce la crédibilité |

### Quand améliorer
- **Avant DLDO** : si concurrence connue
- **Après audience de présentation** : 4 cas dans le dataset (DENTEKA, COSA) — possible de déposer une offre améliorée si le tribunal le demande
- **Très tardive** (post-DLDO) : 1 cas (O'SULLIVANS sur CHEPOOKA) — admise par exception

### Top progressions observées (signal de ce qui fonctionne)
1. CASA IDEAS sur EURODIF : 5k€ → 138k€ (+2660%)
2. AGENCE IF sur COSA : 5k€ → 70k€ + 3→16 emplois (+1300%)
3. J.L sur PROGRESS SUP : 10k€ → 100k€ + 3→44 emplois (+900%)
4. HENRIQUET sur MY LITTLE FINGER : 80k€ → 313k€ (+291%)
5. FINANCIERE DAUNOU 9 sur MY LITTLE FINGER : 80k€ → 281k€ (+251%)

## 10. Erreurs fréquentes observées (à éviter)

| Erreur | Conséquence | Vu dans |
|---|---|---|
| Prix laissé en placeholders `[3]€` | Offre non chiffrée = rejetée | MEDIVI v1 sur CLAYE/ENGHIEN |
| Incohérence chiffres / lettres | Doute sur le montant réel | Yource sur D&N (60k€ mots vs 75k€ chiffres) |
| Conditions suspensives auto-contradictoires | Tribunal sceptique | QASTI sur DME ("aucune cond." + 4 listées) |
| Doublons techniques (mêmes Docusign 3x) | Pas un vrai défaut mais perte de clarté | FAIRSPACE, EURODIF, FAUCHON |
| Périmètre flou ou contradictoire | Risque de re-négo post-jugement | SIMA sur SEGI (lots déjà vendus) |
| Manque Kbis / état civil dirigeant | Annexes incomplètes = report ou rejet | Plusieurs PDFs scannés vague 1 |
| Faculté substitution sans précision actionnariat | Tribunal demande complément | Plusieurs |
| Prix élevé mais 0 emploi | Tribunal défavorable si offre concurrente avec emplois | Common pattern retail |

## 11. Brantham — Checklist opérationnelle

Quand on accompagne un client repreneur, on doit lui produire **AVANT dépôt** :

### Phase 1 — Sourcing & qualification (J-30 avant DLDO)
- [ ] Identifier la cible via scraping TC Paris (script `_scrape_tc_paris.py`)
- [ ] Lire TOUTES les offres concurrentes déjà déposées (data room)
- [ ] Identifier les repreneurs concurrents et leur stratégie (foncière, consolidateur, industriel)
- [ ] Mapper les emplois (critique au #1 du tribunal)

### Phase 2 — Stratégie de l'offre (J-20)
- [ ] Fixer le prix selon : ratio prix/emploi cible 10-50 k€ + charges augmentatives à reprendre
- [ ] Ventilation incorporels/corporels/stocks
- [ ] Faculté de substitution : valider la NewCo
- [ ] Sécuriser le financement (banque/levée) AVANT pour éviter cond. susp.

### Phase 3 — Rédaction (J-10)
- [ ] Préambule + identification (Kbis + RCS + capital)
- [ ] Périmètre + exclusions exhaustives
- [ ] Conditions suspensives : viser ZÉRO ou maximum 2
- [ ] Annexes complètes (Kbis, comptes 3 ans, BP, attest. fisc/soc, RIB, CNI, organigramme, déclaration L.642-3)
- [ ] Engagement non-licenciement chiffré
- [ ] Cotation BdF si gros ticket

### Phase 4 — Dépôt (J-1)
- [ ] Signature Docusign (avec Envelope ID traçable)
- [ ] Dépôt au greffe via espace AJ
- [ ] Copie à l'AJ et au MJ
- [ ] Confirmation écrite (mail horodaté)

### Phase 5 — Audience (J)
- [ ] Préparer le repreneur à l'audience (questions probables du tribunal)
- [ ] Avoir une offre améliorée prête (post-audience si demandée)
- [ ] Surveiller les actes d'amélioration concurrents

### Phase 6 — Post-jugement (J+1 à J+30)
- [ ] Surveiller BODACC pour le jugement de plan de cession (publié sous 8-30j)
- [ ] Suivi Pappers du repreneur (constitution NewCo, changements dirigeants)
- [ ] Accompagnement closing : signature des actes définitifs
