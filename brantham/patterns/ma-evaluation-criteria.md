# 3 Critères d'évaluation M&A clés pour Brantham

## 1. Strategic Product Fit Score (1-10)

**Critère clé : Complémentarité produit/clientèle**

### Éléments d'évaluation :
- **Synergies technologiques** : Stack compatible, API intégrables
- **Chevauchement clientèle** <30% ou >70% (éviter la concurrence directe)
- **Réseau de distribution** : Canaux complémentaires ou synergiques
- **Tech debt ratio** : <20% pour les targets acquises
- **Innovation pipeline** : Minimum 1 produit commun à 18 mois

**Poids** : 40% du scoring global

**Pourquoi essentiel** : Brantham se positionne sur le marché B2B SaaS high-tech - les acquisitions doivent créer un écosystème produit cohérent plutôt que des patchworks technologiques.

---

## 2. Financial Health Acceleration Score (1-10)

**Critère clé : Potentiel de croissance financière post-acquisition**

### Éléments d'évaluation :
- **Revenue per employee** : >100K€ (benchmark SaaS européen)
- **Cash conversion cycle** : <60 jours (liquidité opérationnelle)
- **Dettes nettes/EBITDA** : <3x (soutenabilité financière)
- **Marge brute** : >60% (margeabilité SaaS)
- **Croissance organique** : >15% YoY (trajectoire ascendante)

**Poids** : 35% du scoring global

**Pourquoi essentiel** : Les cibles doivent immédiatement contribuer au P&L consolidé sans exiger des investissements de transformation majeurs.

---

## 3. Team Integration Risk Score (1-10)

**Critère clé : Alignement culturel et rétention clé**

### Éléments d'évaluation :
- **CEO founder fit** : Vision alignée sur Brantham values
- **Tech lead compatibility** : Stack engineering similaire
- **Turnover rate** : <15% (stabilité équipe)
- **Key person dependency** : <30% (répartition rôles)
- **Remote work adoption** : >50% (hybrid compatible)

**Poids** : 25% du scoring global

**Pourquoi essentiel** : Le succès des acquisitions SaaS dépend 70% de la rétention des équipes techniques - Brantham ne peut se permettre des départs massifs post-acquisition.

---

## Méthodologie d'application

### Scoring composite :
```
Score Total = (Product Fit × 0.4) + (Financial Health × 0.35) + (Team Integration × 0.25)
```

### Seuils d'investissement :
- **<5/10** : Pas d'investissement
- **5-7/10** : Étude préliminaire seulement
- **7-8/10** : Due diligence approfondie
- **>8/10** : Proposition d'acquisition immédiate

### Indicateurs rouges :
- Tech debt >30%
- Turnover >20%
- Chevauchement client 30-70%
- Dettes >4x EBITDA
- Cash conversion >90 jours
