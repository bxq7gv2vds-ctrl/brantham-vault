# Checklist de Valuation Multiples Standards

### 🚀 SaaS & Tech
| Multiple | Valeur typique | Usage |
|----------|----------------|-------|
| **EV/EBITDA** | 12-20x | Standard principal |
| **EV/Revenue** | 3-8x | Croissance rapide |
| **ARR Multiple** | 4-10x | Subscription models |
| **Revenue per Employee** | 100-250K€ | Productivité équipe |

- France : 15-18x EBITDA

## Méthodologie d'Application

### 🎯 Étape 1 : Sélection des multiples appropriés
1. **Analyser le stade de maturité** : Seed/Série A/B/C/Growth
2. **Identifier le secteur** : SaaS/B2B/B2C/Marketplace
3. **Vérifier les comparables** : Deals récents sectoriels
4. **Adapter au modèle économique** : Recurring vs transactional

### 📈 Étape 2 : Calcul des ranges
```
EV = Multiple × Métrique
Prix = EV - Dette Nette + Cash
```

### 🔍 Étape 3 : Validation croisée
| Métrique | Valeur calculée | Benchmark | Écart |
|----------|------------------|-----------|-------|
| **EV/EBITDA** | | 12-20x | ±10% |
| **EV/Revenue** | | 3-8x | ±15% |
| **P/E** | | 20-30x | ±20% |

## Multiples par Taille d'Entreprise

### 🏢 Grandes entreprises (>100M€ CA)
- **EV/EBITDA** : 8-15x
- **P/E Ratio** : 15-25x
- **EBITDA Margin Multiple** : 6-12x

### 🏢 Moyennes entreprises (10-100M€ CA)
- **EV/EBITDA** : 10-18x
- **P/E Ratio** : 18-30x
- **Revenue Multiple** : 3-6x

### 🏢 Petites entreprises (<10M€ CA)
- **EV/EBITDA** : 6-12x
- **P/E Ratio** : 12-20x
- **Revenue Multiple** : 1-4x

---

## Facteurs d'Ajustement

### ⬆️ Positifs (+20% à +50%)
- **Market leader** >30% share
- **Recurring revenue** >80%
- **High gross margin** >70%
- **Low churn** <5%
- **Strong growth** >30% YoY

### ⬇️ Négatifs (-20% à -50%)
- **Concentration client** >50% single client
- **High churn** >15%
- **Seasonality** >30% variation
- **Regulatory risk** élevé
- **Dependence** supplier/customer

## Quality Checklist

### ✅ Validation requise
- [ ] Minimum 3 comparables sectoriels récents (<24 mois)
- [ ] Multiple within industry range ±20%
- [ ] Adjusted for company-specific factors
- [ ] Cross-validated with multiple methodologies
- [ ] Risk premium properly applied

### ❌ Red flags
- Single multiple only
- Using outdated comparables (>24 mois)
- Ignoring sector-specific adjustments
- No risk premium for private companies
- Over-reliance on one metric

---

## Deal Stage Application

### 🎯 Early Stage (Pre-seed to Series A)
- **Focus** : Traction, team, market size
- **Multiples** : EV/Revenue, Cap tables
- **Timeline** : 3-6 mois de données

### 🚀 Growth Stage (Series B to C)
- **Focus** : Unit economics, growth rate
- **Multiples** : EV/EBITDA, EV/Revenue
- **Timeline** : 12-18 mois de données

### 🏢 Mature Stage (Profitable)
- **Focus** : Cash flow, margins, stability
- **Multiples** : EV/EBITDA, P/E, DCF
- **Timeline** : 24+ mois de données
