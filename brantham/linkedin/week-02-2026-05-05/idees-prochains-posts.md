# Idées prochains posts LinkedIn — suite vautour

> Suite logique au [[post-06-distressed-moral-vautour]]. Audience qui vient d'absorber un post moral choc, il faut varier le registre. **Le user a refusé les angles purement pédagogiques** (vrai prix d'une reprise, framework PAPI, builder funnel) au profit d'angles meme / sympa / build-in-public.

## Angles validés à explorer

### A. Meme suite vautour, format Anakin 2x2
Capitalise sur la DA Anakin déjà calibrée ([[dsys-lp/meme-anakin-lp]]). Dialogue 4 cases :

```
Anakin (naïf) : "J'ai repris une boîte à 1€."
Padmé (souriante) : "Avec la dette URSSAF reprise ?"
Anakin (silence) : "..."
Padmé (sourire glacé) : "Avec la dette URSSAF reprise ?"
```

Sujet : la naïveté du repreneur qui ne voit pas le passif. Continuité visuelle Brantham, screenshot-able, viral potentiel haut.

Variantes possibles :
- *"On va racheter la boîte sans le passif." / "Avec les contrats fournisseurs renégociés ?"*
- *"Plan de cession à 50k€." / "Avec l'AGS pour les indemnités ?"*
- *"Le bail est protégé." / "Tu as vu la clause de résiliation en cas de procédure ?"*

### B. Stat-shock 4 lignes
Format sec, Hormozi-style, pas d'image (ou image minimaliste cream texte Newsreader) :

```
67% des défaillances françaises finissent en liquidation directe.
47 078 entreprises mortes en 2025.
Pas faute de repreneur.
Faute d'être détectées à temps.
```

Très court, diagnostic brutal, pose Brantham comme solution implicite au problème.

### C. Build-in-public — "À quoi ressemble mon mardi"
Coulisses du métier, vibe Karpathy / agentic-micro-company :

```
8h : agent scrape 47 nouvelles procédures BODACC nuit
9h : 3 dossiers passent le scoring, je les ouvre
10h : appel AJ Marseille sur un dossier industriel
11h : data room sur un autre, 12 docs à digérer
14h : teaser envoyé à 4 acquéreurs stratégiques
17h : un retourne, on cale un call demain

J'ai un cabinet M&A qui tient dans un MacBook. C'est volontaire.
```

Sourcé sur [[writing-vault/drafts/banger-seeds/08-context-farmer-paul-signature]]. Très différent du vautour, plus humain, plus sympa.

### D. Verbatim humain, sans commentaire
Une phrase entendue, sortie brute :

```
Le dirigeant : "Je préfère que ma boîte ferme plutôt que d'être rachetée par un Parisien."

Sa boîte a 34 ans. Elle fait 2,1M€. Elle ferme en juillet.

Le narratif tue plus que les marchés.
```

Court, visuel, humain. Marche bien en commentaires / partages.

## Angles refusés (pour mémoire)

- **"Le vrai prix d'une reprise à 1€"** (décomposition pédagogique avec calculateur) — refusé : trop pédagogique sec
- **"Le M&A distressed se joue à 5, pas à 2"** (framework PAPI) — refusé : trop intellectuel/abstrait
- **"Builder flex : 3 918 procédures scrapées"** (funnel chiffres) — refusé : trop technique/flex

**Pattern de feedback** : user veut continuer dans le registre meme/punchy/sympa du vautour, pas pivoter sur pédagogique-expertise. Capitaliser sur la DA visuelle et le ton offensif posé hier.

## Architecture éditoriale proposée

1. J+0 (publication post-06 vautour) : opinion choc moralité
2. J+1 : meme Anakin URSSAF (option A) — continuité visuelle, registre meme
3. J+2 : stat-shock 4 lignes (option B) — varier vers data brute
4. J+3 : build-in-public mardi (option C) — humaniser le compte
5. J+5 : verbatim dirigeant (option D) — clôture émotionnelle de la séquence

Cinq posts qui se renforcent, chacun avec un registre différent (moral / meme / data / coulisses / humain), tous dans le même univers Brantham distressed.
