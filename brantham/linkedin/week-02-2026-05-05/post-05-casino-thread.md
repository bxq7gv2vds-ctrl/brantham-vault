## COMMENTAIRE 1/7 — La pyramide

```
1/

Avant la procédure, le groupe Casino tenait par une pyramide à quatre étages.

Au sommet, Jean-Charles Naouri. Sous lui, Euris. Sous Euris, Finatis. Sous Finatis, Foncière Euris. Sous Foncière Euris, Rallye, qui détenait 52 % du capital de Casino.

Naouri possédait environ 25 % du capital économique réel. Mais grâce à la mécanique pyramidale, il contrôlait 63 % des droits de vote.

L'édifice tenait tant que Casino remontait des dividendes vers le haut. Quand le cash a manqué, chaque étage s'est mis à porter sa propre dette.
```

## COMMENTAIRE 2/7 — L'accord avant la procédure

```
2/

Juillet 2023. Casino reçoit deux offres concurrentes pour reprendre la main.

Le 27 juillet, la direction signe un accord de principe avec un consortium mené par Daniel Kretinsky (EP Global Commerce, Fimalac, Attestor).

Ce n'est pas une OPA. C'est une négociation menée en privé entre actionnaires, créanciers et repreneur. Le marché ne tranche rien. Aucune action ne change de main sur Euronext.

À ce stade, l'accord existe sur le papier. Il reste à le rendre opposable à tout le monde.
```

## COMMENTAIRE 3/7 — L'ouverture de la procédure

```
3/

Le 25 octobre 2023, le tribunal de commerce de Paris ouvre une procédure de sauvegarde accélérée.

Une procédure de droit français pensée pour ça. Elle est réservée aux groupes qui ont déjà négocié leur restructuration en amont. L'objectif n'est pas de chercher des repreneurs. L'objectif est de transformer un accord privé en plan opposable à tous les créanciers, y compris ceux qui refuseraient de signer.

La durée maximale est de quelques mois. Pas de période d'observation longue, pas d'incertitude prolongée. C'est un outil de finalisation, pas d'enquête.
```

## COMMENTAIRE 4/7 — Le mécanisme

```
4/

Le mécanisme central s'appelle classes de parties affectées.

Les créanciers et les actionnaires sont répartis en classes selon leur rang juridique : banques, obligataires senior, porteurs de titres hybrides, fournisseurs, actionnaires.

Chaque classe vote sur le plan. Une majorité interne à chaque classe doit dire oui. Si une majorité des classes vote en faveur, le plan s'impose aux classes dissidentes. Y compris aux actionnaires existants.

C'est la pièce technique. Sans elle, rien de ce qui suit n'aurait été possible.
```

## COMMENTAIRE 5/7 — Le verrou cassé (réécrit pour clarté)

```
5/

Avant 2021, Naouri pouvait dire non. Et son non bloquait tout.

Pourquoi. Un plan de restructuration devait être approuvé par l'assemblée générale des actionnaires. À l'assemblée, ce sont les droits de vote qui décident. Avec 63 % des votes, Naouri avait un veto absolu, peu importe ce que voulaient les créanciers.

En 2021, la France transpose la directive européenne 2019/1023. Un outil nouveau entre dans le Code de commerce : le cram-down inter-classes.

À partir de là, le plan n'est plus voté par une assemblée unique. Il est voté classe par classe. Les banques votent entre elles. Les obligataires entre eux. Les actionnaires entre eux.

Si une majorité des classes vote oui, le plan passe. Même si la classe des actionnaires vote non.

Naouri n'est plus une majorité. Il est une classe parmi d'autres. Son veto a disparu.

C'est ce changement, technique et passé presque inaperçu, qui a redessiné les rapports de force du M&A français.
```

## COMMENTAIRE 6/7 — Le résultat

```
6/

11 janvier 2024. Les classes votent.
26 février 2024. Le tribunal valide le plan.
27 mars 2024. Le plan est exécuté.

Le consortium Kretinsky injecte 1,2 milliard d'euros et prend 53 % du capital.
4,9 milliards d'euros de dette sont convertis en actions, donnant 46 % du capital aux anciens créanciers.
Les anciens actionnaires sont dilués à hauteur de 99 %.

Naouri, aux commandes depuis 30 ans, quitte la présidence. Le groupe change de main sans qu'une seule action n'ait été achetée sur Euronext.

Six mois entre l'ouverture de la procédure et le changement de contrôle.
```

## COMMENTAIRE 7/7 — La portée

```
7/

Casino n'est pas un cas isolé. C'est un signal.

Le canal qui a permis ce transfert s'appelle distressed M&A. En français, restructuration d'entreprises en difficulté. Le mot « difficulté » trompe. Il ne s'agit plus seulement de PME en liquidation.

C'est devenu, depuis 2021, le canal légal pour transférer le contrôle de groupes que personne d'autre ne peut bouger. Y compris des groupes à plusieurs dizaines de milliards d'euros de chiffre d'affaires.

La plupart des dirigeants, des repreneurs et des conseils M&A n'ont jamais regardé ce canal de près. Il est rapide. Il est encadré. Il fonctionne. Et il ne fait presque jamais l'actualité, parce que les négociations qui le préparent restent confidentielles.

C'est dans cet angle mort que se font, aujourd'hui, certaines des plus grosses opérations européennes.
```
