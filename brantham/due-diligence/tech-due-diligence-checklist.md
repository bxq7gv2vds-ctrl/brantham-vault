# Due Diligence Tech - Checklist Complète

**Version** : 1.0  
**Secteur** : Technology/SaaS  
**Durée** : 4-6 semaines standard

## 🔍 Technology Due Diligence

### 🖥️ Infrastructure & Architecture
- [ ] **Cloud providers**: AWS/GCP/Azure + costs breakdown
- [ ] **Multi-region deployment**: HA/DR capabilities
- [ ] **Infrastructure code**: Terraform/CloudFormation reviewed
- [ ] **CI/CD pipelines**: Jenkins/GitLab/GitHub Actions status
- [ ] **Containerization**: Docker/Kubernetes implementation
- [ ] **Monitoring**: Prometheus/Grafana/Sentry setup
- [ ] **Backup strategy**: RPO/RTO documented

### 💾 Data & Security
- [ ] **Data encryption**: At rest + in transit standards
- [ ] **Access controls**: RBAC implementation reviewed
- [ ] **Compliance**: SOC2, ISO 27001, GDPR status
- [ ] **Penetration testing**: Last 2 years reports
- [ ] **Vulnerability management**: Patch cadence tracked
- [ ] **Data classification**: Schema + sensitivity mapping
- [ ] **Third-party integrations**: Security assessments done

### 🧱 Code Quality & Tech Debt
- [ ] **Code coverage**: Unit/integration tests >80%
- [ ] **Code review process**: Frequency + participation
- [ ] **Technical debt**: % time spent vs new features
- [ ] **Dependencies**: Security scans for vulnerabilities
- [ ] **Architecture documentation**: Up-to-date diagrams
- [ ] **API contracts**: OpenAPI/Swagger documentation
- [ ] **Build pipeline**: Success rate + average duration

### 🚀 Product & Engineering Team
- [ ] **Team structure**: Org chart + skill distribution
- [ ] **Key personnel**: Retention risk assessment
- [ ] **Technical leadership**: CTO background + experience
- [ ] **Engineering metrics**: Deployment frequency, lead time
- [ ] **Hiring pipeline**: Key roles identified
- [ ] **Knowledge sharing**: Documentation quality

---

## 💰 Financial Due Diligence

### 📊 Revenue Recognition
- [ ] **Revenue recognition**: ASC 606 compliance
- [ ] **Contract review**: Terms + auto-renewal clauses
- [ ] **Revenue streams**: Breakdown by product/segment
- [ ] **Deferred revenue**: Schedule + churn impact
- [ ] **Revenue recognition schedule**: Monthly tracking

### 📈 Unit Economics
- [ ] **CAC payback period**: <12 months target
- [ ] **LTV:CAC ratio**: >3x target
- [ ] **Gross margin**: >70% SaaS target
- [ ] **Churn analysis**: Cohort retention curves
- [ ] **Upsell/cross-sell**: Effectiveness metrics

### 🏦 Financial Controls
- [ ] **Accounting system**: QuickBooks/Xero setup
- [ ] **Revenue reconciliation**: Monthly process
- [ ] **Expense management**: Policies + controls
- [ ] **Burn rate**: Runway analysis
- [ ] **Cash flow**: Monthly projections reviewed

### 📊 Projections & Metrics
- [ ] **Financial models**: Assumptions documented
- [ ] **Key metrics dashboard**: Real-time tracking
- [ ] **Budget vs actual**: Monthly variance analysis
- [ ] **Cap table**: Current ownership structure
- [ ] **Option pool**: Remaining + exercise history

---

## ⚖️ Legal Due Diligence

### 📄 Corporate Structure
- [ ] **Entity structure**: Subsidiaries + ownership
- [ ] **Corporate records**: Bylaws + minutes updated
- [ ] **Board approvals**: Material decisions documented
- [ ] **Shareholder agreements**: Transfer restrictions
- [ ] **Employee agreements**: Non-competes + IP assignment

### 🏷️ Intellectual Property
- [ ] **Patents**: Filed + granted status
- [ ] **Trademarks**: Registered in key markets
- [ ] **Copyrights**: Software registration
- [ ] **Trade secrets**: Protection mechanisms
- [ ] **Open source**: License compliance audit

### 🤝 Contracts Review
- [ ] **Customer contracts**: Top 20 reviewed
- [ ] **Supplier contracts**: Critical vendors
- [ ] **Partnership agreements**: Terms and exclusivity
- [ ] **Employment contracts**: Key personnel
- [ ] **Leases**: Office + equipment agreements

### ⚠️ Litigation & Compliance
- [ ] **Active litigation**: Cases reviewed by counsel
- [ ] **Past disputes**: Resolution history
- [ ] **Regulatory compliance**: Industry-specific requirements
- [ ] **Data privacy**: GDPR/CCPA compliance
- [ ] **Tax filings**: All jurisdictions up to date

---

## 📊 Market Due Diligence

### 🎯 Customer Analysis
- [ ] **Customer concentration**: Top 10% < 50% revenue
- [ ] **Reference calls**: 3-5 customer interviews
- [ ] **Churn analysis**: Reasons + trends
- [ ] **Expansion potential**: Upsell/cross-sell opportunities
- [ ] **Customer satisfaction**: NPS + survey data

### 🏆 Competitive Landscape
- [ ] **Competitor analysis**: Positioning vs alternatives
- [ ] **Market share**: Estimated + trends
- [ ] **Barriers to entry**: Technical + regulatory
- [ ] **Disruption risks**: New entrants + technologies
- [ ] **Pricing analysis**: Competitor comparison

### 📈 Market Opportunity
- [ ] **Market sizing**: TAM/SAM/SOM validated
- [ ] **Growth trends**: Historical + projected
- [ ] **Regulatory changes**: Impact assessment
- [ ] **Technological shifts**: Adaptation strategy
- [ ] **Customer needs**: Evolution analysis

---

## 🏢 Operational Due Diligence

### 👥 Human Resources
- [ ] **Employee handbook**: Policies + procedures
- [ ] **Performance reviews**: Process + ratings
- [ ] **Compensation structure**: Market competitiveness
- [ ] **Benefits packages**: Health + retirement
- [ ] **Culture assessment**: Team dynamics review

### 📦 Supply Chain & Vendors
- [ ] **Key vendors**: Dependencies + alternatives
- [ ] **Service level agreements**: Performance metrics
- [ ] **Contract terms**: Renewal + termination
- [ ] **Vendor concentration**: Risk assessment
- [ ] **Backup providers**: Contingency planning

### 🏛️ Facilities & Equipment
- [ ] **Office leases**: Terms + renewal options
- [ ] **Equipment assets**: Condition + valuation
- [ ] **IT inventory**: Hardware + software licenses
- [ ] **Disaster recovery**: Plans + tested procedures
- [ ] **Insurance coverage**: Types + limits

---

## 🚨 Critical Red Flags

### Technology
- [ ] **Single point of failure**: No HA/DR
- [ ] **Legacy codebase**: >50% technical debt
- [ ] **Security breaches**: Past incidents
- [ ] **Key dependency**: Single vendor lock-in

### Financial
- [ ] **Negative burn rate**: >18 months runway
- [ ] **Revenue concentration**: >50% from one customer
- [ ] **Contract disputes**: Active litigation
- [ ] **Tax liens**: Outstanding liabilities

### Legal
- [ ] **IP ownership**: Unclear assignment
- [ ] **Regulatory violations**: Compliance issues
- [ ] **Employee lawsuits**: Active claims
- [ ] **Change of control**: Problematic clauses

### Documentation Complete
- [ ] All requested documents provided
- [ ] Data room organized and accessible
- [ ] Management interviews conducted
- [ ] Site visit completed
- [ ] Reference calls completed

### Analysis Complete
- [ ] Risk assessment finalized

- [ ] Legal opinion received
- [ ] Financial model audited

### Decision Ready
- [ ] Investment committee briefing prepared
- [ ] Term sheet draft circulated
- [ ] Stakeholder alignment achieved
- [ ] Contingent risks documented
- [ ] Exit strategy validated
