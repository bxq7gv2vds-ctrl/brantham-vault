## DD Accélérée - Checklist Critique (48h)

### 🔴 Rouge (obligatoire)
- [ ] Bilan 2023 certifié + rapport des commissaires aux comptes
- [ ] Ordonnance procédure complète + mandataire
- [ ] Liste complète créanciers avec hiérarchisation
- [ ] Plan de cession prévisionnel
- [ ] Contrats clients/fournisseurs clés (> 5% CA)
- [ ] Conflits internes identifiés

- [ ] Trésorerie actuelle + flux prévisionnels 3 mois
- [ ] Situation fiscale et sociale régularisée
- [ ] Etat contentieux actifs
- [ ] Baux commerciaux - résiliation/renouvellement
- [ ] Agreements sanitaires/qualité transférables
- [ ] Liste salariés clés - rétention plan

### � Jaune (importante)
- [ ] Business plan post-restructuration
- [ ] Etudes marché concurrentiel
- [ ] Contrats assurance en cours
- [ ] Documentation technique/processus
- [ ] Relations clients/fournisseurs critiques
- [ ] Plan de turnaround (si applicable)

### 🟢 Vert (complémentaire)
- [ ] Documentation marketing/branding
- [ ] Archives contrats historiques
- [ ] Etudes internes R&D
- [ ] Analyse coûts complets
- [ ] Reporting réglementaire sectoriel

## Organigramme de la Data Room

```
🔒 Mandataire Judiciaire
    ├─ 📊 Documents légaux (accès restreint)
    ├─ 💰 Situation financière (accès restreint)
    └─ ⚖️ Contentieux (accès restreint)

👥 Direction
    ├─ 🏭 Opérationnel (accès standard)
    ├─ 💼 Commercial (accès standard)
    └─ 👥 RH (accès standard)

🏛️ Tribunal
    └─ ✅ Plan de cession (accès final)

🤝 Repreneur Potentiel
    └─ 📋 Dashboard de suivi (accès progressif)
```

## Processus d'Update

### Mises à jour quotidiennes
- [ ] Trésorerie
- [ ] Situation procédure
- [ ] Nouveaux contentieux

### Mises à hebdomadaires
- [ ] Reporting opérationnel
- [ ] Avancée négociations
- [ ] Documentation complémentaire

### Versionning
- Versionner chaque document avec date
- Conserver historique des modifications
- Journal d'accès aux documents sensibles

## Security & Compliance

### Accès
- 🔒 Niveau 1: Admin (Brantham + Mandataire)
- 🔒 Niveau 2: Standard (Repreneurs + Experts)
- 🔒 Niveau 3: Final (Tribunal only)

### Audit
- Journal des connexions
- Watermarking documents sensibles
- Time-limited access links
