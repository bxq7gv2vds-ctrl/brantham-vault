# Pitch Investisseurs - Deals M&A Distressed

## 🎯 Objectif

**Objectif** : Présenter une opportunité M&A distressed aux investisseurs en mettant en avant le profil de risque/rendement attractif basé sur les patterns de réussite observés.

**Public** : Investisseurs privés, family offices, fonds d'investissement spécialisés

---

## 📊 Structure du Pitch - 10 Slides Essentielles

### Slide 1 : Titre Accrocheur
```
[Repreneur] x [Secteur Leader] : 
Opportunité M&A Distressed avec [Multiple]x EBITDA
+ [Chiffre]% de croissance organique prévue

Basé sur le succès de [Nombre] reprises réussies
```

**Points clés** :
- Repreneur expérimenté dans le secteur
- Opportunité distressed avec timing court
- Profil risque/rendement quantifié

---

### Slide 2 : Le Contexte Distressed
```
[Nom Entreprise] : Chiffres Clés en Distress

• CA : [Montant] (-[Chiffre]% vs. pic)
• EBITDA : [Montant] (-[Chiffre]% marge)
• Dettes : [Montant] ([Multiple]x EBITDA)
• Effectif : [Nombre] (-[Chiffre]% depuis 2023)
```

**Message** :
- Opportunité timing parfait (market bottom)
- Distress sectoriel confirmé
- Valeur sous-jacente identifiable

---

### Slide 3 : Le Repreneur - Profil Confirmed
```
Track Record Sectoriel + Vision Industrielle

• [Nombre] acquisitions réussies dans [Secteur]
• [Chiffre]% de croissance réalisée
• Expertise opérationnelle prouvée
• Réseau d'experts validé
```

**Patterns à mettre en avant** :
- Repreneur récurrent (MUVICO pattern)
- Vision industrielle vs. pure financière
- Equipe technique intégrée

---

### Slide 4 : La Stratégie de Reprise
```
Stratégie "Fix & Flip" à [Période] mois

Phase 1 : Stabilisation (J+1 à J+30)
• [Action] → [Objectif] → [Budget]

Phase 2 : Intégration (J+31 à J+90)
• [Action] → [Objectif] → [Budget]

Phase 3 : Croissance (J+91 à J+180)
• [Action] → [Objectif] → [Budget]
```

**Focus** :
- Plan exécutable avec milestones
- Budget de restructuration visible
- Timeline réaliste (<6 mois)

---

### Slide 5 : Les Synergies Quantifiées
```
Synergies Opérationnelles Annuelles

• Achats groupés : -[Chiffre]% coûts
• Cross-selling : +[Chiffre]% CA
• Optimisation énergétique : -[Chiffre]%
• Digitalisation : -[Chiffre]% coûts fixes

Total synergies : [Montant]/an ([Chiffre]% du CA)
```

**Preuves** :
- Chiffres basés sur précédentes acquisitions
- Timeline de réalisation (3-12 mois)
- Risques de non-réalisation mitigés

---

### Slide 6 : Le Deal Structure
```
Structure d'Investissement Optimisée

• Prix d'acquisition : [Montant]
• Capex travaux : [Montant]
• Dette structurelle : [Montant]
• Apport personnel : [Montant]

Multiple payback : [Nombre] ans
ROI annualisé : [Chiffre]%
```

**Key Terms** :
- Prix vitrine + Capex visible (AJA pattern)
- Conditions suspensives limitées
- Couverture du passif explicite

---

### Slide 7 : Le Tribunal de Commerce
```
Stratégie Judiciaire Confirmée

• Administrateur Judiciaire : [Nom] - [Expérience] ans
• Procédure : [Type] - [Durée] moyenne
• Compétiteurs : [Nombre] offres prévues
• Probabilité de succès : [Chiffre]%

Calendrier : Jugement prévu le [Date]
```

**Risks Management** :
- Contact direct avec l'AJ
- Documentation complète (attestation bancaire)
- Présence physique à l'audience

---

### Slide 8 : Les Risques Quantifiés
```
Profil Risque/Rendement

Rouges (mitigés) :
• Passif caché : [Chiffre]% probabilité
• Perte clients : [Chiffre]% max
• Retard synergies : [Chiffre] mois max

Jaunes (monitorés) :
• Exécution opérationnelle
• Marché concurrentiel
• Financement conditions

ROE annualisé : [Chiffre]%
```

**Mitigations** :
- Due diligence complète
- Plan B pour chaque risque majeur
- Team expérimentée

---

### Slide 9 : L'Investissement - Term Sheet
```
Proposition d'Investissement

• Montant à lever : [Montant]
• Dilution proposée : [Chiffre]%
• Valorisation pré-money : [Montant]
• Durée : [Nombre] ans avec sortie

• [Clause 2]
• [Clause 3]
```

**Deal Points** :
- Governance standard
• Tag-along/drag-along équilibrés

---

### Slide 10 : Prochaines Étapes
```
Roadmap d'Investissement

Semaine 1 : Due diligence accélérée
Semaine 2 : Term sheet signé

Semaine 4 : Début opérations

Contact : [Nom] - [Email] - [Téléphone]
```

**Call to Action** :
- Investissement prioritaire

- Track record prouvé

## 📊 Script de Pitch - 5 Minutes Clés

### Ouverture (30 secondes)
"Nous identifions une opportunité M&A distressed dans le [Secteur] avec un profil risque/rendement attractif. Le repreneur a déjà réalisé [Nombre] acquisitions similaires avec un succès de [Chiffre]%."

### Le Deal (2 minutes)
"Le cible génère [Montant] de CA mais est en distresse suite à [Cause]. Notre stratégie de reprise prévoit [Nombre] mois pour stabiliser, intégrer et créer de la valeur avec synergies de [Montant]/an."

### Le ROI (1 minute)
"La structure d'investissement offre un ROE annualisé de [Chiffre]% avec un payback en [Nombre] ans. Le risque est quantifié et mitigé par notre expérience sectorielle et notre due diligence approfondie."

### L'Appel (1 minute)
"Nous levons [Montant] pour atteindre une valorisation de [Montant]. L'opportunité est limitée car le calendrier judiciaire impose une décision sous [Nombre] semaines. Votre investissement nous positionne pour une sortie lucrative."

## 📊 Données Clés à Personnaliser

### Sector Data
```
Retail traditionnel :
• Multiple payback : 2-3 ans
• Risque clients : 15-20%
• Synergies achats : 10-15%

B2B <50 salariés :
• Multiple payback : 3-4 ans
• Risque tech : 25-30%
• Synergies croisées : 20-25%

Foncière IDF :
• Multiple payback : 1-2 ans
• Risque immobilier : 10-15%
• Synergies locatives : 25-30%
```

### Success Metrics
- ROI annualisé >20%
- Payback <4 ans
- Probabilité de succès >80%
- Synergies réelles >80% planifié

## 📊 Common Questions & Answers

### Q1 : "Pourquoi cette entreprise est-elle en distress ?"
**Réponse** : "[Cause spécifique] a créé une situation de trésorerie critique, mais l'activité sous-jacente reste saine avec des marges de [Chiffre]% avant la crise."

### Q2 : "Comment garantir le succès du Tribunal ?"
**Réponse** : "Notre offre inclut un plan industriel 5 ans détaillé, l'emploi préservé à >90%, et une structure financière documentée. Le repreneur a un track record dans le secteur."

### Q3 : "Quel est le risque maximum ?"
**Réponse** : "Le risque majeur est le passif caché, que nous couvrons par une due diligence complète et une clause de résolution. Le risque maximal quantifié est de [Chiffre]% du montant investi."
