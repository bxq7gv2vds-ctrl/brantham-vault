# Checklist data room, [NOM DE CODE DEAL]

**Objet** : Liste exhaustive des pièces à demander à l'administrateur judiciaire et à analyser en data room.
**Périmètre** : Reprise à la barre, plan de cession L.642-1 du Code de commerce.
**Date d'ouverture data room** : [JJ/MM/AAAA]
**Échéance de dépôt d'offre** : [JJ/MM/AAAA]
**Suivi par** : Brantham Partners, Paul Roulleau.
**Confidentialité** : Pièces couvertes par l'engagement de confidentialité signé auprès de l'AJ. Diffusion interne repreneur uniquement.

> **Mode d'emploi.** Calibrée sur la structure réelle d'une data room de cession PME (dossier ingebime, 53 pièces). Colonne **Obtenu** : O si la pièce est en data room, N sinon, demander explicitement à l'AJ. Colonne **Criticité** : Haute (deal-breaker potentiel), Moyenne, Basse. Une pièce critique manquante se reporte dans la section 11 du mémo de due diligence. Toute donnée dépendante de la data room est taguée **[À CONFIRMER DR]** tant que la pièce n'est pas reçue.
