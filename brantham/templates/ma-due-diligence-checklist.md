# M&A Due Diligence Checklist - Enhanced

## 📋 Quick Scan Initial (2h max)

### 🔍 Identification entreprise
- [ ] **SIREN/SIRET** validé via Pappers
- [ ] **Forme juridique** exacte et statut au RCS
- [ ] **Date création** et durée d'existence
- [ ] **Capital social** et titres actuels
- [ ] **Objet social** vs activité réelle (match)
- [ ] **Nombre d'établissements** (siège + annexes)

### ⚖️ Procédure collective en cours
- [ ] **Type procédure** (RJ/LJ/pré-pack)
- [ ] **Tribunal** compétent et jugement d'ouverture
- [ ] **Administrateur Judiciaire** nommé + contact
- [ ] **Date limite dépôt offres** (DLDO)
- [ ] **MJ** en charge (mention BODACC)
- [ ] **État des créances** (niveau d'impayés)

### 📊 Indicateurs clés instantanés
- [ ] **CA dernière année** disponible
- [ ] **Nombre salariés** actuel
- [ ] **Dettes totales** estimées
- [ ] **Actifs principaux** (immobilier, équipement)
- [ ] **Top 3 clients** (% CA)

## 🎯 Due Diligence Approfondie (Pattern-Based)

### 1. **Analyse Risques Kill-Switch** (Must Have)
- [ ] **Périmètre cohérent** (pas de cherry-picking)
- [ ] **Plan industriel crédible** (pas de promesses vagues)
- [ ] **Financement documenté** (attestation bancaire + preuve)
- [ ] **Structure prix défendable** (pas de placeholders)
- [ ] **Passif repris explicité** (pas de charges masquées)

### 2. **Due Diligence Contractuelle** 🔐
- [ ] **Top 10 clients** : contrats analysés, dépendance, risques
- [ ] **Top 10 fournisseurs** : créances en cours, pénalités rupture
- [ ] **Contrats long terme** : durée, clauses de résiliation, pénalités
- [ ] **Accords non-concurrence** : en cours, durée, zone
- [ ] **Propriété intellectuelle** : brevets, marques, licences (valables)
- [ ] **Accords distribu** : exclusivité, territoire, minimum CA

### 3. **Analyse Juridique Avancée** ⚖️
- [ ] **Contentieux en cours** (tribunal, administratif, social)
- [ ] **Recours fiscaux** et risques associés
- [ ] **Conformité environnementale** : autorisations, pénalités
- [ ] **Concurrence déloyale** : plaintes passées, risques actuels
- [ ] **Droit du travail** : inspection du travail, contentieux
- [ ] **Procédures collectives antérieures** (si 2+ RJ/LJ)

### 4. **Due Diligence Financière Deep Dive** 💰
- [ ] **Comptes sociaux** 3 ans (analyse BODACC)
- [ ] **Évolution CA** : croissance organique vs externe
- [ ] **Marge brute** : analyse par segment produit
- [ ] **BFR** : évolution, couverture, cycles
- [ ] **Dettes financières** : échéances, garanties, covenants
- [ ] **Trésorerie** : flux de trésorerie à 12 mois
- [ ] **Actifs incorporels** : valorisation, amortissements

### 5. **Social & RH** 👥
- [ ] **Effectif** : évolution 3 ans, turnover, absentéisme
- [ ] **CCN applicable** : clauses spécifiques, échéance renégociation
- [ ] **Comité social** : en place, représentants, tensions
- [ ] **PSE existant** : en cours, impacts financiers
- [ ] **Clés de personnel** : turnover, dépendance
- [ ] **Obligations légales** : DUERP, pénalités, contrôles

### 6. **Marché & Concurrence** 🏢
- [ ] **Part de marché** estimée (source récente)
- [ ] **Positionnement** prix vs concurrents
- [ ] **Barrières à l'entrée** : réglementaires, technologiques
- [ ] **Tendances sectorielles** : disruption, contraction
- [ ] **Clients clés** : risques de perte, dépendance
- [ ] **Fournisseurs clés** : ruptures possibles, pénalités

### 7. **Synergies & Potentiel** 🚀
- [ ] **Synergies réalistes** : quantifiées, calendrier
- [ ] **Réduction coûts** : immédiate vs progressive
- [ ] **Croissance externe** : accessible via acquisition
- [ ] **Optimisation opérationnelle** : gains potentiels
- [ ] **Intégration technologique** : compatibilité, coûts

## 🔴 Red Flags Immédiats (Rejet Automatique)
- [ ] **Périmètre partiel** sans justification stratégique
- [ ] **Prix symbolique** sans plan industrial concret
- [ ] **Financement non documenté** ou contingent
- [ ] **P&E massif** sans cash-flow de couverture
- [ ] **Concurrence déloyale** non déclarée
- [ ] **Dépendance client/fournisseur** > 30% sans plan

## 🎯 Patterns d'Offre Gagnante (Intégrer)

1. **Prix vitrine + Capex affichage** : Prix bas + enveloppe travaux visible
2. **Plan industriel 5 ans** : Chiffres détaillés + storytelling humain
3. **Repreneur sectoriel** : Track record dans le même secteur
4. **Engagements sociaux** : Emploi protégé + plan de formation
5. **Structure juridique propre** : SCI pour immobilier, SAS pour opérationnel
