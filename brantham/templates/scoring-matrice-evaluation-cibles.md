## 🎯 Diagnostic Rapide par Type de Cible

### Retail Multi-Sites
- **Poids fort** : Périmètre cohérent (critère 6) >30%
- **Red flag** : Dépendance client >30% sans plan (critère 7)
- **Score cible** : >70 avec synergie réseau

### B2B <50 salariés  
- **Poids fort** : Plan industriel crédible (critère 10) >25%
- **Red flag** : Passif masqué fournisseurs (critère 7)
- **Score cible** : >65 avec track record sectoriel

### Tech/Innovation
- **Poids fort** : Propriété intellectuelle (critère 9) >20%
- **Red flag** : Indivisibilité technique (critère 6)
- **Score cible** : >75 avec brevets validés

### Foncière
- **Poids fort** : Structure juridique (critère 8) >25%
- **Red flag** : Charges masquées locataires (critère 7)
- **Score cible** : >70 avec capex travaux affiché

## 📝 Notes Évaluateur

### Signal positif fort (>+10 pts)
- 🟢 Repreneur récurrent sur 2+ deals (MUVICO pattern)
- 🟢 Offre partie mais lots cohérents (NEOCAMP pattern)
- 🟢 Engagements sociaux > moyenne sectorielle
- 🟢 Documentation financement bancaire complète

### Signal négatif fort (-15 pts)  
- 🔴 Périmètre partiel sans cohérence (MINELLI case)
- 🔴 Promesses "ultérieurement" (CASTEL pattern)
- 🔴 Indivisibilité "château de cartes"
- 🔴 Conditions suspensives auto-contradictoires

### Formule Scoring Pondéré
```
Score Final = (Σ Critères × Pondération) / 10
```

**Règles décisions** :
- **<50 pts** : Offre non compétitive → abandon
- **50-65 pts** : Offre possible mais risque → amélioration requise  
- **>65 pts** : Offre compétitive → préparation dépôt
- **>80 pts** : Offre premium → dépôt prioritaire

## 🎯 Patterns d'Offre Gagnante à Intégrer

1. **Prix vitrine + Capex affichage** : Prix bas + enveloppe travaux visible (AJA)
2. **Plan industriel 5 ans** : Chiffres détaillés + storytelling sectoriel  
3. **Repreneur sectoriel** : Track record dans même secteur
4. **Engagements sociaux** : Emploi protégé + plan formation
5. **Structure juridique propre** : Adaptée au type d'actifs
