# Multiples Sectoriels M&A Distressed Benchmark

## 📊 Overview

This benchmark provides adjusted multiples for distressed M&A transactions across sectors, normalized for distress premium, market conditions, and sector-specific factors.

## 💡 Key Methodology

### Distressed Multiple Adjustments
```markdown
**Base Multiple × Distress Factor × Sector Factor × Market Condition Factor = Final Multiple**

Distress Factors:
- Mild distress: 0.8-0.9 × base
- Moderate distress: 0.6-0.8 × base  
- Severe distress: 0.4-0.6 × base
- Critical distress: 0.2-0.4 × base

Sector Adjustment:
- Tech: +0.1-0.2 (option value)
- Manufacturing: -0.1-0.2 (asset value)
- Services: 0 (standard)
- Commodities: -0.2-0.3 (cyclical)
```

## 📊 Application Framework

### Quick Valuation Template
```yaml
# Input Fields
sector: [Technology/Manufacturing/Healthcare/E-commerce/Financial/Real Estate]
base_multiple: [X]x
distress_level: [Mild/Moderate/Severe/Critical]
market_condition: [Bull/Bear/Neutral]
recovery_potential: [High/Medium/Low]

# Calculation
distress_factor: 
  Mild: 0.7-0.8
  Moderate: 0.4-0.7  
  Severe: 0.2-0.5
  Critical: 0.1-0.3

sector_factor:
  Technology: +0.15
  Manufacturing: -0.15
  Healthcare: +0.1
  E-commerce: 0
  Financial: 0
  Real Estate: -0.1

market_factor:
  Bull: +0.15
  Neutral: 0
  Bear: -0.15

final_multiple = base_multiple × distress_factor × (1 + sector_factor) × (1 + market_factor)

# Recovery Premium
recovery_multiplier:
  High: 2.0-2.5
  Medium: 1.5-2.0
  Low: 1.2-1.5

target_multiple = final_multiple × recovery_multiplier
```
