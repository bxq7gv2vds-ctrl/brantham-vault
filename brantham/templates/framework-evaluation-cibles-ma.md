## 🎯 Grille d'Évaluation Cible (0-100 points)

### 🏆 Potentiel de Croissance (30 points)

#### Croissance Historique (10 points)
- [ ] CA > 15% YoY sur 3 ans → 10 pts
- [ ] CA > 10% YoY sur 3 ans → 7 pts
- [ ] CA > 5% YoY sur 3 ans → 4 pts
- [ ] CA stable → 2 pts
- [ ] CA décroissant → 0 pts

#### Potentiel de Marché (10 points)
- [ ] TAM > 1B€ → 10 pts
- [ ] TAM 500M€-1B€ → 7 pts
- [ ] TAM 100M€-500M€ → 4 pts
- [ ] TAM < 100M€ → 2 pts
- [ ] Marché saturé → 0 pts

#### Innovation & Scalabilité (10 points)
- [ ] Innovation disruptive → 10 pts
- [ ] Innovation modérée → 7 pts
- [ ] Innovation incrémentale → 4 pts
- [ ] Pas d'innovation → 0 pts

---

### 💰 Potentiel Financier (25 points)

#### Rentabilité (10 points)
- [ ] EBITDA > 20% → 10 pts
- [ ] EBITDA 15%-20% → 7 pts
- [ ] EBITDA 10%-15% → 4 pts
- [ ] EBITDA < 10% → 2 pts
- [ ] Société non profitable → 0 pts

#### Structure de Coûts (8 points)
- [ ] Structure optimisée → 8 pts
- [ ] Structure standard → 5 pts
- [ ] Structure coûteuse → 3 pts
- [ ] Structure inefficace → 0 pts

#### Qualité du Bilan (7 points)
- [ ] Trésorerie positive, dettes < 2x EBITDA → 7 pts
- [ ] Trésorerie neutre, dettes 2-3x EBITDA → 4 pts
- [ ] Trésorerie négative, dettes > 3x EBITDA → 0 pts

---

### ⚖️ Risques (20 points)

#### Risques Juridiques (7 points)
- [ ] Aucun risque juridique → 7 pts
- [ ] Risques mineurs contrôlés → 4 pts
- [ ] Risques significatifs → 2 pts
- [ ] Risques majeurs → 0 pts

#### Risques Opérationnels (7 points)
- [ ] Aucune dépendance → 7 pts
- [ ] Dépendance limitée → 4 pts
- [ ] Dépendance significative → 2 pts
- [ ] Dépendance critique → 0 pts

#### Risques Marché (6 points)
- [ ] Position dominante → 6 pts
- [ ] Position solide → 4 pts
- [ ] Position moyenne → 2 pts
- [ ] Position faible → 0 pts

---

### 🔄 Synergies Brantham (15 points)

#### Synergies Commerciales (5 points)
- [ ] Clients cible communs → 5 pts
- [ ] Marchés adjacents → 3 pts
- [ ] Aucune synergie → 0 pts

#### Synergies Opérationnelles (5 points)
- [ ] Intégration immédiate → 5 pts
- [ ] Intégration complexe → 3 pts
- [ ] Aucune synergie → 0 pts

#### Synerg Technologiques (5 points)
- [ ] Complémentarité parfaite → 5 pts
- [ ] Complémentarité partielle → 3 pts
- [ ] Aucune synergie → 0 pts

---

### 👑 Équipe & Direction (10 points)

#### Expérience Management (5 points)
- [ ] Team expérimenté, track record → 5 pts
- [ ] Team mixé expérience/junior → 3 pts
- [ ] Team junior → 1 pt

#### Stabilité Équipe (5 points)
- [ ] Turnout < 10% → 5 pts
- [ ] Turnout 10-20% → 3 pts
- [ ] Turnout > 20% → 0 pts
