# Template de Scoring Cible M&A

**Usage** : Évaluation rapide et standardisée des opportunités M&A distressed. Base sur le système à 9 composantes pondérées et la grille d'offre TC Paris.

---

## Section 1 - Qualification Rapide (Score 0-100)

### Métriques clés à compléter

| Composante | Poids | Note (0-100) | Pondéré | Source |
|------------|-------|--------------|---------|--------|
| **1. Taille** | 30% | | CA, effectif, total bilan | Pappers/BODACC |
| **2. Secteur** | 25% | | Attractivité NAF (industrie>services) | SIRENE/INSEE |
| **3. Procédure** | 20% | | Type (LJ>RJ>SV) + stade | BODACC/tribunal |
| **4. Fraîcheur** | 5% | | Jours depuis jugement (<30j=100) | BODACC |
| **5. Localisation** | 5% | | Bassin économique, densité | INSEE |
| **6. Effectif** | 5% | | 10-100 salariés optimal | SIRENE |
| **7. AFDCC** | 5% | | Score défaut (si dispo) | AFDCC |
| **8. Mandataire** | 3% | | Track record taux cession | AJ historique |
| **9. Actifs** | 2% | | Qualité valeur identifiés | Etude |

### Score Qualification = **__/100**

**Seuil décision** :
- **75+** : URGENT - Lancement pipeline complet
- **60-74** : LAUNCH - Pipeline standard  
- **50-59** : WATCH - Surveillance
- **<50** : ARCHIVE - Non qualifié
