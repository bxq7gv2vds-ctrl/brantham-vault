# Session 2026-03-22 — Teaser One-Pager

## Ce qui a ete fait

### Teaser One-Pager HTML/PDF

Cree un template de teaser d'entreprise one-pager aligne sur la direction artistique du site branthampartners.fr. Plusieurs iterations pour trouver le bon equilibre entre densite d'information et clarte visuelle.

**Iterations :**
1. Portrait A4 — trop basique, pas assez data-driven
2. Landscape 4 colonnes — trop dense, trop de couleurs/badges/polices
3. Landscape 3 colonnes avec modules + pills colores — trop chargé visuellement
4. Landscape 2 colonnes simplifie — trop simple
5. **Version finale** : Landscape 3 colonnes, modules avec headers gris, 1 couleur accent (navy), 2 tailles de texte. Bon equilibre.

**Fichiers crees :**
- `api/templates/teaser-onepager.html` — Template avec placeholders
- `api/templates/teaser-onepager-example.html` — Exemple avec deal Artemis
- `api/generate_teaser_onepager.py` — Script Python : genere HTML + PDF via Chrome headless
- `api/output/teaser-onepager-example.pdf` — PDF genere

**Approche PDF :** Python genere le HTML avec les donnees du deal, puis Chrome headless le convertit en PDF. Rendu pixel-perfect vs PPTX qui etait "horrible".

**Portrait teste** : `api/templates/teaser-onepager-portrait.html` — fonctionnel mais l'utilisateur prefere le landscape.

### Pattern enregistre

- `vault/brantham/patterns/teaser-onepager-html-pdf.md` — Documentation complete du pattern

## Decisions

- HTML + Chrome headless > PPTX pour les one-pagers (rendu superieur)
- Landscape A4 > Portrait (plus de place pour les 3 colonnes de data)
- Design restraint : 1 couleur accent (navy), 2 tailles de texte, pas de badges colores
