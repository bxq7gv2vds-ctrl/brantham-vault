## Findings clés

### 🔴 Red flags

1. **Effondrement CA** : -37% 2024→2025 = signal perte client majeure OU marché s'écroule
2. **Pasif social critique** : salaires impayés depuis 01/01/2026 (~600k€ estimé)
3. **BFR destructeur** : machine spéciales = cycle 2-3 mois avant paiement → insolvabilité structurelle
4. **Zero client disclosure** : data room muet sur top clients (concentration risk invisible)

### ✅ Green flags

1. **Actifs réalisables** : immobilier neuf 1.5M€, machines 383k€ + WIP 540k€
2. **Équipe stable** : 28 ans, mêmes personnes depuis longtemps
3. **Secteur défensif** : machines spéciales = demand endogène (industrie 4.0, nearshoring)
4. **Procédure clean** : pas de litiges, administrateur/mandataire compétents

### ⚠️ Blind spots

- Cause CA-37% : client concentration ? perte secteur ? marché ? → **CRITICAL TO DIAGNOSE**
- Qualité créances clients (823k€) : saines ou douteuses ?
- Levier immobilier : hypothèques cachées ?
- Volonté management : Olivier THEIL veut reprendre ou sort ?
