# Skill: Legal Analysis

## Quand l'utiliser

Utiliser cette skill pour analyser contrats, statuts, baux, contentieux, mandats, LOI, NDA, SPA et risques juridiques.

## Inputs necessaires

- documents juridiques ;
- contexte de transaction ;
- juridiction ;
- questions specifiques.

## Etapes

1. Lire `06_Playbooks/Analyse_Juridique.md`.
2. Classer les documents.
3. Extraire parties, dates, obligations, clauses sensibles.
4. Identifier risques et incoherences.
5. Lister validations avocat/humain.

## Sorties

- note juridique Markdown ;
- matrice clauses sensibles ;
- tableau risques ;
- questions avocat/conseil ;
- output JSON cowork.

## Regles qualite

- L'analyse ne remplace pas un avis juridique.
- Toujours signaler incertitudes et limites.
- Ne jamais inventer clause, obligation, partie ou date.
