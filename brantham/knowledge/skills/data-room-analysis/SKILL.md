# Skill: Data Room Analysis

## Quand l'utiliser

Utiliser cette skill pour analyser une data room, produire une synthese de due diligence et identifier risques, incoherences et questions ouvertes.

## Inputs necessaires

- index de data room ;
- documents financiers, juridiques, sociaux, fiscaux, commerciaux ;
- contexte de transaction ;
- questions prioritaires.

## Etapes

1. Lire `06_Playbooks/Analyse_Data_Room.md`.
2. Inventorier les documents.
3. Classer par categorie.
4. Extraire les faits.
5. Identifier incoherences, manquants et risques.
6. Produire matrice de risques et questions.
7. Demander QC si livrable client.

## Sorties

- note Markdown ;
- tableau documents ;
- matrice risques ;
- questions de due diligence ;
- output JSON cowork.

## Regles qualite

- Ne pas conclure au-dela des documents disponibles.
- Toujours citer fichier/page/section si disponible.
- Signaler les incertitudes et limites.
