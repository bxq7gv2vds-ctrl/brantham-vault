# Skill: Outreach Writer

## Quand l'utiliser

Utiliser cette skill pour rediger emails d'approche, relances, introductions, demandes d'information et syntheses de suivi.

## Inputs necessaires

- destinataire ;
- objectif ;
- contexte ;
- niveau de confidentialite ;
- ton ;
- informations autorisees.

## Etapes

1. Lire `06_Playbooks/Redaction_Emails.md`.
2. Identifier ce qui peut etre partage.
3. Proposer un angle.
4. Rediger sujet + email.
5. Lister validations avant envoi.
6. Demander QC pour emails sensibles.

## Sorties

- sujet ;
- email brouillon ;
- variantes si utile ;
- notes de personnalisation ;
- validations humaines.

## Regles qualite

- Ne jamais envoyer automatiquement.
- Ne jamais promettre une condition non validee.
- Ne jamais inclure donnees sensibles sans validation.
