# Agriculture et Agroalimentaire -- Playbook Sectoriel M&A Distressed

> Fiche operationnelle Brantham Partners. Derniere MAJ : 27 mars 2026.

---

## 1. Panorama sectoriel

### Chiffres cles defaillances

| Indicateur | Valeur 2025 | Evolution |
|---|---|---|
| Defaillances agriculture | **~1 500** | En hausse |
| Defaillances agroalimentaire (IAA) | **~1 000-1 200** | En hausse |
| Part des defaillances totales | **~2%** (agriculture) + **~1,5%** (IAA) | Stable |
| Exploitations agricoles en difficulte | ~3 000/an (estimation FNSEA) | Hausse structurelle |

### Contexte macro

- **Meteo** : annees de secheresse/gel a repetition. Recoltes 2024-2025 affectees (cereales, viticulture).
- **Prix agricoles** : volatilite extreme. Les prix a la production ne couvrent pas toujours les couts.
- **PAC** : nouvelle PAC 2023-2027. Redistribution des aides, eco-conditionnalite renforcee.
- **Pression reglementaire** : Egalim 2, zones de non-traitement (ZNT), reduction phytosanitaires.
- **Concurrence internationale** : Mercosur, Ukraine = pression sur les prix des cereales et de la viande.
- **Surendettement** : endettement moyen des exploitations = 200-400K EUR (MSA).

### Sous-segments

| Segment | Risque | Facteur |
|---|---|---|
| Grandes cultures (cereales) | Modere | Volatilite prix, meteo, PAC |
| Elevage bovin lait | Eleve | Prix du lait < cout de production, normes bien-etre animal |
| Elevage bovin viande | Eleve | Meme probleme + concurrence importations |
| Elevage porcin | Modere | Cycles, concurrence espagnole/allemande |
| Viticulture | Eleve | Surproduction, baisse consommation, arrachage |
| Maraichage/arboriculture | Modere | Circuits courts en croissance |
| **IAA (Industrie Agroalimentaire)** | Modere-Eleve | Couts matieres + energie + reglementation |
| IAA -- Abattoirs | Eleve | Concentration, normes sanitaires, investissements lourds |
| IAA -- Laiteries | Modere | Contrats long terme, export |

## 2. Specificites sectorielles

### Juridiction : tribunal judiciaire (pas tribunal de commerce)

**Particularite majeure** : les exploitants agricoles (personnes physiques ou societes civiles agricoles) relevent du **tribunal judiciaire**, pas du tribunal de commerce, meme si la structure est commerciale dans les faits.

| Structure | Juridiction | Fondement |
|---|---|---|
| Exploitant individuel | Tribunal judiciaire | L351-1 Code rural |
| GAEC | Tribunal judiciaire | Societe civile |
| EARL | Tribunal judiciaire | Societe civile |
| SCEA | Tribunal judiciaire | Societe civile |
| SARL/SAS agricole | Tribunal de commerce | Forme commerciale (exception) |
| IAA (societe commerciale) | Tribunal de commerce | Activite industrielle/commerciale |

**Impact** : les procedures devant le TJ sont souvent plus lentes et moins familiarisees avec le droit des entreprises en difficulte que les TC. Les juges sont des magistrats professionnels (pas des juges consulaires).

### Procedures specifiques agriculture

| Procedure | Specificite agricole |
|---|---|
| **Reglement amiable agricole** (L351-1 Code rural) | Specifique a l'agriculture. Le president du TJ designe un **conciliateur** (comparable a la [[brantham/knowledge/procedures/mandat-ad-hoc-conciliation|conciliation]]) pour negocier un echeancier avec les creanciers. Dure 2 mois (+ 1 mois renouvelable). |
| **[[brantham/knowledge/procedures/sauvegarde]] / [[brantham/knowledge/procedures/redressement-judiciaire|RJ]] / [[brantham/knowledge/procedures/liquidation-judiciaire|LJ]]** | Procedures classiques du Livre VI du Code de commerce, applicables aux agriculteurs. |
| **Commission departementale d'orientation agricole** | Avis sur la reprise de l'exploitation. |
| **Aide a la relance** (DDTM) | Subvention plafonnee a 10 800 EUR (1ere UTH) + 10 000 EUR/UTH supplementaire. |

**Point cle** : le reglement amiable agricole est une procedure **preventive** specifique au secteur. Elle permet de negocier avec les creanciers (banques, fournisseurs, MSA) avant toute procedure collective. C'est l'equivalent de la conciliation pour les entreprises commerciales.

### Structures agricoles (GAEC, EARL, SCEA)

| Structure | Particularite | Impact M&A |
|---|---|---|
| **GAEC** (Groupement Agricole d'Exploitation en Commun) | 2-10 associes exploitants. Principe de transparence (chaque associe = 1 exploitation). | Reprise = accord de tous les associes ou rachat de parts. Les aides PAC sont individualisees par associe. |
| **EARL** (Exploitation Agricole a Responsabilite Limitee) | 1-10 associes (exploitants ou non). Responsabilite limitee aux apports. | Plus simple a ceder. Associes non exploitants possibles (investisseurs). |
| **SCEA** (Societe Civile d'Exploitation Agricole) | Pas de limite d'associes. Responsabilite indefinie. | Attention a la responsabilite des associes. |
| **Exploitation individuelle** | Personne physique. Patrimoine personnel = patrimoine professionnel (sauf EIRL). | Le repreneur achete le fonds agricole, pas les dettes personnelles. |

### PAC et DPB (Droits a Paiement de Base)

Les aides PAC representent souvent **30-50% du revenu agricole** :

| Element | Detail |
|---|---|
| **DPB** | Droits a paiement de base. Attaches a l'exploitant ET aux hectares. |
| **Transfert** | Les DPB sont transferes **avec les terres** (vente ou location). Le repreneur qui reprend les terres recupere les DPB. |
| **Valeur** | 100-300 EUR/ha selon la region. Represente 10-30% de la valeur fonciere. |
| **Eco-regime** | Aide complementaire conditionnee a des pratiques environnementales (PAC 2023-2027). |
| **En procedure collective** | Les DPB sont des actifs incorporels cessibles. Le liquidateur les cede avec le foncier ou separement. |

**Point cle** : les DPB ne sont pas automatiquement transferes au repreneur. Il faut declarer le transfert a la DDT(M) et respecter les conditions (exploitation effective des terres). Sans DPB, l'exploitation perd 30-50% de ses revenus.

### SAFER (Societe d'Amenagement Foncier et d'Etablissement Rural)

| Mecanisme | Impact |
|---|---|
| **Droit de preemption** | La SAFER peut preempter toute vente de terres agricoles (et de parts de societe agricole > seuil). |
| **Delai** | 2 mois a compter de la notification de la vente. |
| **En procedure collective** | Le droit de preemption SAFER s'applique MEME en procedure collective et en plan de cession. |
| **Prix** | La SAFER peut preempter au prix de la vente ou a un prix different (revision par le juge de l'expropriation). |
| **Strategie** | Contacter la SAFER en amont pour eviter les surprises. La SAFER peut aussi etre un allie (mise en relation avec des repreneurs installes). |

**Point cle** : le droit de preemption SAFER est un **risque majeur** en M&A agricole. Meme si le tribunal valide un plan de cession, la SAFER peut intervenir et racheter les terres a la place du repreneur designe.

### AOP/AOC et labels

| Label | Importance | Impact cession |
|---|---|---|
| **AOP/AOC** | Appellation d'Origine Protegee/Controlee | Attachee au terroir, pas a l'exploitant. Le repreneur peut continuer si le terroir et les pratiques sont conformes. |
| **IGP** | Indication Geographique Protegee | Plus souple. Cahier des charges a respecter. |
| **Label Rouge** | Qualite superieure | Certification a maintenir (audits). |
| **Agriculture Biologique (AB)** | Certification bio | Periode de conversion (2-3 ans) si l'exploitation n'est pas certifiee. Si elle l'est, le repreneur peut continuer sous controle. |

## 3. Specificites agroalimentaire (IAA)

### Certifications et tracabilite

Les certifications sont des **prerequisites obligatoires** pour acceder a la grande distribution :

| Certification | Exigee par | Renouvellement |
|---|---|---|
| **IFS Food** (International Featured Standard) | Grande distribution France, Allemagne, Italie | Audit annuel |
| **BRC** (British Retail Consortium) | Grande distribution UK, international | Audit annuel |
| **ISO 22000** | Standard international securite alimentaire | Audit annuel |
| **HACCP** | Obligatoire (reglementaire) | Continu |
| **FSSC 22000** | Equivalent GFSI | Audit triennal |

**Impact M&A** : la perte d'une certification IFS/BRC = perte des contrats grande distribution = mort de l'entreprise. Le repreneur doit s'assurer que la certification est maintenue pendant la transition (audit de transfert sous 3-6 mois).

### Tracabilite

- Obligation reglementaire (reglement CE 178/2002) : tracabilite ascendante et descendante a chaque etape.
- En cas de crise sanitaire (rappel produit, contamination) : le systeme de tracabilite doit permettre le retrait en < 24h.
- **Due diligence** : verifier le systeme de tracabilite (souvent logiciel ERP specifique). Un systeme defaillant = risque sanitaire = risque reputationnel.

## 4. Risques cles

### Risques eliminatoires

| Red flag | Impact | Verification |
|---|---|---|
| **Perte certification IFS/BRC** | Perte clients GMS | Organisme certificateur |
| **Contamination sanitaire non resolue** | Rappel produit, fermeture DDPP | DDPP, historique alertes RASFF |
| **Droit de preemption SAFER exerce** | Perte du foncier | Notification SAFER pre-cession |
| **Pollution des sols (pesticides, effluents)** | Remise en etat obligatoire | Diagnostic sols |
| **Quotas/droits de production epuises** | Pas de droit a produire | Verification administrative |

### Risques a maitriser

- **Meteo** : aucune garantie. Le repreneur doit prevoir un plan B (assurance recolte, diversification).
- **PAC** : les aides peuvent evoluer (reforme PAC). Ne pas baser le modele economique a > 30% sur les aides.
- **Foncier** : distinguer propriete vs fermage. Le fermier a un droit de preemption et un bail rural tres protecteur (9 ans minimum, renouvellement quasi automatique).
- **Bien-etre animal** : normes en evolution rapide (cages, surface, acces exterieur). Budget mise aux normes.
- **Eau** : retenues collinaires, forages = autorisations administratives a verifier.
- **MSA** : equivalent URSSAF pour l'agriculture. Verifier les cotisations impayees.

## 5. Valorisation

### Methodes privilegiees agriculture distressed

| Methode | Application | Reference |
|---|---|---|
| **Valeur fonciere** | Terres + batiments | Prix/ha regional (SAFER publie les prix annuels) |
| **DPB capitalises** | Aides PAC | DPB/ha x nombre d'hectares x 5-8 ans |
| **Cheptel** | Elevage | Cours du marche (cotations nationales) |
| **Materiel agricole** | Tracteurs, moissonneuses | Cote Argus agricole (Agriaffaires) |
| **EBE normalise** | Exploitation rentable | 3-5x EBE (hors foncier) |
| **Fonds agricole** | Clientele + contrats + labels | 0,5-1,5x EBE |

### Methodes privilegiees IAA distressed

| Methode | Application | Reference |
|---|---|---|
| **EBE normalise** | Usine rentable | 4-6x EBE |
| **Actifs industriels** | Lignes de production, froid | Expertise independante |
| **Contrats commerciaux** | MDD grande distribution | Marge nette capitalisee |
| **Marques propres** | Marque connue (AOP, terroir) | Royalties capitalisees |
| **Valeur liquidative** | Cessation | Materiel (encheres specialisees) + stock |

### Specificites

- **Foncier agricole** : prix moyens 6 000-10 000 EUR/ha (grandes cultures), 15 000-50 000 EUR/ha (vignoble AOP), 100 000+ EUR/ha (Champagne, grands crus).
- **Cheptel** : valeur vive vs valeur boucherie. En distressed, le cheptel est souvent vendu en urgence a la valeur boucherie (-30 a -50% vs valeur vive).
- **Stock IAA** : les stocks de produits finis perissables perdent 50-100% de leur valeur en quelques semaines.
- **Labels** : une exploitation en AOP/AOC vaut significativement plus qu'une exploitation conventionnelle equvalente (terroir = rente).

## 6. Exemples emblematiques France

| Entreprise | Annee | Secteur | Procedure | Issue | Enseignement |
|---|---|---|---|---|---|
| **Doux** | 2018 | Volaille industrielle | LJ | Reprise partielle (LDC, Saudi) | Leader europeen tombe, 3 200 emplois |
| **Bigard/Charal** (restructuration) | Continu | Viande bovine | Restructuration | Fermeture d'abattoirs | Concentration du secteur |
| **Cooperatives laitieres** (multiples) | Continu | Lait | RJ/LJ | Fusions, reprises | Surcapacite, prix du lait bas |
| **Exploitations cerealières Beauce** | 2024-2025 | Grandes cultures | Reglement amiable | Echeanciers | Meteo + baisse des prix |
| **Vignobles Bordeaux** | 2024-2025 | Viticulture | RJ/arrachage | Subventions arrachage | Surproduction + baisse consommation vin |
| **Ecomiam (D&L Commerce)** | 2024 | Surgeles | RJ | En cours | Concept interessant, distribution specialisee |

### Patterns observes

1. **L'agriculture est un secteur politique** : l'Etat, la SAFER, les chambres d'agriculture interviennent systematiquement.
2. **Le foncier est le principal actif** : meme si l'exploitation est deficitaire, la terre conserve sa valeur.
3. **Les cooperatives** jouent un role majeur (collecte, transformation, distribution). Le repreneur doit maintenir les relations cooperatives.
4. **L'IAA se consolide** : les groupes (LDC, Bigard, Lactalis, Savencia) rachetent les acteurs en difficulte.
5. **Les labels protegent** : une exploitation AOP/AOC se vend mieux et plus vite qu'une exploitation conventionnelle.

## 7. Checklist repreneur Brantham

### Phase 1 : Quick Scan (48h)

- [ ] Identifier le type d'exploitation (grandes cultures, elevage, viticulture, IAA)
- [ ] Structure juridique (GAEC, EARL, SCEA, individuel, SARL/SAS pour IAA)
- [ ] Verifier le foncier : propriete vs fermage, baux ruraux en cours, surfaces
- [ ] DPB : montant/ha, nombre d'hectares eligibles
- [ ] Cheptel : effectif, races, etat sanitaire
- [ ] Certifications : IFS/BRC (IAA), bio, AOP/AOC, Label Rouge
- [ ] Contacter la SAFER pour anticiper le droit de preemption

### Phase 2 : Due Diligence (14-21 jours)

- [ ] **Foncier** : cadastre, baux ruraux (duree, loyer, preneur), servitudes, irrigation
- [ ] **PAC** : DPB, telepac, aides percues sur 3 ans, eco-conditionnalite
- [ ] **SAFER** : notification prealable, droit de preemption, historique interventions
- [ ] **Elevage** : IPG (identifiant permanent generalise), sanitaire (prophylaxie, tuberculose, brucellose)
- [ ] **IAA** : certifications IFS/BRC (date prochain audit), HACCP, tracabilite, ICPE si applicable
- [ ] **Materiel** : inventaire, age, etat, credit-bail (CUMA = cooperative de materiel)
- [ ] **Social** : salaries permanents, saisonniers, travailleurs etrangers (conformite)
- [ ] **Environnement** : nitrates, phytosanitaires, effluents, installations classees (ICPE elevage > seuil)
- [ ] **Eau** : autorisations de prelevement, retenues, forage
- [ ] **Fiscal** : regime fiscal (micro-BA, reel simplifie, reel normal), MSA, impots fonciers
- [ ] **Assurances** : assurance recolte, responsabilite civile exploitation, mortalite du betail

### Phase 3 : Structuration offre

- [ ] Perimetre : foncier + cheptel + materiel + DPB + contrats + labels
- [ ] Prix : foncier (prix SAFER) + cheptel (cours) + materiel (cote) + DPB
- [ ] Plan d'exploitation : assolement, cheptel, investissements, debouches
- [ ] Volet emploi : salaries repris, saisonniers
- [ ] SAFER : anticiper la preemption, preparer un dossier argumentant l'interet du projet
- [ ] Financement : credit agricole, BPI, aides a l'installation (DJA si jeune agriculteur)
- [ ] Communication : chambre d'agriculture, cooperative, voisins, commune

### Points d'attention specifiques agriculture/IAA

- **SAFER** : risque n.1. Toujours consulter la SAFER avant de deposer une offre.
- **Bail rural** : le fermier a un droit de preemption et un bail quasi perpetuel. Impossible de le deloger facilement.
- **Cooperative** : l'engagement cooperatif (duree, volumes, penalites de sortie) est un passif potentiel.
- **Saisonnalite** : reprendre une exploitation cerealiere apres les moissons (aout-septembre) = stock de cereales valorisable. Avant = rien.
- **MSA** : les cotisations MSA impayees s'accumulent vite. Verifier le passif MSA.
- **Bien-etre animal** : les normes evoluent vite (poules en cage, truies en stalle). Budget mise aux normes.
- **Eau** : l'acces a l'eau d'irrigation est un actif strategique (droits de prelevement = rares et chers).
