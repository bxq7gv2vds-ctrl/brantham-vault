# Case Study : Courtepaille — Cession à Buffalo Grill (2020)

## Fiche signalétique

| Indicateur | Valeur |
|---|---|
| Secteur | Restauration (grillades, route) |
| Creation | 1961 |
| Restaurants | ~250 (145 propres + 92 franchises) |
| Salaries | ~3 000 |
| CA | ~200 M EUR |
| Actionnaire | ICG (Intermediate Capital Group) depuis 2015 |
| Issue | Cession à Buffalo Grill pour 17 M EUR |

## Chronologie

- **2000, 2015** : deux LBO successifs (Barclays PE puis ICG)
- **Mars-mai 2020** : COVID → fermeture totale des restaurants + refus PGE (situation pré-existante)
- **29 juil 2020** : RJ Tribunal Evry (admin. : Bourbouloux + Abitbol)
- **Sept 2020** : Phase d'offres — Buffalo Grill (TDR Capital), Groupe Bertrand (Hippopotamus)
- **25 sept 2020** : Tribunal retient Buffalo Grill — 17 M EUR + plan investissement 100 M EUR/5 ans
- **Issue** : 145 restaurants repris + 92 franchises. Holding Napaqaro (Buffalo Grill + Courtepaille + Popeyes)

## Patterns identifiés

- **LBO + secteur cyclique + choc exogène** : la dette LBO empêche de constituer des réserves, le COVID achève l'entreprise
- **Refus PGE** : signal que la banque (BPI) estime l'entreprise non viable même avec le soutien de l'Etat
- **Reprise par le concurrent direct** : Buffalo Grill a les synergies, connait le secteur, investit réellement
- **Plan d'investissement crédible** : 100 M EUR sur 5 ans = engagement sérieux devant le tribunal

## Scoring Brantham

| Red Flag | Score |
|---|---|
| Historique LBO (2 LBO successifs) | +3 |
| Secteur restauration en mutation | +1 |
| Concept vieillissant | +1 |
| Refus PGE | signal fort |
**Total : 5+ — redressement difficile sans choc exogène**

## Lessons learned

1. Le refus de PGE par BPI est un signal que la banque estime l'entreprise non viable — à surveiller
2. La restauration thématique en LBO = risque élevé si concept daté
3. L'offre avec le plan d'investissement le plus solide l'emporte sur l'offre la plus chère
4. Deux LBO successifs = double couche de dette + double sous-investissement
