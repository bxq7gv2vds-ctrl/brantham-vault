# Case Study : Orchestra-Premaman — Sauvegarde puis Cession au Fondateur (2019-2021)

## Fiche signalétique

| Indicateur | Valeur |
|---|---|
| Secteur | Mode enfantine et puériculture |
| Creation | 1995 (Montpellier), fusion 2012 avec Premaman |
| CA | ~700 M EUR |
| Magasins | ~530 monde (dont 257 France) |
| Salaries | ~3 500 |
| Fondateur/DG | Pierre Mestre |
| Issue | Cession au fondateur (NewOrch), activité continue |

## Chronologie

- **2012** : Fusion Orchestra + Premaman (belge), cotation bourse — dette ~550 M EUR
- **2017-2018** : Restructurations de dette successives, fragilité structurelle
- **24 sept 2019** : Sauvegarde Tribunal Montpellier
- **29 avr 2020** : Conversion en RJ (COVID + cessation paiements 12 mars 2020)
- **19 juin 2020** : Plan de cession approuvé → NewOrch (Pierre Mestre, fondateur)
  - Concurrent : Al-Othaim (groupe saoudien) — non retenu
  - ~430 magasins repris (dont 257 France)
- **2 fév 2021** : LJ de l'ancienne entité Orchestra-Premaman (NewOrch continue)

## Patterns identifiés

- **Le fondateur peut reprendre sa propre entreprise** via "pre-pack cession" — légal et courant en France
- **La sauvegarde convertie en RJ** : parcours classique quand la situation se dégrade pendant la procédure
- **L'offre du fondateur est souvent retenue** : il connaît l'entreprise, rassure les parties prenantes
- **Risque moral** : les créanciers et le procureur vérifient que le fondateur ne "lessine" pas la dette abusivement
- **Secteur puériculture en déclin** : baisse de natalité + fast fashion + seconde main

## Lessons learned

1. Le fondateur-repreneur est une configuration courante — à utiliser dans le sourcing de repreneurs
2. La sauvegarde est une procédure préventive qui peut être convertie si elle n'aboutit pas rapidement
3. Méfiance envers l'offre étrangère (Al-Othaim) vs fondateur français — le tribunal préfère souvent la continuité locale
4. La dette post-fusion est souvent mal dimensionnée — signale une acquisition opportuniste sans synergies réelles
