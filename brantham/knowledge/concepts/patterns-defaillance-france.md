# Patterns de Défaillance des Entreprises Françaises — Encyclopédie Brantham

Synthèse des patterns récurrents identifiés sur 11 cas d'école majeurs (Camaieu, Go Sport, Aigle Azur, La Halle, Conforama, Courtepaille, Orchestra, Scopelec, San Marina, Casino, Orpea).

---

## 1. Les 5 Patterns de Défaillance

### Pattern 1 : "Le LBO Toxique" ⚠️ CRITIQUE

**Cas** : Camaieu, La Halle/Vivarte, Courtepaille, Casino (partiellement)

**Mécanisme** :
```
Acquisition à prix élevé par dette → sous-investissement chronique
→ perte de compétitivité → choc exogène (COVID) → cessation des paiements
```

**Délai moyen** : 10-15 ans entre le LBO et la liquidation

**Signaux avancés** :
- Restructuration de dette + changement de propriétaire multiple
- Ratio dette/EBITDA > 5x
- Investissements capex en baisse > 3 ans consécutifs

**Score Brantham** : +3 (historique LBO >7 ans) + +2 (restructuration de dette précédente)

---

### Pattern 2 : "L'Érosion de Marché Non Anticipée"

**Cas** : Camaieu, San Marina, Go Sport, Orchestra, Aigle Azur

**Mécanisme** :
```
Changement comportement consommateur (e-commerce, fast fashion, seconde main)
+ absence d'adaptation du modèle → perte de CA → déficits → procédure
```

**Délai** : 5-10 ans d'érosion avant la chute finale

**Signaux avancés** :
- Baisse de CA comparable > 3 ans consécutifs
- Part du canal digital < 10% pour une enseigne retail
- Taux de renouvellement du concept nul

**Score Brantham** : +2 (baisse CA >10% sur 3 ans) + +2 (secteur en crise structurelle)

---

### Pattern 3 : "La Dépendance Client Fatale"

**Cas** : Scopelec (Orange 40% du CA)

**Mécanisme** :
```
Un client représente >30-40% du CA → le client change de stratégie
→ perte brutale de revenus → insolvabilité en 6-12 mois
```

**Délai** : 6-12 mois entre la perte du contrat et la procédure

**Signaux avancés** :
- Concentration CA chez un client >30%
- Annonces de ré-internalisation ou changement fournisseur chez le client
- Absence de contrats LT sécurisés

**Score Brantham** : +3 (concentration CA client >30%) → ALERTE ROUGE SYSTÉMATIQUE

---

### Pattern 4 : "L'Actionnaire Défaillant"

**Cas** : Conforama (Steinhoff), Go Sport (Hopps Group), Aigle Azur (gouvernance), Orpea (CDC)

**Mécanisme** :
```
L'actionnaire est lui-même en difficulté ou incompétent
→ absence de soutien financier → crise de liquidité de la filiale → procédure
```

**Signaux avancés** :
- Difficultés de l'actionnaire parent (press check)
- Changements de direction fréquents (>2 PDG en 5 ans)
- Actionnaire étranger dans un secteur éloigné de son core business

---

### Pattern 5 : "Le Secteur Structurellement Condamné"

**Cas** : Aigle Azur (aérien low-margin), Camaieu/San Marina (mode milieu de gamme physique)

**Mécanisme** :
```
Le secteur lui-même est en crise structurelle
→ même les bons opérateurs souffrent → les faibles meurent
```

**Indicateur** : taux de défaillance sectoriel > 2x la moyenne nationale

**Secteurs à risque élevé en 2025** :
- Retail mode milieu de gamme physique (centres commerciaux)
- Restauration thématique en LBO
- Sous-traitance télécoms / BTP sur mono-contrat
- Imprimerie / édition papier

## 2. Matrice Facteurs de Succès d'une Reprise

| Facteur | Exemple positif | Contre-exemple |
|---|---|---|
| Repreneur industriel (connait le secteur) | Beaumanoir→La Halle, Intersport→Go Sport | FIB→Camaieu (financier, échec) |
| Investissement post-reprise significatif | Buffalo Grill : 100 M EUR/5 ans | FIB/Camaieu : insuffisant |
| Synergies opérationnelles réelles | Intersport+Go Sport (achats, logistique) | Hopps/Al-Thani+Go Sport (aucune) |
| Plan emploi crédible (>60% repris) | La Halle (60%), Go Sport (90%) | Scopelec (29%), Camaieu (0%) |
| Absence de dette LBO historique | — | Camaieu, La Halle, Courtepaille |

## 3. Grille de Scoring "Red Flags" Brantham

| Red Flag | Score | Source de détection |
|---|---|---|
| Historique LBO (>7 ans) | +3 | BODACC, Pappers, presse financière |
| Concentration CA client >30% | +3 | Bilans, rapports sectoriels |
| Baisse CA >10% sur 3 ans | +2 | Bilans, presse sectorielle |
| Actionnaire étranger en difficulté | +2 | Presse financière, notations |
| Secteur en crise structurelle | +2 | Stats défaillances, rapports sectoriels |
| Restructuration de dette précédente | +2 | BODACC, presse financière |
| PGE non remboursé | +2 | BODACC, rapports BPI |
| Multiple changements de direction (<3 ans) | +1 | BODACC, presse |
| Magasins en centre commercial (>50%) | +1 | Sites web enseignes |
| Absence de canal digital significatif | +1 | Analyse web |

**Seuil d'alerte** : score >= 6 = dossier à prioriser pour analyse approfondie

## 4. Statistiques de Référence (11 cas, 2019-2024)

| Issue | Nombre | Cas |
|---|---|---|
| Cession réussie (repreneur) | 5 | La Halle, Go Sport, Courtepaille, Conforama, Orchestra |
| Liquidation sans repreneur | 3 | Camaieu (2022), Aigle Azur, San Marina |
| Cession partielle (démantèlement) | 2 | Scopelec, Casino (partiellement) |
| Restructuration via sauvegarde accélérée | 1 | Orpea/Emeis |

**Taux de survie de l'activité** : ~60% (6-7/11)
**Taux de survie de l'emploi** : ~45% en moyenne (emplois repris / emplois initiaux)
**Délai moyen** (premiers signes → résolution) : 2-4 ans

## 5. Application Brantham : Secteurs à Éviter

Secteurs où le taux de réussite d'une reprise est structurellement faible :
- Transport aérien (Aigle Azur → 0 offres retenues sur 14)
- Retail mode milieu de gamme physique en centres commerciaux
- Sous-traitance mono-client

Secteurs avec reprises réussies possibles :
- Distribution sport (Go Sport → Intersport)
- Ameublement (Conforama → BUT)
- Restauration thématique (Courtepaille → Buffalo Grill)
- Puériculture/enfant (Orchestra → fondateur)
