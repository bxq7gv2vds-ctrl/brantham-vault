## Insights qualité — panel senior check-in

Panel senior a validé que :
- **DSR per-trade Sharpe 3.43 avec kurt +20.7** = distribution convex tails. Ce n'est pas un Sharpe annualisé trading-realisable mais une per-signal expectancy — **à ne pas sur-interpréter**
- **SHAP Kendall tau 1.0** sur KLAX suggère 3-5 features dominantes stables (probablement climo + lag + ens_mean + UHI). Robustesse confirmée mais ROI marginal des nouvelles features limité tant que données ne changent pas
- **Purged K-fold 1.17** avec embargo 6h. Embargo 72h trop aggressif (train set < 20). Signaux sont concentrés temporellement (2 jours d'émission distincte sur 1027 outcomes) → **G1 DSR daily impossible** tant que flow temporel n'est pas étalé
- **3 FAIL révèlent des gaps réels**, pas des bugs du kit. Le kit fait son job : refuser de passer G2 avec strat mal configurée
