# Stratégie de Pricing pour les Acquisitions

## I. Philosophie de Pricing

**Principe fondamental** : Acheter bas, vendre haut - toujours avec une marge de sécurité minimum de 25%

**Approche multi-méthodes** : Croiser 5 méthodes de valorisation pour déterminer la fourchette de prix optimale

---

## II. Méthodes de Valorisation

### A. Méthode des Flux de Trésorerie Actualisés (DCF)
```markdown
**Formule** : Valeur = Σ [FCF / (1+r)^t] + Valeur terminale

**Paramètres clés** :
- FCF = Free Cash Flow (€M)
- r = Taux d'actualisation (WACC)
- t = Période (années)
- Valeur terminale = FCF final × (1+g) / (r-g)

**Application pratique** :
- Croissance perpétuelle : 2-3%
- WACC cible : 8-12%
- Horizon : 5-7 ans
```

### B. Méthode des Multiples
```markdown
**Multiples à utiliser** :
- EV/EBITDA : 6-12x (sector-dependent)
- EV/Sales : 1-5x (sector-dependent)
- P/E : 15-25x (growth companies)
- EV/EBIT : 8-15x

**Benchmarking** :
- Premium sectoriel : +20-30%
- Premium qualité : +15-25%
- Premium croissance : +10-20%
```

### C. Méthode des Actifs Nets
```markdown
**Calcul** : Actifs - Passifs + Goodwill

**Actifs à valoriser** :
- Immobilisations : Valeur marchande
- Stock : Coût ou marché
- Créances : Valeur recouvrable
- Trésorerie : Valeur nominale
- Intangibles : Brevets, brands

**Passifs à provisionner** :
- Dettes financières : Valeur actuelle
- Provisions légales
- Engagements hors bilan
```

### D. Méthode des Comparables Boursiers
```markdown
**Peer group selection** :
- Même secteur
- Même taille
- Même croissance
- Même région

**Multiples ajustés** :
- Size premium
- Growth premium
- Quality premium
- Control premium
```

### E. Méthode des Options Réelles
```markdown
**Prix optionnel** : Valeur de flexibilité stratégique

**Facteurs d'option** :
- Option de croissance
- Option d'abandon
- Option de délai
- Option de transformation
```

## III. Framework de Pricing Stratégique

### A. Fourchette de Prix par Type d'Acquisition

| Type d'Acquisition | Fourchette Multiples | Marge Sécurité | Prime de Contrôle |
|-------------------|-------------------|----------------|-------------------|
| Croissance organique | 8-12x EV/EBITDA | 30% | 15-20% |
| Market share | 6-10x EV/EBITDA | 25% | 20-25% |
| Technology play | 10-15x EV/EBITDA | 35% | 25-30% |
| Turnaround | 4-8x EV/EBITDA | 40% | 30-35% |
| Consolidation | 5-9x EV/EBITDA | 20% | 10-15% |

### B. Prix Psychologique et Négociation

```markdown
**Prix d'ouverture** : 70-80% de la valeur théorique
**Prix cible** : 85-90% de la valeur théorique
**Prix réservation** : 75-80% de la valeur théorique

**Tactiques de négociation** :
- Ancrage à la baisse
- Value engineering
- Earn-out structure
- Déclencheurs de prix
```

---

## IV. Facteurs d'Ajustement de Prix

### A. Ajustements Positifs (+)
- [ ] Synergies claires (>15% EBITDA)
- [ ] Technologie propriétaire
- [ ] Positionnement fort sur le marché
- [ ] Équipe de management excellente
- [ ] Portefeuille clients de qualité
- [ ] Brevets et IP précieuse
- [ ] Croissance organique >10%

### B. Ajustements Négatifs (-)
- [ ] Dette excessive (>3x EBITDA)
- [ ] Dépendance clients >20%
- [ ] Risques réglementaires
- [ ] Technologie obsolète
- [ ] Perte de clés compétences
- [ ] Marché en déclin
- [ ] Management instable

### C. Calcul du Prix Ajusté
```markdown
Prix de base × (1 + ajustements positifs) × (1 - ajustements négatifs) × marge de sécurité
```

## V. Structure de Prix et Paiements

### A. Composantes du Prix Total
```markdown
**Prix upfront** : 60-70%
**Earn-out** : 20-30%
**Retenue de garantie** : 5-10%
** paiements conditionnels** : 5-15%
```

### B. Structure d'Earn-out Optimale
```markdown
**Métriques de performance** :
- Chiffre d'affaires : 40-50%
- EBITDA : 30-40%
- Free Cash Flow : 20-30%
- Nouveaux clients : 10-20%

**Échéancier** :
- Année 1 : 30-40%
- Année 2 : 30-40%
- Année 3 : 20-30%
```

### C. Garanties et Indemnisation
```markdown
**Garanties minimales** :
- Actifs existants
- Passifs connus
- Performance future
- Litiges existants

**Cap d'indemnisation** : 10-20% du prix d'acquisition
**Délai de prescription** : 18-24 mois
```

---

## VI. Processus de Décision de Prix

### A. Workflow d'Approbation
```markdown
**Étape 1** : Analyse financière (Investment Committee)
- Score >100/150 → Continue
- Score 80-100 → Conditions
- Score <80 → Refuse

**Étape 2** : Due diligence (Legal/Financial)
- Sans red flag → Négociation
- Avec red flag → Reformulation

**Étape 3** : Négociation final (CEO/CFO)
- Fourchette confirmée → Offre
- Fourchette révisée → Recalcul
- Hors fourchette → Abandon
```

### B. Triggers de Prix
```markdown
**Green lights** :
- Due diligence clean
- Synergies validées
- Financement secured
- Approbation réglementaire
- Management team committed

**Red flags** :
- Dilution significative (>20%)
- Dettes cachées (>10% CA)
- Risques juridiques
- Culture incompatibilité
- Synergies non réalisables
```

### B. Scénarios de Sensibilité
```markdown
**Scénario base** : Prix standard, earn-out 20%
**Scénario optimiste** : Prix -10%, earn-out 15%
**Scénario pessimiste** : Prix +10%, earn-out 30%
**Scénario abandon** : Coût d'opportunité perdu
```

---

## VIII. Règles d'Or

1. **Jamais payer plus de 12x EBITDA** pour une acquisition de croissance
2. **Toujours avoir une marge de sécurité de 25% minimum**
3. **Les earn-outs doivent être mesurables et réalisables**
4. **La due diligence doit être complète avant offre**
5. **Le prix final doit refléter la valeur future, pas passée**
6. **Privilégier les transactions en cash aux stock swaps**
7. **Savoir dire non si les conditions ne sont pas optimales**
