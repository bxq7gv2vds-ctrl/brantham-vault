# Risques Juridiques Courants en M&A

## I. Cartographie des Risques par Phase

### A. Phase Pré-Transaction (Due Diligence)

**Risques High Impact**
```markdown
**Litiges en cours** :
- [ ] Contentieux clients/fournisseurs (>€1M)
- [ ] Litiges fiscales non provisionnés
- [ ] Litiges administratifs/environnementaux
- [ ] Litiges de propriété intellectuelle
- [ ] Class actions ou recours collectifs

**Actions de mitigation** :
- Demander copies de tous les actes de procédure
- Provisionner 100% + 20% de marge
- Exiger clauses d'indemnisation spécifiques
- Vérifier couverture d'assurance D&O
```

**Risques Medium Impact**
```markdown
**Conformité réglementaire** :
- [ ] Sanctions administratives (RGPD, competition law)
- [ ] Licences et autorisations expirées
- [ ] Non-conformité sectorielle
- [ ] Violations environnementales

**Actions de mitigation** :
- Audit compliance complet
- Checkliste réglementaire par secteur
- Clauses de garantie pour conformité
- Plan de correction post-acquisition
```

### B. Phase de Négociation

**Risques Contractuels**
```markdown
**Earn-out mechanisms** :
- Risque : Définitions floues, impossibilité de mesure
- Mitigation : Métriques objectives, clauses de vérification
- Best practice : 3ème partie indépendante pour validation

**Garanties et Indemnités**:
- Risque : Cap d'indemnisation insuffisant
- Mitigation : Négocier caps sectoriels
- Best practice : Caps différenciés par type de risque

**Exclusions de garantie**:
- Risque : Omissions intentionnelles
- Mitigation : Demander liste exhaustive des exclusions
- Best practice : "Known unknowns" clause
```

**Risques Réglementaires**
```markdown
 **Approbations antitrust** :
- Risque : Déconcentration ou interdiction
- Mitigation : Filing précooc, consultation préalable
- Best practice : HSR filing immédiat

**Approbations sectorielles**:
- Risque : Refus de l'autorité de tutelle
- Mitigation : Early dialogue avec régulateurs
- Best practice : Conditional closing clause
```

### C. Phase Post-Closing

**Risques de Transition**
```markdown
**Employés et sociaux**:
- Risque : Licenciements abusifs, transfert de contrat
- Mitigation : Processus CSE conforme, avis préalable
- Best practice : Mobility clauses dans les contrats

**Données et systèmes**:
- Risque : Non-conformité RGPD post-intégration
- Mitigation : Audit data mapping, consentements clients
- Best practice : Data protection officer dédié
```

## II. Analyse des Risques par Domaine Juridique

### A. Risques Contractuels

**Contrats Clients** (High Frequency)
```markdown
**Points critiques**:
- [ ] Clauses de résiliation (30-90 jours vs 180+ jours)
- [ ] Prix fixes vs indexés
- [ ] Limites de responsabilité asymétriques
- [ ] Exclusivités territoriales
- [ ] Compliance obligations

**Checkliste due diligence**:
- Contrats >€100k : examen complet
- Taux de résiliation >10% : investigation
- Concentration >30% sur un client : risque
- Durée moyenne contrat <12 mois : alerte
```

**Contrats Fournisseurs** (Medium Frequency)
```markdown
**Points critiques**:
- [ ] Clauses d'approvisionnement exclusif
- [ ] Prix long terme indexés
- [ ] Responsabilité environnementale
- [ ] Clauses de non-concurrence
- [ ] Conditions de paiement strictes

**Stratégie de mitigation**:
- Renégocier contrats stratégiques
- Diversifier fournisseurs critiques
- Audit clauses ESG
- Review guarantees et warranties
```

**Baux et Immeubles** (High Impact)
```markdown
**Points critiques**:
- [ ] Durée restante <5 ans
- [ ] Clauses de renégociation
- [ ] Subordination et interdiction de sous-louer
- [ ] Usage incompatible avec nouvelle activité
- [ ] Charges locales excessives

**Actions spécifiques**:
- Vérifier cadastre et servitudes
- Demander copies des baux
- Évaluer risques urbanistiques
- Check compliance construction
```

### B. Risques Propriété Intellectuelle

**Brevets et Marques** (High Impact)
```markdown
**Matrice de risque**:
```
| Catégorie | Risque | Action |
|-----------|--------|---------|
| Brevets en cours | Validité contestée | Vérifier procédures nullité |
| Brevets expirés | Libellé trop large | Freedom-to-operate analysis |
| Marques non déposées | Risk squatters | Filing immédiat post-close |
| Software patents | Prior art claims | Patent landscape audit |
| Trade secrets | Protection adéquate | Review NDAs et confidentiality |
```

**Stratégie IP Due Diligence**:
- [ ] Portfolio audit complet
- [ ] Freedom-to-operate analysis
- [ ] Trademarks clearance search
- [ ] Software licensing review
- [ ] Employee invention policies

### C. Risques Réglementaires

**Conformité Sectorielle** (Variable Impact)
```markdown
**Secteurs à risque élevé**:
- [ ] Santé : Autorisations ANSM, conformité dispositifs médicaux
- [ ] Finance : AMF compliance, capital requirements
- [ ] Tech : GDPR, RGPD, e-privacy
- [ ] Énergie : Permis environnementaux, sécurité nucléaire
- [ ] Alimentaire : HACCP, traçabilité, certifications

**Checklist par secteur**:
- Identifier toutes les licences requises
- Vérifier historique sanctions
- Audit compliance interne
- Review contrats avec autorités
```

**Risques Antitrust** (High Impact)
```markdown
**Seuils de déclaration**:
- Chiffre d'affaires combiné >€250M UE
- Part de marché >25% dans un marché
- Concentration mesurée par HHI

**Signaux d'alerte**:
- [ ] Marché concentré (HHI >2000)
- [ ] Chevauchements géographiques significatifs
- [ ] Clients communs >30%
- [ ] Fournisseurs communs >40%

**Mitigation strategies**:
- Filing précooc avec autorités
- Commitments divestiture structurés
- Behavioral remedies pour éviter blocage
```

### D. Risques Fiscaux

**Droits d'Enregistrement et Taxes** (Medium Impact)
```markdown
**Points critiques**:
- [ ] Droits de mutation (5-7.5% en moyenne)
- [ ] TVA sur le transfert de goodwill
- [ ] Impôts locaux (taxe professionnelle)
- [ ] Taxes sur les actifs incorporels
- [ ] Impôts différés latents

**Stratégie de mitigation**:
- Structure transaction optimisée
- Utiliser stock vs asset selon profil fiscal
- Due diligence fiscale indépendante
- Tax warranty clauses spécifiques
```

**Risques de Double Imposition** (High Impact)
```markdown
**Points à vérifier**:
- Conventions fiscales applicables
- Stabilité juridique des entités
- Pricing transfert intra-groupe
- Impôts différés sur restructuration
- Compliance FATCA/CRS

**Actions de mitigation**:
- Advance pricing agreements
- Ruling fiscal préalable
- Tax efficient structuring
- Due diligence fiscale deep dive
```

## III. Checklist de Due Diligence Juridique

### A. Due Diligence Contractuelle (Jour 1-30)
```markdown
**Priorité 1** - Contrats >€100k ou >3 ans :
- [ ] Clients Top 10 : termes, risques, concentration
- [ ] Fournisseurs stratégiques : dépendance, coûts
- [ ] Partenariats exclusifs : restrictions, alternatives
- [ ] Accords de non-concurrence : portée, durée
- [ ] Licences et franchises : droits, redevances

**Priorité 2** - Tous contracts :
- [ ] Copies complètes de tous les contrats
- [ ] Liste des contrats expirant <12 mois
- [ ] Identification des clauses résiliables
- [ ] Vérification des garanties et assurances
- [ ] Analyse des risques de sous-traitance
```

### B. Due Diligence Propriété Intellectuelle (Jour 1-45)
```markdown
**Portefeuille audit**:
- [ ] Brevets : titres, validations, procédures
- [ ] Marques : enregistrements, oppositions, renouvellements
- [ ] Droits d'auteur : registrations, assignments
- [ ] Secrets commerciaux : protection, NDAs
- [ ] Software licenses : compliance, restrictions

**Freedom-to-operate**:
- [ ] Analysis brevets blocking
- [ ] Clearance trademarks
- [ ] Due diligence IP vendors
- [ ] Employee invention policies
- [ ] IP assignment verification
```

### C. Due Diligence Réglementaire (Jour 1-30)
```markdown
**Licences et autorisations**:
- [ ] Liste complète de toutes les licences
- [ ] Validité, renouvellements, conditions
- [ ] Historique des suspensions/refus
- [ ] Approbations sectorielles spécifiques
- [ ] Conformité réglementaire en cours

**Conformité interne**:
- [ ] Politiques compliance documentées
- [ ] Procédures de reporting
- [ ] Enquêtes internes récentes
- [ ] Sanctions et amendes passées
- [ ] Whistleblowing policies
```

## IV. Gestion des Risques Post-Closing

### A. Intégration Contractuelle (30-90 jours)
```markdown
**Actions prioritaires**:
- [ ] Renégociation contrats stratégiques
- [ ] Transfert de contrats tiers (notifiés)
- [ ] Novation ou résiliation ciblée
- [ ] Harmonisation des conditions générales
- [ ] Audit compliance post-intégration

**Monitoring**:
- Suivi des contentieux émergents
- Review des garanties données
- Vérification des délais de résiliation
- Audit des contrats nouvelles entités
```

### B. Monitoring Réglementaire Continu
```markdown
**Reporting trimestriel**:
- Changements réglementaires impactants
- Audit compliance interne
- Évolution des risques sectoriels
- Performance des warranties
- Nouveaux contentieux

**Alertes systèmes**:
- Changes in relevant legislation
- Regulatory enforcement actions
- Sanction updates
- Case law developments
- License renewals due
```

## VI. Playbooks de Crise

### A. Crisis Response Framework
```markdown
**Incident Severity Levels**:
- Level 1 : Localisé, <€100k
- Level 2 : Régional, €100k-1M
- Level 3 : National, €1M-10M
- Level 4 : International, >€10M

**Response Times**:
- Level 1 : 24 heures
- Level 2 : 12 heures
- Level 3 : 6 heures
- Level 4 : Immédiat

**Escalation Path**:
- Legal counsel → General counsel → CEO → Board
```

### B. Crisis Communication
```markdown
**Internal Communication**:
- Employees : Within 4 hours of incident
- Managers : Detailed briefing within 8 hours
- Board : Summary within 24 hours

**External Communication**:
- Regulators : Within 48 hours
- Clients : Within 72 hours
- Public : Only when necessary, controlled messaging

**Templates pré-approuvés**:
- Crisis statement template
- Q&A document
- Social media guidelines
- Media contact list
```
