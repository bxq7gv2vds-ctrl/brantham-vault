# Trame visuelle — Comment on gère un deal

> Document à mémoriser **visuellement**. C'est ta map mentale pendant le call. Tu dois pouvoir la dessiner les yeux fermés.
>
> Sert deux usages : (1) tu sais ce qui va se passer, (2) tu peux le **raconter au repreneur** en 60 secondes pour qu'il visualise son parcours avec nous.

---

## 1. La timeline en un coup d'œil

```
┌──────────────────────────────────────────────────────────────────────────────────┐
│                                                                                  │
│   AUJOURD'HUI         J+1 → J+3        J+3 → J+15       J+15 → J+25              │
│   ─────────────       ───────────      ───────────      ───────────              │
│   ① CALL INTRO   ➜   ② NDA + DR   ➜   ③ DD REPRENEUR ➜  ④ OFFRE 1                │
│   30 min              + cadrage         + visite site     [FEE #1]               │
│   pas de data         + lettre miss.    + Q&A AJ                                 │
│                                                                                  │
│                                                                                  │
│   J+25 → J+35              J+35 → J+45            J+45 → J+90+                   │
│   ─────────────            ─────────────          ─────────────                  │
│   ⑤ AUDIENCE TRIB.    ➜   ⑥ CLOSING       ➜    ⑦ POST-RACHAT                     │
│   + offre finale           + jugement              + restructuring               │
│   [FEE #2]                 + transfert             + accompagnement              │
│                            [FEE VARIABLE]                                        │
│                                                                                  │
└──────────────────────────────────────────────────────────────────────────────────┘
```

**Durée totale typique** : 6 à 10 semaines de l'appel à la signature, contraint par la deadline du tribunal.

## 2. Les 7 phases — détail visuel

### ① CALL INTRO — Aujourd'hui (30 min)

```
   PAUL/SOREN              REPRENEUR
       │                       │
       │  Présentation 2 min   │
       ├──────────────────────►│
       │                       │
       │  Pédagogie 4 min      │
       ├──────────────────────►│
       │                       │
       │  Deal macro 4 min     │
       ├──────────────────────►│
       │                       │
       │  Discovery 10 min     │
       │◄─────────────────────►│
       │                       │
       │  Closing + NDA        │
       ├──────────────────────►│
       ▼                       ▼
   Envoi NDA              Décision GO/NO-GO
   sous 1h
```

- **Qui** : Paul (ou Soren) + repreneur seul
- **Livrable** : NDA envoyé dans l'heure si GO
- **Pas encore de fee** — c'est notre investissement commercial

---

### ② NDA + DATA ROOM + LETTRE DE MISSION — J+1 à J+3

```
   J+0 (call)           J+1                 J+2                 J+3
   ─────────            ─────               ─────               ─────
   NDA envoyé      ➜    NDA signé      ➜    On demande la  ➜   DR ouverte
   par nous             par repreneur       DR à l'AJ           au repreneur
                        retour mail         + Note cadrage      + Lettre mission
                                            envoyée             entre nous signée
```

**Ce qu'on fait, nous** :
- On envoie le NDA **dans l'heure** post-call.
- On rappelle l'AJ pour annoncer qu'on accompagne un candidat sérieux.
- On envoie au repreneur la **note de cadrage** : nom de la société, chiffres, contexte, points clés (5-10 pages).
- On signe avec le repreneur la **lettre de mission** : 7k phase 1 + 7k phase 2 (ou 15-30k+15-30k selon CA cible).

**Ce que reçoit le repreneur** :
- NDA signé.
- Note de cadrage.
- Accès à la data room AJ (lien sécurisé ou physique).
- Lettre de mission à signer.

**Trigger fee** : aucun encore. Tout est conditionnel à l'avancée vers l'offre.

---

### ③ DUE DILIGENCE REPRENEUR — J+3 à J+15

```
   ┌────────────────────────────────────────────────────┐
   │          DATA ROOM AJ                              │
   │  ─────────────────────                             │
   │  • Bilans 3 ans                                    │
   │  • Comptes intermédiaires                          │
   │  • Contrats clés                                   │
   │  • Baux                                            │
   │  • État du personnel                               │
   │  • Passif détaillé                                 │
   │  • Inventaires actifs                              │
   │  • Contentieux                                     │
   └─────────────────┬──────────────────────────────────┘
                     │
                     ▼
   ┌────────────────────────────────────────────────────┐
   │   REPRENEUR ANALYSE                                │
   │  ──────────────────                                │
   │  • Avec son comptable / avocat / banquier          │
   │  • Visite sur site (organisée par nous)            │
   │  • Q&A passées via nous → AJ                       │
   │  • Premier go/no-go interne                        │
   └─────────────────┬──────────────────────────────────┘
                     │
                     ▼
   GO/NO-GO sur dépôt d'offre
```

**Ce qu'on fait, nous** :
- On organise la **visite de site** avec l'AJ et le dirigeant si possible.
- On centralise les **Q&A** : repreneur nous écrit ses questions, on les fait passer à l'AJ, on remonte les réponses.
- On **challenge** la lecture du repreneur : on a vu d'autres deals similaires, on apporte du benchmark.
- On l'aide à **structurer son go/no-go** : valorisation, périmètre repris, prix cible.

**Durée typique** : 10 à 15 jours. C'est ici que le repreneur décide vraiment.

---

### ④ CONSTRUCTION DE L'OFFRE — J+15 à J+25

```
   ┌──────────────────────────────────────────────────────┐
   │          OFFRE = 9 MENTIONS L642-2 du Code commerce  │
   ├──────────────────────────────────────────────────────┤
   │  1. Désignation des biens repris                     │
   │  2. Modalités de la cession (prix, paiement)         │
   │  3. Engagements de l'auteur de l'offre               │
   │  4. Date de réalisation                              │
   │  5. Niveau et perspectives d'emploi (CRITIQUE)       │
   │  6. Garanties offertes                               │
   │  7. Prévisions d'activité et de financement          │
   │  8. Cessions ultérieures envisagées                  │
   │  9. Contrats nécessaires repris                      │
   └──────────────────────────────────────────────────────┘

         3 leviers pour gagner devant le tribunal
         ────────────────────────────────────────
              PRIX  •  PROJET  •  EMPLOI
                            ▲
                            │
                  CRITÈRE #1 du tribunal
```

**Ce qu'on fait, nous** :
- On **rédige** l'offre avec le repreneur (templates internes).
- On le pousse à **maximiser l'emploi** maintenu — c'est ce que le tribunal regarde en premier.
- On **modélise** le plan de financement : apport, dette bancaire, BFR, trésorerie.
- On prépare le **business plan** post-reprise : 3 ans, sobre et crédible.
- On sécurise les **garanties** (cautions bancaires, séquestres si demandés).

**Dépôt de l'offre = TRIGGER FEE #1** : 7k HT (ou 15k+ selon grille).

---

### ⑤ AUDIENCE TRIBUNAL — J+25 à J+35

```
   ┌─────────────────────────────────────────────────────┐
   │              SALLE D'AUDIENCE                       │
   │                                                     │
   │   [Tribunal]      [AJ]      [MJ]      [Procureur]   │
   │       │            │          │            │        │
   │       └────────────┴──────────┴────────────┘        │
   │                       │                             │
   │                       ▼                             │
   │             [Repreneur + Brantham]                  │
   │             [Autres candidats...]                   │
   │                                                     │
   │   Présentation orale offre · Questions ·            │
   │   Délibéré · Jugement                               │
   └─────────────────────────────────────────────────────┘
```

**Avant l'audience** :
- Possibles **demandes de précisions** de l'AJ → offre revisée (offre finale).
- **Surenchère** possible (10 % en 10 jours en LJ).
- On **prépare le pitch oral** du repreneur : 5-10 min devant le tribunal.

**Pendant l'audience** :
- Le tribunal arbitre selon **L642-5** : prix, emploi maintenu, viabilité du projet.
- Pas toujours la plus chère qui gagne — l'**emploi prime**.

**Dépôt offre finale = TRIGGER FEE #2** : 7k HT (ou 15k+ selon grille).

---

### ⑥ CLOSING — J+35 à J+45

```
   ┌─────────────────────────────────────────────────────┐
   │   JUGEMENT DE CESSION (rendu en audience)           │
   └─────────────────┬───────────────────────────────────┘
                     │
                     ▼
   ┌─────────────────────────────────────────────────────┐
   │   PURGE DES SÛRETÉS                                 │
   │   • Hypothèques levées                              │
   │   • Nantissements purgés                            │
   │   • Réserves de propriété traitées                  │
   └─────────────────┬───────────────────────────────────┘
                     │
                     ▼
   ┌─────────────────────────────────────────────────────┐
   │   TRANSFERT ACTIFS + CONTRATS + SALARIÉS            │
   │   • L1224-1 : transfert auto contrats travail repris│
   │   • Contrats commerciaux désignés transférés        │
   │   • Paiement du prix entre les mains du MJ          │
   └─────────────────────────────────────────────────────┘
```

**Closing = TRIGGER FEE VARIABLE** : % sur prix de cession ou forfait, selon ce qu'on a négocié dans la lettre de mission.

---

### ⑦ POST-RACHAT — J+45 à J+90 et au-delà

```
   J+1                J+30               J+90              J+180
   ────               ────               ────              ────
   Stabilisation  ➜   Quick wins    ➜   Restructuring ➜   Run rate
   trésorerie         opérationnels     organisation      profitable
   fournisseurs       cost cutting      reporting
   clients            relations bancaires
```

**Si le repreneur veut**, on continue en mission de conseil :
- Plan 100 jours
- Stabilisation trésorerie / fournisseurs / clients
- PSE / APC si nécessaire
- Reporting mensuel

**Fee** : forfait mensuel ou mission ponctuelle (à définir au cas par cas).

## 4. Ce que tu dis au repreneur en 60 secondes

> "Pour que vous visualisiez bien comment on travaille ensemble : il y a **7 étapes**.
>
> **Aujourd'hui c'est l'étape 1**, le call d'intro qu'on est en train de faire. Si à la fin ça vous parle, je vous envoie un NDA dans l'heure.
>
> **Étape 2**, sous 48h, NDA signé, vous recevez la note de cadrage avec le nom de la boîte, les chiffres détaillés, et l'accès à la data room. On signe en parallèle une lettre de mission entre vous et nous.
>
> **Étape 3**, 10 à 15 jours de due diligence. Vous analysez avec votre comptable, votre avocat. On organise la visite de site, on fait passer vos questions à l'administrateur. À la fin, vous décidez : on dépose une offre ou pas.
>
> **Étape 4**, on construit l'offre ensemble — c'est un format réglementé, neuf mentions obligatoires, avec un plan de financement et surtout un volet emploi qui est le critère numéro 1 du tribunal. Dépôt de l'offre, c'est notre première facturation.
>
> **Étape 5**, deux à trois semaines plus tard, audience tribunal. Possible offre revisée. Le juge tranche entre les candidats. Deuxième facturation.
>
> **Étape 6**, closing. Jugement, purge des sûretés, transfert des actifs et des contrats. Et là vous êtes propriétaire.
>
> **Étape 7**, optionnelle, on peut continuer à vous accompagner sur la restructuration post-acquisition.
>
> Total : 6 à 10 semaines de bout en bout. Vous avez des questions sur le déroulé ?"

**Trois leviers, un critère prime : EMPLOI.**
Si tu te souviens d'une seule chose : le tribunal préfère une offre moins chère qui sauve plus d'emplois. Toujours.

## 6. Drill mémoire — 5 minutes/jour pendant 7 jours

**Jour 1-2** : redessine la timeline (les 7 phases) sans regarder.
**Jour 3-4** : redessine + écris durée + qui pilote pour chaque phase.
**Jour 5-6** : redessine + ajoute les triggers fee.
**Jour 7** : raconte les 60 secondes au repreneur (script section 4) à voix haute, sans lire.

Tu dois pouvoir, à n'importe quel moment du call, savoir :
- Où on en est dans le process
- Qu'est-ce qui suit
- Quand on facture
- Qui pilote la phase suivante
