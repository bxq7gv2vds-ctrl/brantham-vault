# Due Diligence Checklist — 40+ Points

## Financials (15 points)

- [ ] Last 3 years audited financials (P&L, B/S, CF)
- [ ] Monthly financials (last 12 months)
- [ ] Revenue breakdown (by customer, product, geography)
- [ ] Cost breakdown (COGS, OpEx, by department)
- [ ] Cash conversion cycle analysis
- [ ] Accounts receivable aging (DSO, bad debt reserve)
- [ ] Accounts payable terms
- [ ] Inventory (if applicable) — age, turnover, write-downs
- [ ] Capital expenditures (last 3 years, planned)
- [ ] Bank reconciliations (last 12 months)
- [ ] Tax returns (last 3 years)
- [ ] Deferred revenue (if SaaS)
- [ ] Subscription metrics (MRR, ARR, churn, LTV)
- [ ] Budget vs actuals (forecast accuracy)
- [ ] Key assumptions (growth rate, margins, tax rate)

## Legal & Regulatory (10 points)

- [ ] Articles of association / corporate governance
- [ ] Litigation history (current + last 3 years)
- [ ] Regulatory approvals / licenses required
- [ ] Data protection compliance (GDPR, CCPA)
- [ ] Insurance policies (D&O, general liability, keyman)
- [ ] Antitrust / competition law review
- [ ] Environmental liabilities / permits
- [ ] Tax compliance (VAT, income tax, payroll)
- [ ] Industry-specific regulations (banking, pharma, etc.)
- [ ] Material contracts under review (below)

## Contracts & Counterparties (8 points)

- [ ] Top 20 customer contracts (terms, pricing, renewal)
- [ ] Vendor / supplier contracts (lock-in, termination clauses)
- [ ] Employment agreements (exec, management, key persons)
- [ ] Lease agreements (real estate, equipment)
- [ ] Financing agreements (loans, credit lines)
- [ ] Material partnerships / distribution deals
- [ ] IP licensing / third-party agreements
- [ ] Change of control clauses (critical!)

## Intellectual Property (5 points)

- [ ] Patents (granted, pending, in force, licenses)
- [ ] Trademarks (registered, pending)
- [ ] Copyrights (software, content, registrations)
- [ ] Trade secrets / know-how (documentation, protection)
- [ ] IP indemnification from third parties

## Operations & Technology (7 points)

- [ ] IT infrastructure (servers, cloud, backups)
- [ ] Cybersecurity audit (past breaches?)
- [ ] Key software systems & vendors
- [ ] Data center / hosting agreements
- [ ] Business continuity / disaster recovery plan
- [ ] Key suppliers (single source risks?)
- [ ] Capacity analysis (growth runway)

## People & Organization (8 points)

- [ ] Org chart + key person dependencies
- [ ] Executive team bios + incentive structures
- [ ] Employment contracts (non-competes, severance)
- [ ] Headcount by function (salary levels, benefits)
- [ ] Turnover rate (voluntary + involuntary)
- [ ] Bonus / incentive plans (vesting, cliffs)
- [ ] Pension / benefits liabilities
- [ ] Retention agreements (post-close, key people)

## Customers & Market (6 points)

- [ ] Top 20 customers (concentration, contract terms, lifetime value)
- [ ] Customer acquisition cost (CAC) + payback period
- [ ] Net retention rate (NRR) / expansion revenue
- [ ] Churn rate (by cohort, by reason)
- [ ] Market size + TAM analysis
- [ ] Competitive positioning + SWOT
