# EBITDA Multiples — SaaS Valuation Quick Reference

| ARR Bracket | Gross Margin | Net Retention | Multiple Range (EBITDA) | Typical Price Adj. |
|---|---|---|---|---|
| <$1M | 60-70% | N/A | 2-3x | -30% (risky) |
| $1–5M | 70-75% | 90-100% | 3-5x | Baseline |
| $5–10M | 75-80% | 100-110% | 4-7x | +15% (proven) |
| $10–25M | 80%+ | 110%+ | 6-10x | +30% (strategic) |
| $25M+ | 80%+ | 110%+ | 8-15x | +50% (scale) |

**Adjust for:**
- Customer concentration (-20% if top 5 = 60%+ revenue)
- Churn momentum (+10% if <3%, -15% if >8%)
- MAGIC number >0.75 (+20%)
- Tech debt / single founder (-15%)

**Rule of thumb:** Enterprise Value = (ARR × EBITDA multiple) + earnout upside
