## QUICK CHECKLIST (Week 1 MUST-HAVE)

- [ ] Audited financials (3 years)
- [ ] Cap table (fully diluted)

- [ ] Cap table waterfall (3 scenarios)
- [ ] Employee agreements (all IP signed?)

**Target**: By end of Week 1, Buyer can answer: "Is this financially sound? Are the customers real? Can we afford the people? Is the IP clean?"

## USING THIS TEMPLATE

1. **Copy folder structure** above into Google Drive / Datasite
2. **Week 1**: Fill T1 only. Save time.
3. **Share read-only link** with buyer (via NDA)
4. **Add T2 items** as buyer asks questions
5. **Add T3/T4/T5** if buyer is serious (LOI stage)

**Pro tip**: Use this as your own checklist BEFORE soft shop. Identify gaps. Fix them.
