# AUDIT READINESS — 5 MIN CHECK

**À utiliser maintenant.** Honnête. Coche d'abord seul, puis discute avec fondateurs.

---

## SECTION A : Financials (Deal-Killer si <3/5)

- [ ] Financials T0 = clean (P&L, cash, AR aging). Pas d'auditor surprises.
- [ ] Cap table = totalement clair (shares, vesting, options ouvertes, warrant exercices).
- [ ] ARR = défini sans ambiguïté (cash collected? ou accrual?). Auditor agrees.
- [ ] 24m histor de financials (clean, un-audited ok). Aucun red flag (plateau, churn >10%).
- [ ] Tax position = clair (aucun payroll liability, aucun sales tax issue surprenant).

**Score:** __ / 5  
**Walk-away if <3.** = risque de dead deal à due diligence.

---

## SECTION B : Produit & Tech (Deal-Killer si <3/5)

- [ ] Code = "buyer-ready" (peut être maintenu par ingénieur moyen post-close).
- [ ] Tech debt = inventorié (aucun "ticking time bomb"). Pas de hard stop fondateur.
- [ ] Architecture = bien documentée (ne pas compter sur toi à l'explic). Diagrams, ADRs.
- [ ] Dépendances critiques = <3 external (SaaS, APIs, data feeds). Pas de single-point-of-failure.
- [ ] Customer data = auditée pour privacy/compliance (GDPR, CCPA, SOC 2 si pertinent).

**Score:** __ / 5  
**Walk-away if <3.** = buyer découvre "dette technique cachée" → renegociation or kill.

---

## SECTION C : Traction & Market Fit

- [ ] ARR = >$500k, ou >30% QoQ growth + $100k ARR minimum.
- [ ] NRR = >100% (net negative churn). Ou au minimum, stabilité 12m.
- [ ] Customer concentration = top 3 clients <60% ARR. Diversité acceptable.
- [ ] CAC payback = <18 mois. Ou unit economics <défendable.
- [ ] Marché = identifiable (TAM, SAM, SOM avec sources). Pas "we are a uniorn" vibes.

**Score:** __ / 5  
**Walk-away if <3.** = pas assez de traction pour soft shop. Attend ou pivot.

---

## SECTION D : Founder & Team Alignment

- [ ] **All founders** = alignés sur exit (prix, timing, earnout risk). Pas de "veto founder".
- [ ] Key person = replaceable (pas le seul qui comprend). Succession ready.
- [ ] Vesting = clif <12m ou accélération post-close. Pas de "founder milking equity".
- [ ] Cap table = <4 parties. Pas de founders with <10% (complicates close).
- [ ] Advisor board = silencieux (aucun vetoing advisor avec droit de signature).

**Score:** __ / 5  
**Walk-away if <3.** = political risk dans le deal. Founder misalignment kills 40% of SaaS exits.

---

## SECTION E : Legal & Compliance

- [ ] Customer contracts = "standard" (aucun customer avec "you can't sell without approval").
- [ ] Vendor contracts = non-exclusive, transferable (SaaS, data, integrations tous OK).
- [ ] IP = consolidated (no "co-founder left, we lost access to his IP" vibes).
- [ ] Litigation = zero pending. Aucun "this lawsuit could be a problem" talk.
- [ ] Employment = compliant (no payroll taxes owing, no worker classification risk).

**Score:** __ / 5  
**Walk-away if <3.** = diligence uncover liabilities >$500k or blocking issue.

---

## SECTION F : Documentation & Process Readiness

- [ ] Cap table template = 100% filled + signed by all (shares, vesting, strikes).
- [ ] Data room checklist = planned (you've read [Data Room 60 Items](data_room_checklist_60items.md)).
- [ ] NDA draft = prepped (ready to send to first buyer).
- [ ] Teaser deck = outlined (you know what slide 1-5 look like, no numbers yet).
- [ ] Advisor = identified (broker/counsel, if needed).

**Score:** __ / 5  
**Walk-away if <2.** = you're not process-ready. Soft shop will expose you.

---

## DECISION MATRIX

| Total Score | Decision | Next Step |
|-------------|----------|-----------|
| **25-30** | **🟢 READY** | Start soft shop. Week 1 = buyer list + teaser deck. |
| **20-25** | **🟡 CONDITIONAL** | Fix 2-3 critical gaps first. Likely = cap table + founder alignment. Then soft shop. |
| **<20** | **🔴 NOT READY** | Pause sale process. Fix financials/cap table/founder alignment. Re-audit in 4 weeks. |
