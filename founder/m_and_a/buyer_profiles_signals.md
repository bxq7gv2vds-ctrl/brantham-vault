# Profils d'Acheteurs — Signaux & Motivations

**Utilité** : comprendre le contexte du buyer, anticiper ses priorités, tailler ta négociation.

---

## Type 1 : Strategic Corporate Buyer (Concurrent avec trésorerie)

**Profile** : Entreprise établie (P&G, Veolia, Philips) acquérant pour synergies.

### Motivations
- Accès à technologie/talent
- Cross-selling (leur base client → tes produits)
- Éliminer compétiteur
- Acquérir marché géographique/vertical

### Signaux
```
✓ Rapide due diligence (ont expertise interne)
✓ Focus: réduction coûts, augmentation revenu
✓ Pricing: 5–8× ARR (synergies justifient premium)
✓ Closing: 8–12 semaines (approvals internes lentes, mais peu de surprises)
✓ Earnout: rarement, préfèrent paiement à la clôture
✓ Retention: veulent founders/team pendant 2–3 ans
```

### Opportunités Négociation
```
✓ Emphasize synergies (réduction coûts, cross-sell opportunity)
✓ Argue team value (you can't replace engineers)
✓ Speed: faster close = less renegotiation risk
✓ Reverse termination fee: corporate buyers have reputational skin in game
```

### Pièges
```
❌ Integration planning dragged on (no urgency post-close)
❌ Founders stuck in limbo ("waiting to see if we fit culturally")
❌ Earnout based on buyer controlling metrics (revenue targets they can manipulate)
```

## Type 2 : Financial Sponsor / PE Firm

**Profile** : Private equity (Blackstone, KKR, Bessemer) levant $500M+ fund.

### Motivations
- Arbitrage de valeur: acheter 5×, revendre 8× dans 5–7 ans
- Ajouter synergies opérationnelles (consolider verticale)
- Restructuration (couper coûts, concentrer revenu rentable)

### Signaux
```
✓ Pricing: souvent 5–6× ARR baseline (PE = structured, data-driven)
✓ Earnout: 20–30% deal price courant (PE wants alignment)
✓ Metrics: EBITDA, cash conversion, unit economics (pas growth for growth's sake)
✓ Closing: 12–16 weeks (due diligence extensive, financing complex)
✓ Retention: 2–3 year earnout = founders locked in
✓ Board: PE adds chairman, restructures operations
```

### Opportunités Négociation
```
✓ Pitch operational upside (we can 3× EBITDA margin with 200 people)
✓ Argue defensibility (moat = recurring revenue = higher exit multiple)
✓ Earnout: negotiate seller-controllable metrics (customer retention, gross margin)
✓ Incentive align: founders want 2–3x earnout upside
```

### Pièges
```
❌ PE will cut costs aggressively (can harm revenue/culture)
❌ Earnout often clawed-back if acquired PE seller fails to meet targets
❌ Board seats often taken from founders (loss of control)
❌ 100-day plan can destroy customer relationships (aggressive sales tactics)
```

## Type 3 : Venture Capital Buyer (Growth-stage competitor acquiring)

**Profile** : Well-funded startup (Figma, Canva, Discord) buying complementary product.

### Motivations
- Product synergies (consolidate feature parity)
- Land-and-expand (cross-sell to user base)
- Talent acquisition (hire team without market talent wars)
- Eliminate competitor

### Signals
```
✓ Pricing: 6–10× ARR (venture buyers = growth-obsessed, less margin focus)
✓ Earnout: <10% (venture wants clean break, not strings attached)
✓ Speed: 6–8 weeks (venture = decisive, smaller boards)
✓ Retention: mixed (might gut team 6m post-close if "not cultural fit")
✓ Stock component: common (give you upside on THEIR exit)
```

### Opportunités Négociation
```
✓ Emphasize user crossover (we have segment you don't have)
✓ Talent is asset (argue for retention bonuses for team)
✓ Growth story (not cutting costs, but accelerating)
✓ Secondary stock sales (for early employees, advisors)
```

### Pièges
```
❌ Stock consideration = illiquid (might never go public/IPO)
❌ Earnout tied to venture buyer's IPO timeline (could be 5+ years)
❌ Quick team rotation post-close (some venture buyers gut acquisition teams)
❌ Vesting resets (if you get stock, usually 4-year vest)
```

## Type 4 : Financial Buyer (SPAC, CFO-led)

**Profile** : Financial investor (SPAC, hedge fund, fund manager) buying ROI only.

### Motivations
- Cash flow returns (EBITDA multiple arbitrage)
- Tax plays (write-off debt structure)
- Dividend recap (load with debt, pay dividend to LP)

### Signals
```
✓ Pricing: 4–5.5× ARR (financial buyers = lowest price, highest rigor)
✓ Earnout: 25–35% common (lowest upfront, max incentive alignment)
✓ Focus: EBITDA, FCF, cash conversion, margins
✓ Closing: 16–24 weeks (financial due diligence extensive)
✓ Retention: only if required for operations (no strategic need)
✓ Debt: likely loading with debt (leverage play)
```

### Opportunités Négociation
```
✓ Pitch margin upside (we can cut opex 20%, take EBITDA to 40%)
✓ Argue cash generation (recurring revenue, predictable cash)
✓ Earnout: tie to EBITDA/FCF (metric you control)
✓ Multiple: lower than strategic/venture, so earnout comp for it
```

### Pièges
```
❌ Debt loading = leverage risk (any recession → bankruptcy)
❌ Earnout clawed-back if EBITDA targets missed
❌ Dividend recaps = cash leave company (no reinvestment)
❌ Financial buyers often flip 4–5 year (no long-term vision)
```

## Type 5 : Acqui-hire (Hiring for Talent, Not Business)

**Profile** : Big tech company (Google, Meta, Apple) acquiring for engineer team.

### Motivations
- Hire team without recruiting overhead
- Acquire specific domain expertise
- Prevent competitors from hiring talent

### Signals
```
✓ Pricing: 1–2× ARR (low valuation, paying for people not business)
✓ Earnout: rare (no business to earn-out on)
✓ Integration: shut down product, move team into buyer
✓ Retention: 1–2 year vest + bonuses to stay through transition
✓ Speed: 2–4 months (buyer just needs contract signed)
✓ Focus: team quality, engineering output, architectural decisions
```

### Opportunités Négociation
```
✓ Emphasize team depth (your engineers are all $300k+ talent caliber)
✓ Retention: negotiate signing bonuses, vesting cliffs
✓ Package deal: founders + full team + advisors
✓ Post-close role: negotiate staying on, not being "managed out"
```

### Pièges
```
❌ Business dies (product shut down)
❌ Team dispersed (not everyone places in buyer org)
❌ Low valuation (if you had traction, would get more)
❌ Retention vesting: if you leave early, vesting stalls
```

## Quick Buyer Classification

| Buyer Type | Typical Price | Earnout % | Speed | Motivation |
|------------|---------------|-----------|-------|------------|
| Strategic | 6–8× | <10% | 8–12w | Synergies |
| PE | 5–6× | 20–30% | 12–16w | Arbitrage |
| Venture | 6–10× | <10% | 6–8w | Product fit |
| Financial | 4–5.5× | 25–35% | 16–24w | EBITDA |
| Acqui-hire | 1–2× | rare | 2–4w | Talent |

---

## Negotiation by Buyer Type

**Strategic**: Play synergies, demand premium multiple
**PE**: Demand earnout aligned to controllable metrics
**Venture**: Negotiate stock lockup & upside participation
**Financial**: Tie earn-out to EBITDA/FCF you control
**Acqui-hire**: Maximize founder retention package
