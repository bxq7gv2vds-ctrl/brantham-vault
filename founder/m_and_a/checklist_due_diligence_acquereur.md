# Due Diligence — Checklist Acquéreur

**Utilité** : liste de contrôle exhaustive pour un acheteur avant de finaliser une acquisition.

## Financiers
- [ ] 3 ans de P&L audités (ou compilés)
- [ ] Trésorerie actuelle & prévisions 13 semaines
- [ ] ARR/MRR par produit, taux de churn par cohorte
- [ ] Customer concentration: top 10 clients = % du revenu
- [ ] Replicatability du revenu (contracts vs opportunistic)
- [ ] Unearned revenue (accruals) par segment
- [ ] All-in CAC par canal, LTV, payback period
- [ ] Historical growth rate vs industry benchmarks

## Juridiques
- [ ] Articles of incorporation & bylaws
- [ ] Equity cap table (tous les rounds, options, conversibles)
- [ ] Shareholder agreements & voting rights
- [ ] Pending litigation (client, employee, IP)
- [ ] Contracts clients: durée, termination clauses, prix
- [ ] Contracts vendeurs: lock-in, price escalation
- [ ] Leases (locaux, équipements): transferability
- [ ] IP ownership: brevets, marques, domaines, code
- [ ] Employment agreements (clés + non-competes existants)
- [ ] Regulatory licenses & permits

## Techniques
- [ ] Architecture produit & tech debt assessment
- [ ] Code quality audit (test coverage %, cyclomatic complexity)
- [ ] Security audit (pentest, CVE history)
- [ ] Scalability evaluation (infrastructure, DB)
- [ ] Dependencies analysis (open source licenses, vendor lock-in)
- [ ] Data privacy compliance (GDPR, CCPA, PII audit)
- [ ] Disaster recovery & backup strategy

## Commercial
- [ ] Customer interviews: satisfaction, switching cost, renewal likelihood
- [ ] Sales pipeline (qualified vs speculative)
- [ ] Product roadmap credibility
- [ ] Competitive positioning & moat strength
- [ ] Team depth (founder dependency?)
- [ ] Key person flight risk assessment

## Post-Acquisition
- [ ] Transition services agreement draft
- [ ] Integration plan (first 100 days timeline)
- [ ] Synergy quantification (cost cuts, revenue upside)
- [ ] Key employee retention plan

---

**Timing** : 2-4 semaines (parallèle)
**Coût typique** : €50k–€200k (avocats, accountants, tech auditors)
