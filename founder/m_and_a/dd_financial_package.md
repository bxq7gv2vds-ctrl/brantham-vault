# Financial Due Diligence Package

**What buyers need to verify your numbers:**

| Document | Required? | Notes |
|---|---|---|
| **Tax Returns** | YES | Federal + state, 3 years, signed originals |
| **Audited/Reviewed P&L** | YES (or reviewed) | If not audited, compile with accountant |
| **Normalized EBITDA** | YES | Your add-backs, explained |
| **Cap Table** | YES | Current, all equity accounted |
| **Customer Contracts** | YES | Sample major deals |
| **Revenue Schedule** | YES | Top-20 customers, MRR/ARR breakdown |
| **Expense Breakdown** | YES | COGS, OpEx detail, per-department |
| **Bank Statements** | YES | Last 24 months (prove cash in) |
| **Balance Sheet** | YES | Last 3 quarters |
| **Tax Correspondence** | YES (if any) | IRS notices, audit letters, liens |
| **Payroll Reports** | YES | W-2 filings, 941 quarterly |
| **Board Minutes** | YES | Financial approvals, variance reports |
| **Debt Schedule** | YES | All loans, terms, personal guarantees |

**Buyer will:**
1. Verify revenue by sampling customer contracts
2. Verify COGS vs. gross margin trend
3. Verify OpEx allocation
4. Stress-test your growth assumptions
5. Cross-check tax returns vs. books

**Common findings:**
- Revenue overstated (churn not accounted)
- EBITDA add-backs not justified
- Cap table inconsistencies
- Undisclosed debt

**Timeline:** Accountant + you should have this ready 4 weeks before soft shop.
