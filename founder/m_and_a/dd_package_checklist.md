# Due Diligence Package Checklist ‚Äî Dataroom Prep

**Organize all docs in folders. Gate access behind NDA + sign-in.**

---

## **FINANCIAL**
- [ ] 3-year tax returns (federal + state)
- [ ] 3-year P&L (audited or reviewed)
- [ ] Balance sheet (last 3 quarters)
- [ ] Cap table (current, all equity, options, SAFEs)
- [ ] Board minutes (last 24 months)
- [ ] Budget vs. actual reports
- [ ] Customer contracts (sample, major deals)
- [ ] Vendor contracts (>$10k annual)
- [ ] Loan agreements + bank statements

---

## **LEGAL**
- [ ] Formation docs (Articles of incorporation, bylaws)
- [ ] Shareholder agreements
- [ ] Option plan + grant letters
- [ ] Employment agreements (key people)
- [ ] Contractor agreements
- [ ] NDAs (employee, customer, vendor)
- [ ] IP assignment agreements
- [ ] Material contracts (top 10 by value/importance)

---

## **PRODUCT & TECHNOLOGY**
- [ ] Architecture diagram
- [ ] Tech debt assessment
- [ ] Security audit / penetration test report
- [ ] Dependency scan (npm audit, pip audit)
- [ ] Uptime metrics (12-month SLA report)
- [ ] Product roadmap (next 12 months)
- [ ] Code repository access (via GitHub/GitLab)
- [ ] Database schema documentation

---

## **OPERATIONS**
- [ ] Employee roster + compensation
- [ ] Org chart
- [ ] Customer list (anonymized if needed)
- [ ] Insurance policies (D&O, E&O, cyber)
- [ ] Real estate lease or ownership docs
- [ ] Key supplier agreements
- [ ] Facility certifications (ISO, SOC2)

---
