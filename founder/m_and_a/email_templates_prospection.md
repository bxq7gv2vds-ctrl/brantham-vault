# Email Templates — Prospection Acquéreurs

**Objectif :** 3 templates pour différents profils (acquéreur stratégique, PE, acquéreur marché adjacent).

---

## Version 1 — Acquéreur Stratégique (synergies produit)

```
Subject: Opportunité acquisition — [NOM_STARTUP] + [NOM_ACQUÉREUR] = croissance X

Bonjour [NOM_CONTACT],

Je vous envoie cet email car votre plateforme SaaS [PRODUIT_ACQUÉREUR] a un overlap 
naturel avec ce que nous avons construit chez [MON_STARTUP].

Nous servons [SEGMENT_CLIENT] avec [VALUE_PROP]. En 18 mois, nous avons :
• [METRIC_1] : [RÉSULTAT] (ex: ARR passé de $50k à $2.1M)
• [METRIC_2] : [RÉSULTAT] (ex: NRR = 130%)
• [METRIC_3] : [RÉSULTAT] (ex: customer acquisition cost = $X)

Vos clients auraient accès immédiat à [FEATURE/WORKFLOW] sans dev interne.
Nos clients gagneraient [BÉNÉFICE] via votre infrastructure.

Je serais heureux de partager un teaser (1 page) sous NDA si vous êtes intéressé.
Aucune obligation, juste une discussion exploratoire.

Libre pour 20 min cette semaine?

Cordialement,
[NOM]
[TITLE]
[PHONE]
```

## Version 2 — PE / Consolidateur (synergies opérationnelles)

```
Subject: Profil acquisition pour portefeuille [SECTEUR]

Bonjour [NOM_CONTACT],

[NOM_ACQUÉREUR] cherche à consolider les plateformes [SECTEUR]. 
Notre startup pourrait être un fit excellent pour votre portefeuille.

Nous opérons dans [SEGMENT] avec une position unique :

Chiffres clés :
• Revenue : $X annuel, croissance Y% YoY
• Profitabilité : [EBITDA margin ou trajectory]
• Customer base : [N] clients, [CHURN%], [NRR%]
• Team : [N] FTE, [DOMAIN_EXPERTISE]

Synergies potentielles :
1. Cross-sell via portfolio existant (+[X%] revenue possible)
2. Cost reduction ([X%] SG&A via partagé)
3. Talent / IP pour [AUTRE_PORTFOLIO_CO]

Intéressé par un premier appel de découverte? 

Dispo [DATES].

Cordialement,
[NOM]
```

## Version 3 — Acquéreur Marché Adjacent (vertical expansion)

```
Subject: Expansion [VERTICAL] — opportunité acquisition clé en main

Bonjour [NOM_CONTACT],

Vous déployez [PRODUIT] auprès de [VERTICAL_ACQUÉREUR]. 
Nous avons une startup opérationnelle dans [VERTICAL_ADJACENT] avec les mêmes besoins.

Bref profil :
• Marché : [SEGMENT], TAM = $[X]B
• Notre position : [DEFENSIBILITY] (cf. [MOAT: patents, data, network])
• Revenue run rate : $[X]M, expanding into [ADJACENT_VERTICALS]
• Customers : [LOGOS_NOTABLES ou secteurs clés]

Une acquisition nous permettrait :
- Accélérer product roadmap (vs. fundraise)
- Accès immédiat à [DISTRIBUTION / RESOURCES]
- Leverage your [TECHNOLOGY / TEAM / BRAND]

Je serais ravi de discuter fit produit + commercial.

Libre pour un call de 20 min?

[NOM]
```

---

## Notes de déploiement

| Élément | Conseil |
|---------|---------|
| **Subject line** | Mettre un bénéfice ou métrique (vs. "Acquisition opportunity"). Maximise open rate. |
| **Paragraphe 1** | Établir credibilité immédiate : pourquoi lui, pourquoi maintenant. Pas d'intro générique. |
| **Métriques** | Utiliser chiffres réels. Comparaison à benchmark du secteur si possible. |
| **Synergies** | Spécifier 2-3 combinaisons concrètes (pas "natural fit"). |
| **CTA** | Demander 20 min, pas une réunion de 1h. Moins intimidant. |
| **Timing** | Envoyer Tue-Thu 9-11am. Éviter Lun, Ven, après 5pm. |
| **Follow-up** | 3 jours après = relance "juste pour confirmer" + teaser. Pas spammé. |
