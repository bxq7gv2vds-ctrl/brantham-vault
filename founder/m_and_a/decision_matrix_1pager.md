# M&A Decision Matrix — 1-Pager

**Utilité** : outil d'aide à la décision quand multiple options se présentent.

---

## Readiness Score (Avant de Lancer)

Rate 1–10 on each. Scoring:
- **<6 average** = Not ready. Build more before selling.
- **6–7.5** = Ready-ish. Can explore, but from position of strength.
- **7.5+** = Ready. Can run aggressive process.

| Criteria | Score (1–10) | Notes |
|----------|------------|-------|
| **Product-Market Fit** | ___ | 50%+ YoY growth? Repeatable revenue? |
| **Traction** | ___ | $1M+ ARR? Predictable, recurring? |
| **Team Depth** | ___ | No founder bottleneck? Bench has 3+ leaders? |
| **Financial Clarity** | ___ | Clean financials? No shocking liabilities? |
| **IP/Moat** | ___ | Can you explain defensibility in 1 sentence? |
| **Customer Concentration** | ___ | Top 5 < 40% of revenue? |
| **Technical State** | ___ | Codebase is clean? Scaling path clear? |
| **Board Alignment** | ___ | Investors all want this exit? |

**Calculation:** Sum ÷ 8 = Readiness Score

## Buyer Type Decision Tree

```
START HERE: "Which buyer is best for us?"

├─ Do we want the HIGHEST PRICE?
│  └─ YES → Strategic buyer (6–8× ARR)
│           (emphasize synergies)
│
├─ Do we want SPEED & minimal integration?
│  └─ YES → Venture buyer (6–10× ARR, 6–8 weeks)
│           (stock component OK)
│
├─ Do we want OPERATIONAL LEVERAGE & multiple expansion?
│  └─ YES → PE buyer (5–6× ARR, earnout 20–30%)
│           (accept 2–3 year runway post-close)
│
├─ Do we want CASH NOW & predictable returns?
│  └─ YES → Financial buyer (4–5.5× ARR)
│           (accept lower multiple for certainty)
│
└─ Is the BUSINESS struggling but TEAM is gold?
   └─ YES → Acqui-hire (1–2× ARR, job offers)
            (only if no other option)
```

## Price Negotiation Decision Tree

```
                    Our asking price
                           ↓
        What's our BATNA? (walk-away value)
                           ↓
        ┌──────────────────┼──────────────────┐
        ↓                  ↓                  ↓
    < BATNA × 0.85   BATNA to BATNA × 1.2  > BATNA × 1.2
        ↓                  ↓                  ↓
      WALK            NEGOTIATE            ACCEPT
   (insufficient)     (zone of value)   (excellent outcome)
   
   Exception: if 2 competing bids, leverage = ask 1.3–1.5× BATNA
```

## Deal Structure Decision: Stock vs Asset Sale

```
                   Legal Structure Decision

    ┌────────────────────────────────────────┐
    │ Do buyer's liabilities exceed 10% price?
    └────────────────────────────────────────┘
                           ↓
                    YES            NO
                     ↓               ↓
               ASSET SALE    STOCK SALE ← Preferred
               (cherry-pick)  (simplicity)
               
    Asset Sale Triggers:
    ✗ Pending litigation (avoid buyer inheriting)
    ✗ Debt levels high (buyer only assumes what assigned)
    ✗ Regulatory uncertainty (structure to isolate risk)
    
    Stock Sale Benefits:
    ✓ 90% faster (no customer consent)
    ✓ Lower taxes (capital gains)
    ✓ Cleaner (one-step transaction)
```

## Earnout: Should You Accept It?

```
                    "Earnout proposed"
                           ↓
        ┌──────────────────┴──────────────────┐
        ↓                                      ↓
    What % of deal?                  Metrics seller-controllable?
        ↓                                      ↓
    <20%? YES✓ / NO✗              ✓ YES / ✗ NO (buyer-controlled)
        ↓                                      ↓
      GOOD                                   BAD
    (acceptable)                       (high risk)
    
    ✓ Accept if: <20% price, metric you control, cash paid upfront
    ✗ Reject if: >30% price, metric buyer controls, buyer can manipulate
```

## Due Diligence Red Flags: Abort If...

```
Critical Issues (Walk away)        Negotiable Issues (Renegotiate price)
───────────────────────────        ───────────────────────────────────
❌ Hidden litigation (>10% deal)    ⚠️ Weak IP protection
❌ Material undisclosed liab.       ⚠️ Single key person
❌ Customer concentration >50%      ⚠️ Regulatory uncertainty
   with no stickiness              ⚠️ Tech debt (but team knows)
❌ Revenue inflated/one-time        ⚠️ Churn >10%/month (but fixable)
❌ Founder/investor fight
❌ IP ownership in question
```

**Rule:** 3+ critical issues = walk. 2+ critical = deep negotiation.

## Earnout Metrics: Grade Each Option

| Metric Type | Seller Controllability | Grade | Example |
|-------------|------------------------|-------|---------|
| Revenue | Medium (buyer affects budget) | C+ | "Hit $5M ARR in Y2" |
| Customer Retention | High (sellers control) | A | "Maintain >85% NRR" |
| EBITDA Margin | High (sellers control ops) | A | "Achieve 25% EBITDA margin" |
| Market Share | Low (market controls) | D | "Capture 15% market share" |
| Product Roadmap | Low (buyer may change) | D | "Launch 3 new features" |
| Employee Retention | High (sellers control) | A | "Retain 90% of team 18m" |

**Decision:** Only accept earnout if 2+ A-grade metrics, 0 D-grade.

|--------|-------------|-----------|-------|
| Customer Retention | (Retained $ / Start $) | Monthly | CFO |

**Red flag:** If buyer changes metric definition mid-earnout → escalate to board.

## 1-Minute Decision Framework

**"Should we sell now?"**

```
Checklist:
□ Readiness Score ≥ 7? (product fit, traction, team)
□ Multiple interested buyers? (competition = leverage)
□ Can defend our asking price? (DCF, comps, synergies)
□ BATNA is acceptable? (not desperate to sell)
□ Board aligned? (no veto risk)

If all 5: ✓ YES, run process
If 3–4: ~ MAYBE, depends on buyer offer
If <3: ✗ NO, build more, come back in 12m
```
