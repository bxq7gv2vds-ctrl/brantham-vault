# Checklist Auto-Évaluation Seller

**À faire : 2 semaines avant de lancer les approches.**

Scoring: ✅ Ready | ⚠️ Can fix (< 30 days) | ❌ Blocker (> 60 days ou unfixable)

## A. VALORISATION & FINANCIALS (40 points)

| Item | Score | Notes |
|------|-------|-------|
| Revenue last 12m audited or clean P&L (3 years) | ✅ | Si Q4 > Q3 > Q2 > Q1, croissance visible |
| EBITDA margin > 20% (ou positive trend) | ✅ | Si négatif, buyer demande réconciliation |
| Working capital normalized (AR/AP in order) | ✅ | Sans ça, true-up complique la deal |
| No large one-time costs obscuring run-rate | ✅ | Post-acquisition cleaning penalty |
| Revenue concentration < 30% (top 3 customers) | ✅ | Si > 40%, discount de 10-20% |
| NRR ≥ 100% (SaaS) ou churn < 5%/month (SaaS) | ✅ | Dealbreaker pour most buyers |
| No significant customer refunds pending | ✅ | Discover-able in audit, kills credibility |
| Tax filings clean, no audit flags | ✅ | IRS letter from lawyer reassures buyer |

**Subtotal A:** 8 ✅ = **+40 pts**

## B. ASSETS CLEANNESS (25 points)

| Item | Score | Notes |
|------|-------|-------|
| IP: all IP assigned to company (no co-ownership) | ✅ | Get contractor IP assignment letters ASAP |
| Domains & trademarks properly registered | ✅ | US + main markets (EU if selling to EU buyer) |
| No open-source license issues (GPL, AGPL) | ✅ | Audit code w/ tools (FOSSA, Black Duck) |
| Third-party integrations not dependent on founder | ✅ | Stripe, Twilio, etc. — check who owns keys |
| No material debt (< 30% of deal value) | ✅ | If > 50%, renegotiate payoff schedule |
| No liens, UCC filings, or pledged assets | ✅ | Get UCC search report |
| Data privacy compliant (GDPR, CCPA, SOC 2) | ✅ | SOC 2 takes 3-6 months if missing |

**Subtotal B:** 7 ✅ = **+25 pts**

## C. TEAM & RETENTION (20 points)

| Item | Score | Notes |
|------|-------|-------|
| 2-3 key persons willing to sign 2yr + retention bonus | ✅ | If not, discount 15-25% |
| No imminent departures (key person signals?) | ✅ | Ask: "Would you stay under new owner?" |
| Founder / CEO willing to stay 12-24m post-close | ✅ | Buyer almost always requires this |
| Org chart clean (no dotted lines, unclear reporting) | ✅ | Buyer values clarity |
| Compensation levels market (no Red Sox deals) | ✅ | If payroll bloated, buyer negotiates down |

**Subtotal C:** 5 ✅ = **+20 pts**

## D. OPERATIONS READINESS (15 points)

| Item | Score | Notes |
|------|-------|-------|
| Financial records digitized, audit-ready | ✅ | Spreadsheets are fine if clean; QuickBooks ideal |
| Customer contracts standardized (not all bespoke) | ✅ | If 100 unique contracts, DD takes 6 weeks |
| No pending litigation (civil, employment) | ✅ | Disclose early; surprises kill deals |
| Insurance current (D&O, E&O, liability) | ✅ | Tail coverage available post-close |
| No major environmental / regulatory flags | ✅ | If applicable (healthcare, fintech) get pre-audit |

**Subtotal D:** 5 ✅ = **+15 pts**

| Score | Readiness | Action |
