## 📋 LIVRABLES PRODUITS

### CYCLE 1: INTERACTIVE TOOLS (20 min)

#### 1️⃣ Post-Close Earnout Tracker
**Fichier:** `post_close_earnout_tracker.md`

Ce que c'est :
- Tableau de bord Excel-ready pour KPI post-clôture
- Calculatrice earnout automatisée (formules Python)
- Red flags pour détecter manipulations buyer
- Protocole escalade si écart > 15%

Pourquoi utile :
- Earnout disputes sont la #1 source de conflict post-close
- La plupart des founders laissent buyer contrôler les métriques
- Ce tracker force transparence + documentation

Temps économisé :
- Setup standard: 5-6h (créer spreadsheet, lier données)
- Avec ce template: 45 min (copier, personnaliser, lancer)

---

#### 2️⃣ Buyer Comparison Matrix
**Fichier:** `buyer_comparison_matrix_interactive.md`

Ce que c'est :
- Grille scoring (5-10 buyers side-by-side)
- 5 dimensions: Valuation, Financial Strength, Strategic Fit, Earnout Confidence, Cultural Fit
- Rubric détaillé (1-5 scale per criterion)
- Weighted total score (30% valuation, 20% financial, etc)

Pourquoi utile :
- Founders choisissent le premier buyer qui appelle
- Cette matrice force discipline de sélection
- Identifie red flags précocement (earnout >65%, no track record, etc)

Temps économisé :
- Analyse manuelle: 2-3h par buyer
- Avec ce template: 5 min + formules Excel

---

#### 3️⃣ Deal Decision Tree
**Fichier:** `deal_decision_tree_quick.md`

Ce que c'est :
- 10 questions binaires (YES/NO)
- Chaque branche mène à red flags spécifiques
- Scoring: 0-2 flags = proceed, 3+ = escalate, kill-level = walk

Format :
```
Q1: Valuation ≥ BATNA? → YES: Q2 | NO: Valuation Path
Q2: 80%+ cash at close? → YES: Q3 | NO: Earnout Path
...
Q10: Explain to advisors? → YES: ✅ GREEN LIGHT | NO: ❌ NO-GO
```

Pourquoi utile :
- Tue les bad deals early (90% of regrets are ignored red flags)
- Forces binary decision (not endless discussion)
- Can be done in 2 minutes during LOI review

### CYCLE 2: ADVANCED PLAYBOOKS (20 min)

#### 4️⃣ Objections Handler (Phase-Based)
**Fichier:** `objections_handler_phase_based.md`

Ce que c'est :
- 10 objections courageous (2-3 par phase du deal)
- Chaque objection: Pourquoi ils le disent → Ton rebuttal → Follow-up question
- Format phase-by-phase (Outreach, Initial Conversations, LOI, DD, Final Negotiation)

Exemples :
- Phase 1: "We're not looking to acquire right now"
- Phase 2: "Your valuation is too high"
- Phase 3: "We want 50%+ earnout"
- Phase 4: "Your customer concentration is too high"
- Phase 5: "We're reducing the offer by 10%"

Temps économisé :
- On-the-fly prep: Bégayer, lose negotiation
- Avec ce playbook: Pré-answer, stay calm, reframe

---

#### 5️⃣ Negotiation Scripts (Difficult Conversations)
**Fichier:** `negotiation_scripts_difficult_conversations.md`

Ce que c'est :
- 5 word-for-word scripts pour situations tendues
- Format: Situation → Your opening → Expected pushback → Your response → Next step

Scénarios :
1. **Lowball offer** (30% below ask) — How to counter without caving
2. **Earnout dominates** (60%+) — How to push back
3. **Earnout dispute** (Month 10: "You missed targets") — How to defend with SPA
4. **Team wants to leave** — How to handle post-close key person departure
5. **Integration plan changes** — How to catch bait-and-switch

Temps économisé :
- Unprepared call: Get emotional, accept bad terms
- Avec scripts: Stay calm, reference SPA, offer solutions

---

#### 6️⃣ Buyer Red Flags Detection
**Fichier:** `buyer_red_flags_detection.md`

Ce que c'est :
- 30-flag checklist organized by category (Financial, Deal Structure, Team, Operational, Buyer Behavior, Legal)
- Severity scoring (🟢 Low to 🔴 High to ❌ KILL-LEVEL)
- Remediation guide (Can you fix it? How?)

Format :
```
| Flag | How to check | Severity |
| Earnout >65% | Math | ❌ KILL-LEVEL |
| Earnout = combined metric | SPA | 🔴 High |
| Customer concentration >70% | Top 3 ARR | 🔴 High |
```

Scoring :
- 0 flags = ✅ GO
- 1-2 flags = 🟡 Yellow (get advice)
- 3+ flags = 🔴 Escalate
- 1 kill-level = ❌ WALK

Temps économisé :
- Missing red flags: Post-close regrets (80% of deals have them)
- Avec ce checklist: 5 min diagnosis + escalate early

#### 8️⃣ Cold Email Sequences
**Fichier:** `cold_email_sequences_phase_outreach.md`

Ce que c'est :
- 3 email templates (progression: Hook → Interest → Traction)
- 3 variations per email (warm intro, cold, referral request)
- Response scenarios + rebuttals
- Email cadence (Day 1, 3, 5, 8, 12...)
- Tracking template (Google Sheets)

Email 1 : Initial Outreach
- 5-7 sentences
- Mention something specific about THEM
- Soft ask (soft CTA)

Email 2 : Post-Teaser (48h after opened)
- Move from intro to value prop
- Give 3 specific reasons for acquisition
- Shorter tone (more urgent)

Email 3 : Move to Conversation (after 2+ exchanges)
- Propose valuation range
- Ask about structure (all cash / earnout?)
- Set timeline expectations

Success rate :
- Email 1 → Open: 25-40%
- Email 1 → Reply: 5-10%
- Email 3 → Move to call: 30-50%
- Overall: 20 cold emails → 3-5 meetings

Temps économisé :
- Write from scratch: 3-4h
- Avec ce template: 45 min (personalize + send)

#### 9️⃣ Buyer Targeting Playbook
**Fichier:** `buyer_targeting_prioritization.md`

Ce que c'est :
- Tier buyers by Warmth + Fit (Tier 1 = warm + perfect, Tier 5 = cold + weak)
- Conversion likelihood per tier
- Execution order + timeline
- Team assignment (who does what)
- Common mistakes (blasting cold emails to 100 Tier 5 = wrong)

Tiers :
- **Tier 1** (Warm + Perfect): 40-60% conversion → Call FIRST
- **Tier 2** (Warm + Good): 30-40% → Call NEXT
- **Tier 3** (Cold + Perfect): 15-25% → Then this
- **Tier 4** (Cold + OK): 5-15% → Later
- **Tier 5** (Cold + Weak): <5% → Skip (unless desperate)

Execution order :
- Week 1: Identify Tier 1, ask for intros
- Week 2: Tier 1 calls, start Tier 2 outreach
- Week 3-4: Tier 1 → LOI, Tier 2-3 calls
- Ongoing: Keep feeding pipeline (Tier 4)

Temps économisé :
- Spray and pray: Email 100 people randomly (10% response, wasted time)
- Avec ce playbook: Focus on Tier 1-2 first (40%+ response, faster deals)

## 🎓 LEARNING CAPTURED

### M&A Playbook Patterns
- Buyer objections are predictable by phase (outreach vs LOI vs close)
- Red flags are detectable early (but founders usually ignore them)
- Earnout disputes are THE killer (need independent tracking)
- Cold outreach works only if tiered properly (Tier 1 warm > Tier 4 cold)

### Design Decisions
- Scripts are word-for-word (not bullet points) — founders need confidence
- Checklists are scored (not just lists) — prioritization needed
- Templates are customizable (not prescriptive) — one-size-doesn't-fit-all
- Each template connects to others (`template_buyer_personas.md` → `cold_email_sequences`)
