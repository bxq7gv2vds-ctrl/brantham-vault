# M&A Red Flags — Détection d'anomalies

**Utilité** : identifier rapidement les signaux de risque majeur avant d'engager temps/ressources.

## Financiers
🚩 **Croissance non durée** : pic de 300% suivi de plateau (deal ponctuel plutôt que produit)
🚩 **Churn élevé** : >10%/mois = perte de value post-deal catastrophique
🚩 **CAC augmente, LTV stagne** : espace de prix limité ou saturation marché
🚩 **Revenue concentration** : 1 client = >30% ARR
🚩 **ASP declining** : compression de pricing = perte de leverage
🚩 **Negative cash flow sans chempath clair** : dilution accélérée
🚩 **Contingent liabilities** : provisions cachées, crédits parajudiciaires

## Juridiques
🚩 **Cap table compliquée** : >15 shareholder classes, conversibles non clairs
🚩 **Founders pas alignés** : veto rights, disputes internes
🚩 **IP non cédé** : code = freelancer externe, brevets = cofondateurs
🚩 **Litige actif** : classe action, patent troll, employee claim
🚩 **Contrats sans transferability** : clients qui repartent si acquisition
🚩 **Non-competes faibles** : founders peut redémarrer concurrent dans 6 mois
🚩 **Regulatory amber** : poursuite gouvernementale, licence menacée

## Techniques
🚩 **Tech debt démentiel** : monolithe 500k lines, 0% tests
🚩 **Vendor lock-in** : architecture AWS-propriétaire non migreable
🚩 **Security incidents cachés** : breach non disclosée
🚩 **Architecture non scalable** : capacity plafonnée à 1M users, déjà à 900k
🚩 **Open source obligations** : GPL dans le core (force open-source)
🚩 **Data quality issues** : dups, PII exposée, ERR taux anormalement bas

## Commercial
🚩 **Founder ego-driven roadmap** : "pas de produit market-fit, mais j'aime faire ça"
🚩 **Customer dependency** : top 3 = 70% revenu, tous en contrats annuels (dead-pool post-renewal)
🚩 **Sales team = founder** : zéro pipeline de replacement
🚩 **Product = 1 personne** : genius qui refuse de documenté
🚩 **Board conflict** : investisseurs + founders = factions

## Deal Mechanics
🚩 **Pricing divorced de metrics** : "on vaut $50M sur $2M ARR" sans synergy story
🚩 **Seller financing >30% du prix** : risque de default
🚩 **Earn-out trigger impossible** : metrics buyer ne contrôle pas
🚩 **LOI vague** : definitions floues = disputes post-close
🚩 **Reps partagées** : "seller warrants 95% customer retention" (non-falsifiable)

## Proces & People
🚩 **Founder wants to "stay and help" 3 ans** : burnout mutual garanti
🚩 **No key person retention plan** : critical hires dès jour 1
🚩 **Asymmetric information** : seller dodges technical deep-dives
🚩 **Rushed timeline** : "closing en 6 semaines" = pas le temps de vérifier
🚩 **Advisor conflicts** : même cabinet pour buyer ET seller

---

**Decision rule** : 3+ red flags = pause et audit profond. 5+ = walk.
