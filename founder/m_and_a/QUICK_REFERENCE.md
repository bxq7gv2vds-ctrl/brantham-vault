# M&A Quick Reference Card

Imprimez ceci ou gardez dans votre navigateur pendant le deal.

---

## 🚀 PHASE 1 : PRÉPARATION (Weeks 1-4)

### Your To-Do List
- [ ] Run [Self-Audit](self_audit_checklist.md) (15 min) — Am I ready?
- [ ] Calculate [BATNA](batna_calculator.md) (10 min) — What's my walk-away?
- [ ] Build cap table (2-3 days) — Who owns what, exactly?
- [ ] Read [Playbook Négociation](playbook_negociation_vendeur.md) (30 min)
- [ ] Hire [lawyer + accountant] (1 week) — Call your advisors
- [ ] Build buyer list (10-15 names) — Who might want us?

### Key Docs to Share
- Teaser deck (5-10 slides) → [Template](quickstart_90_days.md)
- NDA (confidentiality) → [Template](template_loi.md)
- Cap table spreadsheet → Check [Data Room Structure](data_room_structure.md)

### Success Metric
You have **3+ qualified buyer discussions** before LOI.

---

## 🤝 PHASE 2 : LOI (Weeks 3-6)

### Your To-Do List
- [ ] Get LOI draft from buyer
- [ ] Counter-propose on: price, earnout %, reps cap
- [ ] Negotiate: [Playbook reference](playbook_negociation_vendeur.md)
- [ ] Board/shareholder approval
- [ ] Sign LOI → **Exclusivity starts now** (usually 60-90 days)

- [LOI Template](template_loi.md) — If buyer doesn't have one
- [Term Sheet Guide](template_term_sheet.md) — Key terms to fight for
