# Due Diligence Seller Checklist — 40 Items à Préparer

**Objectif :** Anticiper 100% des demandes de DD et les préparer *avant* que l'acheteur ne les pose. Transparence = leverage.

---

## FINANCIAL (10 items)

- [ ] **Cap table historique** (founder, investors, prédilution+dilution) + convertible/SAFE terms
- [ ] **Derniers 3 ans financials** (P&L, balance sheet, cash flow statement) + monthly breakdown (derniers 12 mois)
- [ ] **EBITDA ajusté** (add-backs: founder compensation, one-time items, non-recurring costs)
- [ ] **Revenue breakdown** : par customer, segment, région, produit (si applicable)
- [ ] **Unit economics** : CAC, LTV, payback period, NRR, churn (monthly + annual)
- [ ] **Cash runway** : current balance, burn rate, months of runway
- [ ] **Debt/liabilities** : prêts, credit cards, deferred revenue, contingent liabilities
- [ ] **Tax returns** : 2 years (ou 3 si levée de fonds)
- [ ] **Audit trail** : qui a préparé les financials? Externe ou interne?
- [ ] **Working capital estimates** : AR, AP, inventory (si applicable)

---

## LEGAL (10 items)

- [ ] **Certificate of incorporation** + all amendments (y/c stock option plan)
- [ ] **Articles of incorporation & bylaws**
- [ ] **Cap table shareholder agreements** + side letters
- [ ] **Founder vesting schedules** (4-year cliff common? Options?)
- [ ] **Convertible instruments** (SAFE, convertible notes, MFN clauses)
- [ ] **Material contracts** (customer, vendor, partnership >5% revenue, key talent)
- [ ] **IP assignment agreements** (founder → company, contractor IP, university if applicable)
- [ ] **Employment agreements** (cofounders, execs, templates for rest of team)
- [ ] **Litigation history** : lawsuits, claims, disputes (current + resolved)
- [ ] **Regulatory/compliance** : permits, licenses, certifications (SOC2, GDPR, CCPA, etc.)

---

## INTELLECTUAL PROPERTY (5 items)

- [ ] **Patent portfolio** : granted + pending (with prosecution docs)
- [ ] **Trademarks** : registered + pending (US + international if applicable)
- [ ] **Source code escrow agreements** (if applicable)
- [ ] **Third-party IP licenses** (libraries, frameworks, commercial software)
- [ ] **Copyright registrations** (if any)

---

## TECHNOLOGY (8 items)

- [ ] **Tech stack & architecture overview** (1-2 page summary)
- [ ] **Security audit results** (pentest, vulnerability scan, last 2 years)
- [ ] **Data privacy compliance** (GDPR, CCPA, HIPAA if healthcare, etc.)
- [ ] **Disaster recovery & backup plan** (RTO, RPO, last successful test)
- [ ] **Cloud infrastructure costs** (AWS/GCP/Azure monthly spend)
- [ ] **Third-party dependencies** : critical libraries, SaaS tools, data sources
- [ ] **Technical debt assessment** : areas of concern, refactoring needed
- [ ] **API/integration documentation** : if others depend on us, contracts?

---

## CUSTOMERS & CONTRACTS (5 items)

- [ ] **Customer list** : name, revenue, contract start/end, renewal date
- [ ] **Master service agreements / SLAs** : template + any custom ones
- [ ] **Change-of-control clauses** : qui peut terminer post-acquisition?
- [ ] **High-risk contracts** : customers with >10% revenue, option à quitter
- [ ] **Customer concentration risk** : top 5 customers = X% of revenue

---

## EMPLOYEES & HR (3 items)

- [ ] **Org chart** + bios (background, role, salary, equity)
- [ ] **Option pool status** : total granted, vested, unvested, remaining pool
- [ ] **Key person insurance** : exists? Beneficiary? Amount?

---

## OPERATIONAL (4 items)

- [ ] **Board minutes** (derniers 12 mois)
- [ ] **Insurance policies** : D&O, general liability, cyber, errors & omissions
- [ ] **Affiliate agreements** (if any founders/employees have related companies)
- [ ] **Expense/accounting records** : clean books? Receipts? Any odd items?

---

## ANTI-PATTERN CHECK (Optional but important)

- [ ] **Undisclosed liabilities** : spouse lawsuits, tax liens, regulatory investigations?
- [ ] **Related-party transactions** : founder consulting other companies? Conflicts?
- [ ] **Founder IP history** : did we properly acquire IP from ex-employers?
- [ ] **Contract violation risk** : any competitor non-competes that could bite us?
- [ ] **Off-the-books deals** : gray market revenue? Barter? Non-compliant?

---

## Timeline recommandé

- **Week 1-2** : Rassembler tout (avant approche first buyer)
- **Week 3** : Review avocats (pas d'erreurs légales)
- **Week 4** : Set up data room (virtual, Merrill DataSite ou Intralinks)
- **Week 5** : Ready pour NDA + LOI phase

**Golden rule** : Tout délai > 2 jours pour produire un doc = red flag. Si on a pas ça maintenant, on l'aura pas pendant DD. Better to disclose proactively.

---

## Red flags à éviter

- ❌ Financials qui ne match pas bank statements (flag immédiat)
- ❌ Manque de documentation pour contracts > 5% revenue
- ❌ Founder IP non-assignment d'anciens emplois (show-stopper)
- ❌ Tax liens ou judgments non-divulgués (criminal)
- ❌ Customer contracts avec termination rights à acquisitions (ruin deal valuation)
