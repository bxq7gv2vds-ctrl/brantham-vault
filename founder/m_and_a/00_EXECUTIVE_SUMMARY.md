# M&A — 1-PAGE EXECUTIVE SUMMARY

**Utilité** : répondre aux 5 questions critiques en 5 minutes, avec liens vers docs détaillés.

---

## 5 Questions Critiques

### 1. **Devrais-je vendre maintenant ?**
```
✓ 4+ signaux = considérer soft shop
  □ Croissance >50% YoY
  □ Revenue >$1M ARR, churn <7%/m
  □ Produit non founder-dépendant
  □ Multiple options buyer (strategic, PE, venture)
```
→ Détail : [Quickstart 90 jours](quickstart_90_days.md)  
→ Valider : [Red Flags M&A](red_flags_m_et_a.md)

---

### 2. **À quel prix ?**
```
Financial floor: BATNA (alternative sans deal)
  = option levée (dilution %) OU rester indépendant (croissance) OU IPO

Fair value = max(DCF, comparable × multiple)
  DCF:        intrinsic value (revenues, margin, growth)
  Comps:      SaaS multiple = 5-15× ARR (selon croissance)

Anchor price = 1.3× fair value
Walk-away     = 0.8× fair value
```
→ Détail : [Valuation DCF](valuation_dcf_template.md)  
→ Multiples : [SaaS Benchmarks](multiples_saas_benchmarks.md)

---

### 3. **Quel type d'acheteur ?**
| Type | Cherche | Paye | Risque |
|------|---------|------|--------|
| **Strategic** | Synergies | Haut | Moyen (culture) |
| **PE** | Cash flow | Moyen | Élevé (earnout) |
| **Venture** | Produit | Moyen-Haut | Faible (vision) |
| **Financial** | Rendement | Bas | Passif |

→ Détail : [Buyer Profiles](buyer_profiles_signals.md)  
→ Outreach : [Buyer Outreach Playbook](buyer_outreach_playbook.md)

---

### 4. **Stock vs Asset sale ?**
```
Stock sale: je cède 100% ownership
  ✓ Simpler, founder exit
  ✗ Seller warrants toutes les liabilities

Asset sale: je vends produit/clients
  ✓ Contractuel, je garde shells/IP
  ✗ Complexe, tax-inefficient (double taxation)
```
→ Détail : [Deal Structures](deal_structures_stock_vs_asset.md)

## Red Flags — Arrêtez immédiatement si :
🚩 Churn >10%/m  
🚩 Top 3 clients = >70% ARR  
🚩 Pricing divorced de valeur (ex: $50M valuation sur $2M ARR sans synergies)  
🚩 Cap table embrouillée + founders pas alignés  
🚩 Fondateur ego-driven ("je suis indispensable")  

→ Full list : [Red Flags M&A](red_flags_m_et_a.md)

## Templates Essentiels (à préparer)
1. **Cap Table** : shares → $ à chaque valuation
2. **Teaser Deck** : 5-10 slides (problem, traction, why now)
3. **LOI Draft** : key terms (price, earnout %, reps cap)
4. **Data Room** : 60+ items (legal, financials, tech, customer)
5. **Founder NDA** : protection pendant exploration

→ Full templates : [Templates M&A](template_loi.md), [Data Room](data_room_checklist_60items.md)

## Playbook Négociation (tactical)
**Phase 1: First Contact** (semaine 1-2)
```
Toi:   "Intéressé par acquisition ?"
Buyer: "Peut-être, envoie teaser"
→ Teaser neutre, pas de price
```

**Phase 2: LOI** (semaine 3-6)
```
Toi:   [LOI draft] price = $50M, 20% earnout
Buyer: Contre-propose $40M, 30% earnout
→ Anchor à 1.3× fair value, walk-away à BATNA
```

**Phase 3: Due Diligence** (semaine 6-14)
```
Buyer: [1000 questions]
Toi:   Data room + Q&A log
→ Flag issues BEFORE they escalate
```

**Phase 4: Closing** (semaine 14-18)
```
Reps/warranties finalized
Escrow structure agreed
→ Minimum reps cap <1% deal
```

→ Full playbook : [Playbook Négociation](playbook_negociation_vendeur.md)
