## ADJUSTMENT FACTORS

**Your multiple might be higher if:**
- ✓ High growth (>100% YoY) → +3–5x vs. baseline
- ✓ Positive unit economics → +2–3x
- ✓ Strong retention/NPS → +2–3x
- ✓ Defensible moat (IP, network) → +2–4x
- ✓ Strategic fit (acquirer gets synergies) → +2–5x

**Your multiple might be lower if:**
- ✗ Slowing growth (<20% YoY) → −2–4x
- ✗ High churn (>10% MoM) → −2–3x
- ✗ Customer concentration (top 1 = >30%) → −2–4x
- ✗ Technical debt / integration risk → −1–3x
- ✗ Regulatory risk → −2–5x

## WORKED EXAMPLES

### Example 1: Vertical SaaS, High Growth

```
Company: Construction project management SaaS
ARR: $5M
Growth: 120% YoY
Churn: 5% MoM
Gross margin: 75%
EBITDA: -$200K (investing to grow)

Baseline (Vertical SaaS): 8x = $40M
Adjustments:
  + 120% growth: +4x → 12x
  + Positive unit economics: +2x → 14x
  + Strong retention: +2x → 16x
  - Modest customer concentration: -1x → 15x

Fair valuation: $5M × 15x = $75M
```

### Example 2: Infrastructure SaaS, Mature

```
Company: Monitoring/observability platform
ARR: $10M
Growth: 40% YoY
Churn: 3% MoM
Gross margin: 80%
EBITDA: +$500K (profitable)

Baseline (Infrastructure SaaS): 10x = $100M
Adjustments:
  + 40% growth: +1x → 11x
  + Profitable: +2x → 13x
  + Enterprise customers (sticky): +2x → 15x
  - Competitive market: -2x → 13x

Fair valuation: $10M × 13x = $130M
```

### Example 3: Fintech, Regulatory Risk

```
Company: Credit scoring API
ARR: $3M
Growth: 60% YoY
Regulatory: CFPB scrutiny
Gross margin: 70%

Baseline (Fintech): 6x = $18M
Adjustments:
  + 60% growth: +2x → 8x
  + Recurring API revenue: +1x → 9x
  - Regulatory risk/uncertainty: -3x → 6x

Fair valuation: $3M × 6x = $18M
```

## RED FLAGS (Low Multiples)

If you're offered <4x revenue, watch for:
- Buyer thinks growth is slowing
- Hidden churn / customer concentration risk
- Tech debt that requires rewrite
- Regulatory uncertainty
- Weak cultural fit → integration risk

**Response**: Push back with comps or highlight growth trajectory.

---

## BLUE FLAGS (High Multiples)

If offered >15x revenue, consider:
- Acquirer sees specific strategic synergies
- They expect to retain most customers (high confidence)
- Buyer is strategic (not financial) = willing to overpay
- Market is frothy (maybe negotiate faster close before sentiment shifts)

**Action**: Lock in the price quickly; cap any earnout risk.

## HOW TO USE THIS DATA

**To set your anchor:**

1. Find your segment in table above
2. Identify your growth rate
3. Adjust median multiple for your specific situation (+/−)
4. Multiply by ARR to get fair value range
5. Use median of range as opening ask
6. Use top of range as "would need THIS to walk away"

**Example:**
```
SaaS segment: "Vertical SaaS" → baseline 8–10x
Your growth: 80% YoY → add 3x
Adjusted range: 11–13x
Your ARR: $4M
Fair value: $4M × 12x = $48M

Opening ask: $50M
Walk-away: $45M (4M × 11x)
```
