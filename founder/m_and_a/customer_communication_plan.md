# Customer Communication Plan — M&A Announcement

**Timing:** Announce day-of or day-before close (avoid leaks)

---

## **Email Script (To Customers)**

```
Subject: [Company Name] + [Buyer Name] — Exciting Partnership

Hi [Customer Name],

We're thrilled to announce that [Company Name] is joining the 
[Buyer Name] family.

What this means for you:
✓ Same product, same support (no changes)
✓ New integrations + features coming [Month]
✓ Expanded team backing the product
✓ Price: stable for 12 months [if applicable]

[Your founder name] stays as [VP Product / CTO] to ensure smooth transition.

Questions? Contact [Support Email].

Cheers,
[Your name]
```

---

## **Reference Call Script (If Buyer Requests)**

```
"Hi [Customer], thanks for 15 minutes.
Just confirming: you're happy with [Product] today?
→ Any concerns with potential change of ownership?
→ Would [Buyer Company] product line improve your setup?

If answer = YES:
"Perfect. Buyer will be in touch next week with integration roadmap."

If answer = NO / CONCERNED:
"That's fair. We can lock in renewal terms, or discuss features roadmap.
What would make you most comfortable?"
```

**Don't volunteer:** Financial details, earnout terms, non-competes
