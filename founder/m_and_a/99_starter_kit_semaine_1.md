# Starter Kit M&A — Les 7 jours décisifs

**But :** Passer de "on envisage de vendre" à "on court après des buyers réels"

---

## JOUR 1 — Matin
- [ ] Définir la cible de valuation (ex: $2-3M, $5-7M)
- [ ] Lister 3-5 acquéreurs prioritaires (par archétype)
- [ ] Créer un dossier partagé & template d'outreach

**Signal OK :** Vous avez les noms + valuation cible → pas de doute

---

## JOUR 1 — Après-midi
- [ ] Préparer **1-pager** non-confidential (ex: "Growth SaaS boîte, 15% MoM, $500k MRR")
- [ ] Préparer **3-slide teaser** (pas de chiffres, juste "qui c'est")
- [ ] Vérifier emails de vos acheteurs cibles

**Signal OK :** Vous pouvez envoyer le 1-pager en 10 sec, il convertit en "envoyez teaser"

---

## JOUR 2
- [ ] Lancer 3 emails de prise de contact (à 3 acheteurs type different)
- [ ] Préparer **CIM** (25-30 pages, valorisation + marché + traction)
- [ ] Audit technique : "Quels sont nos vrais problèmes de code/infra ?"

**Signal OK :** Vous avez 1 meeting en demande, CIM en cours

---

## JOUR 3
- [ ] Préparer **data room** : folder structure + liste des 50 documents clés
- [ ] Signer **NDA model** (template léger, pas 10 pages)
- [ ] Appeler vos top 3 clients pour "température" (pas demander, écouter)

**Signal OK :** 2+ meetings en semaine, CIM 60% done, clients positifs

---

## JOUR 4
- [ ] Préparer **valuation memo** : DCF + 3 scénarios (conservative/base/aggressive)
- [ ] Lister **red flags** que vous allez pré-emptively disclose
- [ ] Vérifier que vos financials sont normalisées (add-backs, EBITDA)

**Signal OK :** Vous connaissez votre valuation juste (±20%)

---

## JOUR 5
- [ ] Première réunion avec buyer #1 : pitch 30min, Q&A
- [ ] Envoyer post-meeting summary avec **next steps clairs**
- [ ] Update data room access (phased : doc intro → financials → tech → deets)

**Signal OK :** Buyer demande accès data room = progrès réel

---

## JOUR 6
- [ ] Documenter toutes les questions des buyers
- [ ] Préparer **Q&A master doc** (réponses pré-écrites, partagées)
- [ ] Préparer **competitive response** : si on vous lowball, voici le script

**Signal OK :** Q&A doc >20 questions, vous avez réponses percutantes

---

## JOUR 7
- [ ] Triage des leads : qui est sérieux ? (déjà dans data room, demande calls réguliers)
- [ ] Mettre à jour pipeline tracker : score chaque buyer
- [ ] Planifier semaine 2 : qui avancer, qui arrêter

**Signal OK :** Vous classez 2-3 buyers sérieux vs 3-5 "just kicking tires"

---

## Métrique de réussite semaine 1

✅ Vous visez ça :
- 2+ meetings sérieux en calendrier
- CIM finalisée
- Data room structure live
- Pipeline scoring en place

❌ Red flag :
- 0 meetings après jour 7 → réajuster targets ou messaging
- CIM rejetée (pas assez charismatique) → faire teaser pitch video
- Pas de data room → vous n'êtes pas ready

---

## Erreur courante à éviter
**"Attendre d'être parfait"** → data room 100% clean, CIM 50 révisions...
→ **Plutôt :** Lancez avec 70% prêt, améliorez en live

Buyers préfèrent un processus actif + quelques questions, vs attendre 2 mois pour "parfait".
