# 5 Profils d'Acquéreurs : Identification & Tactiques

## Profil 1 : **Predator** (Consolidateur rapide)

**Motivation** : Roll-up, économies d'échelle, domination de marché  
**Signaux** : Acquisitions >2/an, intégration rapide, marque de l'acquéreur conservée  
**Exemples** : HubSpot, Salesforce, Vista Equity, Insight Partners  

| Aspect | Caractéristique |
|--------|---|
| **Timeline** | 45-90 jours (strict) |
| **Pricing** | Multiples élevés (5-8x EBITDA), moins de négociation |
| **Due Diligence** | Rapide, standardisé |
| **Équipe fondateur** | 0-2 restent, intégration rapide |
| **Tactique** | Montrer traction, PMF, économies de frais |

**→ Approche** : Insister sur la traction, les synergies coûts, la churn basse. Pas de demandes spéciales.

---

## Profil 2 : **Strategic** (Croissance organique)

**Motivation** : Intégration verticale, customer base, IP/talent, réduction de risque  
**Signaux** : 1 acquisition/2-3 ans, long délai d'intégration, leadership peut rester  
**Exemples** : Zendesk (service), Okta (identity), Figma, Notion  

| Aspect | Caractéristique |
|--------|---|
| **Timeline** | 120-180 jours (flexible) |
| **Pricing** | Multiples moyens (3-5x EBITDA), synergies valorisées |
| **Due Diligence** | Approfondie (customer calls, tech audit) |
| **Équipe fondateur** | 2-4 restent souvent, leader de produit possible |
| **Tactique** | Montrer synergies produit, customer overlap, talent |

**→ Approche** : Préparer cas d'usage clients, roadmap partagée, vision long-terme. Prêt à garder indépendance opérationnelle.

---

## Profil 3 : **Financial Sponsor** (Buyout private equity)

**Motivation** : Rendement financier (20-30% IRR), optimalisation, revente  
**Signaux** : Décisions basées sur multiples et LBO, plateforme + add-ons  
**Exemples** : KKR, Thoma Bravo, Zenith Capital, Insight Partners  

| Aspect | Caractéristique |
|--------|---|
| **Timeline** | 120-150 jours (process strict) |
| **Pricing** | Multiples justes (3-4x EBITDA), financement par levier |
| **Due Diligence** | Très approfondie (financière surtout) |
| **Équipe fondateur** | Rôle réduit possible, CEO change souvent |
| **Tactique** | SaaS metrics parfaits, cash flow stable, peu d'ajustements |

**→ Approche** : Normaliser financements, montrer churn basse, NRR haut. Accepter réduction équipe de fondateurs.

---

## Profil 4 : **Acqui-Hire** (Talent + optionalité)

**Motivation** : Équipe talentueuse, réduire coûts de recrutement, optionalité produit  
**Signaux** : Prix bas (0.5-2x ARR), intégration rapide des gens, produit shutdown souvent  
**Exemples** : Google, Meta, Apple, startups en hypercroissance  

| Aspect | Caractéristique |
|--------|---|
| **Timeline** | 30-60 jours (rapide) |
| **Pricing** | Très bas (0.5-2x ARR), peu de multiples |
| **Due Diligence** | Tech + talent, pas financière |
| **Équipe fondateur** | Tous restent, roles opérationnels possibles |
| **Tactique** | Montrer talent, tech élégante, culture |

**→ Approche** : Valoriser l'équipe. Accepter prix bas. Négocier rôles et eq post-acq.

---

## Profil 5 : **Micro-Acquirer** (Petit joueur stratégique)

**Motivation** : Compléter offre, entrer nouveau segment, acqui-hire  
**Signaux** : 1ère ou 2e acquisition, timeline incertaine, due diligence légère  
**Exemples** : Startups Serie B/C acquérant des outils, boîtes classiques en pivot numérique  

| Aspect | Caractéristique |
|--------|---|
| **Timeline** | 60-120 jours (variable) |
| **Pricing** | Multiples bas-moyens (2-3x ARR), négociables |
| **Due Diligence** | Légère, plus de réunions commerciales |
| **Équipe fondateur** | Souvent restent, autonomie possible |
| **Tactique** | Vision partagée, cultural fit, growth synergies |

**→ Approche** : Construire relation long-terme, valoriser complémentarité. Patience exigée.

---

## Matrice d'Identification Rapide (2 minutes)

Posez ces 3 questions :

| Question | Réponse → Profil |
|----------|---|
| **Fréquence acq.** | > 2/an → Predator ; 1/2-3 ans → Strategic ; < 1/5 ans → Micro-Acquirer |
| **Type d'intégration** | Rapide + marque effacée → Predator ; Lente + leader reste → Strategic |
| **Structure financ.** | Financement levier → PE Sponsor ; Cash opérationnel → Strategic/Predator |

---

## Prochaines Actions

1. **Identifier 3-5 acquéreurs potentiels** dans votre catégorie
2. **Classer chacun** dans un profil (15 min)
3. **Adapter votre pitch** par profil :
   - **Predator** : Multiples, synergies coûts, traction
   - **Strategic** : Roadmap partagée, customer overlap
   - **PE** : Metrics SaaS, cash flow, peu de dettes
   - **Acqui-Hire** : Équipe, tech, culture
   - **Micro** : Vision + complémentarité

4. **Créer priorité d'outreach** : Predator/Strategic d'abord (plus rapide et $ plus haut)
