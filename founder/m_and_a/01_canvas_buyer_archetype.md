# Canvas Buyer Archetype — M&A

## **Archetype 1 : Strategic Acquirer (Secteur complémentaire)**
- **Motivation** : Croissance inorganique, cross-sell, éliminer concurrent
- **Signaux** : Portfolio d'acquisitions, R&D budget élevé, IPO récente
- **Timeline** : 3-6 mois (VDD complète + intégration stratégique)
- **Multiple moyen** : 5-8x ARR (premium sur synergies)
- **Risque** : Peut lever des objections opérationnelles sérieuses
- **Levier** : "Autres buyers à la table"

## **Archetype 2 : Financial Buyer (PE/VC)**
- **Motivation** : Rendement sur investissement (IRR 20-30%)
- **Signaux** : Multiple initial bas, carry-back structure (founder reste)
- **Timeline** : 2-4 mois (rapide mais rigoureux)
- **Multiple moyen** : 3-5x ARR (pure financier, pas synergies)
- **Risque** : Optimisation opérationnelle agressive post-vente
- **Levier** : Croissance + marges prouvées

## **Archetype 3 : Acquéreur Français dominant**
- **Motivation** : Consolidation marché local, économies d'échelle
- **Signaux** : Leader de marché, IPO Euronext, ≥€100M revenu
- **Timeline** : 4-8 mois (bureaucratie, comité de direction)
- **Multiple moyen** : 6-10x ARR (premium français + synergies)
- **Risque** : Culture clash, slow decision-making
- **Levier** : Unique positioning, cross-sell synergies

## **Archetype 4 : Roll-up Play**
- **Motivation** : Consolider 5-10 petits players → scale
- **Signaux** : Portfolio de marques, actif m&a, holding struc
- **Timeline** : 2-3 mois (procédure standardisée)
- **Multiple moyen** : 4-6x ARR (batch pricing, mais rapide)
- **Risque** : Dilution brand, perte de contrôle opérationnel
- **Levier** : Conformité templates, "other founders joined"

## **Archetype 5 : International Buyer**
- **Motivation** : Entrée géographique France, savoir-faire tech
- **Signaux** : Footprint global, IPO étrangère, >$1B valuation
- **Timeline** : 5-12 mois (cross-border, FDI scrutiny)
- **Multiple moyen** : 7-12x ARR (premium géographique)
- **Risque** : Compliance (RGPD, local content), retards
- **Levier** : Déjà établis en France, "add foreign buyers"
