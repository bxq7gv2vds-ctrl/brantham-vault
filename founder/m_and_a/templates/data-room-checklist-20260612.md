# Data Room Checklist — 50 Items Essentiels

**Utilité** : se préparer à l'accès acheteur; inclure/exclure intentionnellement.

---

## Financial (10)

- [ ] P&L statements (36 months, monthly + annual summary)
- [ ] Balance sheet + general ledger (36 months)
- [ ] Cash flow statement + working capital schedule
- [ ] Cap table with all rounds (with executed agreements)
- [ ] Options ledger (vesting schedules, strike prices, expiration dates)
- [ ] Revenue breakdown by customer/vertical/geography (36m)
- [ ] ARR/MRR cohort analysis + retention/churn
- [ ] Capitalized software schedule + depreciation policy
- [ ] Credit card/loan agreements + outstanding balances
- [ ] Tax returns (last 3 years + state/local filings)

## Legal (10)

- [ ] Articles of Incorporation & Bylaws
- [ ] Cap table shareholder agreements
- [ ] Board resolutions (M&A authorization, share issuances)
- [ ] Signed NDAs (samples for data room)
- [ ] Customer contracts (top 20 + master SLA)
- [ ] Vendor/supplier agreements (software licenses, hosting, SaaS)
- [ ] Employment agreements & offer letters
- [ ] Non-compete/non-solicitation clauses (employee handbook)
- [ ] IP assignment agreements (from founders, contractors, advisors)
- [ ] Litigation history (all pending + settled cases, threats)

## Intellectual Property (5)

- [ ] Patent prosecution files (US + international)
- [ ] Trademark registrations + pending applications
- [ ] Copyright registration certificates
- [ ] Open source audit (all licenses + list of OSS in product)
- [ ] Code repository access (audit of codebase + architecture)

## Operations (8)

- [ ] Customer list (names, ARR, contract end date, discount %)
- [ ] Employee roster (names, titles, salary, equity, start date)
- [ ] Headcount forecast + burn rate (12-month projection)
- [ ] Insurance policies (D&O, cyber, product liability)
- [ ] Security certifications (SOC2, ISO, HIPAA status)
- [ ] Data privacy audit (GDPR compliance, DPA status)
- [ ] Cloud/hosting costs + SLA agreements
- [ ] Vendor concentration (top 5 vendors, lock-in risk)

## Product & Technology (7)

- [ ] Product roadmap (current + 12-month forward)
- [ ] Tech debt assessment (architecture docs + known issues)
- [ ] Scalability stress test results (max capacity, current usage)
- [ ] Uptime/SLA data (last 12 months, incident log)
- [ ] Security audit reports (pentest, bug bounty, incident log)
- [ ] API documentation (if third-party integrations)
- [ ] Data model & database schema (anonymized or full access)

## Commercial (5)

- [ ] Sales pipeline (top 10 prospects, stage, likelihood)
- [ ] Sales compensation plan (commission %, quotas, attainment)
- [ ] Marketing spend breakdown (channels, CAC by source)
- [ ] Competitive positioning docs (win/loss analysis)
- [ ] Customer case studies (references, testimonials, logos)

## HR & Culture (5)

- [ ] Organizational chart (reporting, spans)
- [ ] Key person retention agreements (target list, incentives)
- [ ] 401k plan documents (vesting schedule, matching)
- [ ] Employee handbook & policies
- [ ] Diversity/pay equity snapshot

---

## Access Control & Logging

- [ ] All documents password-protected (access via VPN/portal)
- [ ] View logs enabled (track who views what, when)
- [ ] Question log (buyer queries → seller responses, tracked by date)
- [ ] Download restrictions (screenshots/copies forbidden; read-only)

---

## Intentional Exclusions (Do NOT Include)

- Founder personal financials (tax returns, net worth)
- Email chains with investors (privileged communications)
- Strategy docs about other M&A targets (conflicted)
- Internal debates on valuation ("we're worth $100M, bottom line $80M")
- Employee termination files or performance issues (privacy risk)
- Customer complaints or negative feedback (unless litigated)
- Board meeting minutes discussing sale strategy

---

## Timeline

- **T+5 days** : Data room live with financials + cap table
- **T+15 days** : Legal + operational sections added
- **T+30 days** : Product/tech deep dive materials ready
- **T+45 days** : All items indexed with search function

---

## Q&A Process SLA

- Buyer question submitted → Seller response within 48 hours
- Complex/sensitive questions → escalate to legal counsel
- Redacted answers OK if privilege/confidentiality at risk
