# M&A Quick Start — 2-Week Checklist

**Use** : First time thinking about a sale? Do these 15 things first (2 weeks, max 30h total).

---

## Week 1 (5 Things)

### Day 1-2: Audit & Self-Assessment

- [ ] **Tech Debt Scorecard** (10 min)
  - Run [[tech_debt_scorecard.md]] assessment
  - Identify top 3 fixes needed
  - Cost/timeline estimate

- [ ] **Customer Risk Assessment** (15 min)
  - Run [[customer_churn_risk_calculator.md]]
  - Which top 3 customers are at risk?
  - Any churn issues last 6 months?

- [ ] **Financial Snapshot** (10 min)
  - ARR / MRR current
  - Growth rate (last 12 months)
  - EBITDA % / unit economics
  - Keep 1-pager, update monthly

### Day 3-4: Advisors & Prep

- [ ] **Hire Tax Advisor** (2h)
  - Expert-comptable (France) or tax attorney (US)
  - Ask: "What's my tax impact under asset vs share deal?"
  - Budget: €2K-5K for initial advice

- [ ] **Draft Deal Story** (1h)
  - Use [[deal_story_template.md]]
  - 500-word narrative (problem → solution → ask)
  - Get feedback from 1 trusted advisor

### Day 5: Plan Week 2

- [ ] **List Top Buyers** (30 min)
  - Who would want to buy you? (10-20 candidates)
  - Strategic buyers > PE > Financial buyers
  - Save as spreadsheet for outreach

---

## Week 2 (10 Things)

### Day 6-7: Deeper Prep

- [ ] **Tech Debt Fix Priority** (1h)
  - Pick top 3 tech issues to fix pre-sale
  - Timeline: Can you fix in 4 weeks?
  - Cost/benefit for each

- [ ] **Cap Table Cleanup** (2h)
  - Is cap table clean? Any weird option grants?
  - Are all shares properly issued?
  - If messy, hire lawyer to clean up ($5K-10K)

- [ ] **Customer Outreach Plan** (1h)
  - Top 3 customers: Are they sticky?
  - Any renewal coming up? Lock them in NOW.
  - Any legal issues with key customers? Surface now.

### Day 8-9: Financial Deep-Dive

- [ ] **Normalize EBITDA** (2h)
  - What's your "true" profit if you remove one-time costs?
  - Salary adjustment (add back excess if you're underpaid)
  - One-time items (legal, audit, etc.)
  - Result: "SDE" (seller's discretionary earnings)

- [ ] **Draft Valuation Range** (1h)
  - Use [[tax_structure_decision_tree.md]] as guide
  - Multiples: 3-6x ARR typical (depends on growth, margin)
  - Formula: ARR × [multiple] = base valuation
  - Adjust for growth, churn, risk

- [ ] **Talk to Advisor/Lawyer** (1h)
  - Share financials + valuation thoughts
  - Get gut check: "Is this realistic for our market?"
  - Ask: "What's legal due diligence going to surface?"

### Day 10: Communication Plan

- [ ] **Prep Your Team** (30 min)
  - Who knows you might sell? (CEO, COO, key engineers?)
  - Prepare talking points ("We're exploring strategic options")
  - Avoid panic

- [ ] **Prepare Board/Investors** (if applicable) (30 min)
  - Who has say in sale decision?
  - Get buy-in before reaching out to buyers

- [ ] **Ready to Reach Out?** (30 min)
  - If YES → Move to outreach (next phase)
  - If NO → Fix 1-2 things first (tech debt, cap table, customer risk)
  - If UNSURE → Talk to advisor

## Red Flags (If Any of These Are True, PAUSE & FIX)

🚩 "Cap table is a mess (weird option grants, unclear ownership)"  
→ Hire lawyer, fix first (cost: €5-10K, time: 2-4 weeks)

🚩 "Top 3 customers = 60%+ ARR, all month-to-month"  
→ Lock in longer contracts NOW (before soft shop)

🚩 "Tech is unmaintainable (1 person knows codebase, no tests, legacy stack)"  
→ Start fixing NOW (4-8 weeks) or accept 20-30% price haircut

🚩 "We just had 20% churn this quarter"  
→ Investigate + fix BEFORE sale conversation (fix first, then sell)

🚩 "Liabilities or legal issues pending"  
→ Resolve or disclose EARLY (buyer will find out anyway)
