# Adjusted EBITDA Reconciliation Template

SaaS companies normalize EBITDA by adding back one-time/non-recurring expenses.

## Year-End 2025 Reconciliation

| Item | GAAP Reported | Adjustment | Adjusted |
|------|---------------|-----------|----------|
| **Revenue** | $[X]k | $0 | $[X]k |
| **COGS** | $[Y]k | - | $[Y]k |
| **Gross Profit** | $[Z]k | - | $[Z]k |
| **Operating Expenses** | | | |
| – R&D | $[A]k | $0 | $[A]k |
| – S&M | $[B]k | $0 | $[B]k |
| – G&A | $[C]k | + Add-backs | $(Adj)k |
| **EBITDA (reported)** | $[D]k | | |
| **Add-Backs** | | | |
| + Stock option expense (non-cash) | | $[E]k | |
| + M&A advisory fees (one-time) | | $[F]k | |
| + Relocation/severance (non-recurring) | | $[G]k | |
| + Office lease termination costs | | $[H]k | |
| + Accounting/audit fees (acquisition-specific) | | $[I]k | |
| + Depreciation (if capitalizing assets) | | $[J]k | |
| **Adjusted EBITDA** | | | $[Final]k |

**Interpretation:**
- **Adjusted EBITDA %** = Adjusted EBITDA / Revenue
- Should be 20–40% for growing SaaS

---

## Common Add-Backs (Ask Your Accountant)

| Category | Examples | Typical Amount |
|----------|----------|---|
| **Non-recurring compensation** | Sign-on bonus, severance, signing costs | 2–5% of revenue |
| **Non-recurring professional fees** | M&A advisory, legal, accounting | $10–50k per deal |
| **Stock-based compensation** | Employee options, founder grants | 3–8% of revenue |
| **Amortization of intangibles** | Software, customer lists (if acquired) | 1–3% |
| **Gain/loss on asset sale** | If you sold equipment, IP | 0–2% |
| **Facility costs (if closing office)** | Lease termination, renovation | $10–100k one-time |

---

## Red Flags (Buyer's Perspective)

🚩 **Add-backs > 20% of reported EBITDA** — Buyer will reject most; use conservative add-backs only.

🚩 **Recurring items labeled "non-recurring"** — Buyer's accountant will audit and remove them.

🚩 **Stock-based comp add-back > 10% revenue** — Suggests over-issuance; buyer may counter-argue sustainability.

---

## Prep for Buyer Challenge

**Buyer's team will ask:**
- "Why is [Specific expense] one-time?" → Have documentation (contract, email saying it won't repeat)
- "Isn't [Expense] recurring?" → Show historical trend (if it didn't happen in 2024, it's non-recurring)
- "Can we verify this with your accountant?" → Provide CPA sign-off on add-backs

**Best practice:** Get your accountant to sign off on add-backs before you send to buyer. Buyer trusts independent CPA more than founder.
