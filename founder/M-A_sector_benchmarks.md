# Sector Benchmarks — P/E, EV/EBITDA, Margins

## SaaS (High-Growth)
```
Revenue CAGR:      25-40%
EV/Revenue:        6-12x (growth premium)
EV/EBITDA:         20-35x (pre-profit many)
EBITDA Margin:     10-20%
Rule of 40:        Growth% + Margin% ≥ 40
```
**Comparables:** Salesforce (7.5x rev), Datadog (11x), Stripe (private, est. 10x)

---

## E-Commerce / Retail
```
Revenue Growth:    5-15%
EV/Revenue:        0.5-2x (margin-dependent)
EV/EBITDA:         8-15x
Gross Margin:      30-50%
EBITDA Margin:     5-12%
```
**Comparables:** Shopify (2.2x rev), Best Buy (0.4x)

---

## Manufacturing / B2B Services
```
Revenue Growth:    3-8%
EV/Revenue:        0.8-2x
EV/EBITDA:         7-12x
Gross Margin:      25-40%
EBITDA Margin:     8-15%
```
**Comparables:** Xylem ($50B cap, 2.5x rev)

---

## Healthcare / Pharma
```
Revenue Growth:    4-10%
EV/Revenue:        3-8x (regulatory premium)
EV/EBITDA:         12-20x
R&D % Revenue:     15-25%
```
**Comparables:** J&J (3x rev), small biotech (8-15x)

## Quick Valuation Screening

**Para tu target:**
1. Lookup 5-10 peers públicos (search "{sector} public companies")
2. Extrae multiples: Bloomberg, CapitalIQ, Yahoo Finance
3. Median EV/EBITDA sector → tu valuation
4. Si target está 30%+ bajo median → bargain o red flag

**Ejemplo:**
- Sector median EV/EBITDA: 10x
- Tu target EBITDA: $10M
- "Fair" valuation: $100M
- Si precio asked >$120M (12x) → puede overpay
