# Valuation 1-Pager — DCF & Multiples

## Input Assumptions
```
LTM Revenue:           $50M
LTM EBITDA:            $10M (20%)
Revenue Growth (3yr):  15% CAGR
Terminal Growth:       3% (GDP)
WACC:                  8%
Net Debt:              -$2M (cash position)
```

**Enterprise Value:** $52.8M + $246M = **$298.8M**  
**+ Cash:** -$2M  
**Equity Value:** **~$300M**
