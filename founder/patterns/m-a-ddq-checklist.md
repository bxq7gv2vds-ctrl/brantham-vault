# Due Diligence (DDQ) Checklist M&A

## Financial Due Diligence

### 📊 Financial Statements & Records
- [ ] **3 ans de comptes audités** (KPMG/Ernst/etc.)
- [ ] **Reconciliation bancaire** (comptes courants, d'épargne, crédits)
- [ ] **Déclaration de TVA** (vérifier conformité)
- [ ] **Factures impayées** (>90 jours)
- [ ] **Créances douteuses** (provision calculée)
- [ ] **Stocks** (méthode de valorisation)
- [ ] **Actifs immobilisés** (amortissements, dépréciation)
- [ ] **Dettes fiscales et sociales** (DAS, TVA, impôts)

### 📈 Revenue & Customer Analysis
- [ ] **Contrats clients** (annexes, pénalités de rupture)
- [ ] **Top 10 clients** (>30% du CA ?)
- [ ] **Taux de rétention client** (historique + projection)
- [ ] **Prix de vente** (historique, politiques)
- [ ] **Churn rate** (détaillé par segment)
- [ ] **LTV/CAC ratio** (cohérent avec secteur)

### 💰 Cost Structure & Profitability
- [ ] **Break-even analysis** (point mort mensuel)
- [ ] **Coûts fixes/variables** (détaillés)
- [ ] **Marge brute** (par produit/service)
- [ ] **EBITDA margin** (tendance historique)
- [ ] **Capex/OPEX ratio** (investissement vs opérationnel)
- [ ] **Synergy targets** (réalistes ?)

### 🏦 Banking & Financing
- [ ] **Crédits bancaires** (conditions, garanties)
- [ ] **Covenants financiers** (breaches ?)
- [ ] **Cash flow** (12 mois rolling forecast)
- [ ] **Working capital** (cycle de trésorerie)
- [ ] **Debt schedule** (remboursements, taux)
- [ ] **Credit lines** (utilisation disponible)

## Legal Due Diligence

### 📋 Corporate Structure
- [ ] **Statuts** (version actuelle + toutes versions)
- [ ] **Organigramme juridique** (filiales, holdings)
- [ ] **Capital social** (historique des augmentations)
- [ ] **Actionnariat** (transferts récents)
- [ ] **Prêts aux actionnaires** (montants, conditions)
- [ ] **Pactes d'actionnaires** (clauses spéciales)

### ⚖️ Contracts & Litigation
- [ ] **Clients contracts** (durée, prix, KPIs)
- [ ] **Fournisseurs contracts** (longueur, clauses)
- [ ] **Leases immobiliers** (durée, option de renouvellement)
- [ ] **Licenses & IP** (validité, paiement)
- [ ] **Litigation en cours** (détails + probabilité de perte)
- [ ] **Contentieux sociaux** (prud'hommes, URSSAF)
- [ ] **Regulatory compliance** (non-conformités)

### 🏢 Employment & HR
- [ ] **Contrats de travail** (CDI/CDD, durée moyenne)
- [ ] **Convention collective** (applicable, respectée)
- [ ] **CSE** (fonctionnement, relations)
- [ ] **Pénalités de départ** (clés de départ)
- [ ] **Réserve spéciale** (calculée, utilisée)
- [ ] **Phased retirement** (mises en place)

### 📊 IP & Technology
- [ ] **Brevets** (détenteurs, validation, procès)
- [ ] **Marques** (enregistrement, renouvellement)
- [ ] **Copyright** (logiciels, contenus)
- [ ] **Data privacy** (RGPD, CNIL, consentement)
- [ ] **Cybersecurity** (audit, incidents, certifications)
- [ ] **Open source** (compliance, licenses)

## Operational Due Diligence

### 🏭 Facilities & Equipment
- [ ] **Titres de propriété** (bâtiments, terrains)
- [ ] **Baux commerciaux** (durée, loyers, charges)
- [ ] **Certifications** (ISO, qualité, sécurité)
- [ ] **Maintenance contracts** (équipements critiques)
- [ ] **Insurance coverage** (risques couverts)
- [ ] **Environmental compliance** (normes, permis)

### 🚚 Supply Chain & Logistics
- [ ] **Key suppliers** (concentration, dépendance)
- [ ] **Supply chain risks** (geopolitical, climate)
- [ ] **Inventory management** (turnover, obsolescence)
- [ ] **Logistics operations** (outsourced vs in-house)
- [ ] **Transport contracts** (coûts, performance)
- [ ] **Customs compliance** (imports/exports)

### 👥 Customer & Market
- [ ] **Customer concentration** (>20% sur un client ?)
- [ ] **Customer satisfaction** (NPS, réclamations)
- [ ] **Market position** (share, growth rate)
- [ ] **Competitive landscape** (barrières à l'entrée)
- [ ] **Regulatory changes** (impact opérationnel)
- [ ] **Technology disruption** (risques d'obsolescence)

## Environmental & Governance (ESG)

### 🌿 Environmental
- [ ] **Carbon footprint** (scope 1, 2, 3)
- [ ] **Environmental permits** (validité, conformité)
- [ ] **Waste management** (déchets dangereux, recyclage)
- [ ] **Energy efficiency** (consommation, certifications)
- [ ] **Climate risks** (physical, transition)
- [ ] **ESG reporting** (transparence, standards)

### 👥 Governance
- [ ] **Board composition** (indépendance, diversité)
- [ ] **Ethics policy** (compliance, whistleblowing)
- [ ] **Anti-corruption** (FCPA, UK Bribery Act)
- [ ] **Data governance** (privacy, security)
- [ ] **Stakeholder engagement** (investisseurs, communautés)
- [ ] **ESG integration** (risques, opportunités)

## Tax Due Diligence

### 💰 Tax Position
- [ ] **Tax compliance** (déclarations, paiements)
- [ ] **Tax authorities** (litiges, audits)
- [ ] **Transfer pricing** (documentation, compliance)
- [ ] **VAT optimization** (schemes, risques)
- [ ] **Tax incentives** (crédits d'impôt, subventions)
- [ ] **International taxation** (double taxation, treaties)

### 📊 Tax Liabilities
- [ ] **Tax contingencies** (litiges en cours)
- [ ] **Tax loss carryforwards** (utilisation, expirations)
- [ ] **Sales tax/VAT** (obligations, déclarations)
- [ ] **Payroll taxes** (cotisations, déclarations)
- [ ] **Property taxes** (valuations, paiements)
- [ ] **Exit taxes** (impacts de l'acquisition)

## Integration Planning

### 🎯 Synergy Assessment
- [ ] **Revenue synergies** (cross-sell, upsell)
- [ ] **Cost synergies** (overhead, procurement)
- [ ] **Technology integration** (compatibilité, coûts)
- [ ] **Operational efficiencies** (processes, people)
- [ ] **Timeline realization** (quand/combien)
- [ ] **Risks to synergies** (dépendances, résistances)

### 👥 People Integration
- [ ] **Headcount analysis** (doublons, compétences)
- [ ] **Compensation alignment** (parity, equity)
- [ ] **Cultural integration** (valeurs, communication)
- [ ] **Key retention** (risques, plans)
- [ ] **Integration timeline** (phases, milestones)
- [ ] **Change management** (résistance, communication)

### 📋 Legal Entity Integration
- [ ] **Legal structure** (consolidation, fusion)
- [ ] **Contract transfers** (novation, substitution)
- [ ] **IT systems** (migration, data)
- [ ] **Brand transition** (timeline, communication)
- [ ] **Regulatory approvals** (nécessaires, timeline)
- [ ] **Exit planning** (scénarios de sortie)

## Red Flags & Critical Issues

### 🔴 High Risk Items
- [ ] **Liquidity issues** (cash runway < 6 mois)
- [ ] **Litigation material** (>€1M de risque)
- [ ] **Regulatory violations** (sanctions possibles)
- [ ] **Customer concentration** (>50% sur un client)
- [ ] **IP infringement** (risque de procès)
- [ ] **Key dependency** (personne, fournisseur, technologie)

### 🟡 Medium Risk Items
- [ ] **Margin decline** (tendances négatives)
- [ ] **High customer churn** (>10% par mois)
- [ ] **Underfunded pensions** (gap >10% de fonds)
- [ ] **Compliance gaps** (non-conformités mineures)
- [ ] **Technology debt** (maintenance coûteuse)
- [ ] **Geographic concentration** (risque de marché)

## Due Diligence Timeline

### Phase 1: Initial Review (Week 1-2)
- [ ] Document collection
- [ ] Financial review
- [ ] Legal review
- [ ] High-level risks

### Phase 2: Deep Dive (Week 3-4)
- [ ] Customer interviews
- [ ] Site visits
- [ ] Technical assessment
- [ ] Market analysis

### Phase 3: Verification (Week 5-6)
- [ ] Third-party confirmations
- [ ] Financial modeling
- [ ] Integration planning
- [ ] Final report
