# Pre-Closing Checklist — 100 Items (D75-D90)

**When:** Deploy 21 days before target close date  
**Owner:** Founder + CFO + General Counsel  
**Cadence:** Daily standup (15 min); weekly board review  
**Goal:** Zero surprises on closing day

---

## SECTION 1: LEGAL DOCUMENTS (15 items)

- [ ] **SPA signed by all parties** (no pending edits)
- [ ] **Exhibits A-G complete** (cap table, IP, contracts, leases)
- [ ] **Representations & warranties schedule** (agreed list of reps, warranties, exceptions)
- [ ] **Indemnification cap/basket** (agreed: typically 0.5–1% of purchase price)
- [ ] **Working capital adjustment mechanism** (formula agreed, target WC confirmed)
- [ ] **Closing conditions cleared** (all conditions precedent resolved or waived)
- [ ] **Earn-out schedule signed** (if applicable; KPI targets, measurement, payment dates)
- [ ] **Non-compete/non-solicit executed** (founder + key employees)
- [ ] **Employment agreements (key hires)** (new buyer terms offered, accepted)
- [ ] **IP assignment confirmations** (all IP transferred to buyer post-close)
- [ ] **Material contracts list** (all identified, consents obtained)
- [ ] **Escrow arrangement details** (account, trustee, holdback amount, release schedule)
- [ ] **Closing statement prepared** (signed by buyer + seller CFO, reconciled)
- [ ] **Deed(s) of assignment** (if real estate, personal property, etc.)
- [ ] **Mortgage/security agreements satisfied** (lender payoff letters obtained)

---

## SECTION 2: FINANCIAL & TAX (15 items)

- [ ] **Tax clearance certificates obtained** (federal, state, local)
- [ ] **Final working capital statement prepared** (audited/reviewed by accountant)
- [ ] **Payroll taxes paid through close date** (no pending liabilities)
- [ ] **Sales tax returns filed/paid** (current through close)
- [ ] **Property tax assessed (if real estate)** (no outstanding liens)
- [ ] **Accounts receivable aged** (bad debt reserve calculated, agreed with buyer)
- [ ] **Inventory write-down (if applicable)** (obsolete items cleared)
- [ ] **Accrued expenses finalized** (all known liabilities recorded)
- [ ] **Bonus/commission calculations done** (employee bonuses, contractor fees)
- [ ] **Founder personal expenses paid** (no outstanding reimbursements)
- [ ] **Loan/debt schedules updated** (all lenders provided payoff letters)
- [ ] **Bank account statements final** (through close date)
- [ ] **Intercompany transactions cleared** (if multi-entity, all reconciled)
- [ ] **Deferred revenue analyzed** (timing, buyer responsibilities post-close)
- [ ] **Final tax provision calculated** (federal, state, local, sales, payroll)

---

## SECTION 3: CONTRACTS & CUSTOMERS (12 items)

- [ ] **Customer contract consents obtained** (top 20 accounts, if required)
- [ ] **Customer list provided to buyer** (updated through close date; pricing, contract terms)
- [ ] **Vendor contracts consents obtained** (critical vendors; IT, hosting, legal, accountant)
- [ ] **Lease assignment/consent obtained** (landlord approval for space, if required)
- [ ] **Key employee retention agreements signed** (bonus locked in post-close)
- [ ] **Affiliate/partnership contracts reviewed** (no material changes post-close)
- [ ] **License agreements updated** (if buyer name changes, update all software licenses)
- [ ] **Insurance policies notified** (D&O, liability; notify of transaction if required)
- [ ] **Service level agreements (SLAs) updated** (buyer to assume, any SLA changes agreed)
- [ ] **Third-party IP licenses assigned** (no restrictions on buyer use)
- [ ] **Material adverse change (MAC) review** (no undisclosed material changes since LOI)
- [ ] **Supplier/vendor transition plan** (what changes post-close; communication plan)

---

## SECTION 4: INTELLECTUAL PROPERTY (8 items)

- [ ] **Copyright registrations listed** (all proprietary code, content, designs)
- [ ] **Patent portfolio documented** (issued + pending; assignment agreements ready)
- [ ] **Trademark registrations current** (federal, state, international; renewal dates tracked)
- [ ] **Domain names listed & transferred** (registrar access provided to buyer)
- [ ] **Source code escrow (if applicable)** (if buyer wants code backup in escrow)
- [ ] **Third-party IP audit complete** (no infringement claims identified)
- [ ] **Employee IP assignments collected** (all employees signed IP assignment agreements)
- [ ] **Open-source license audit done** (GPL, MIT, Apache, etc.; no GPL virality issues)

---

## SECTION 5: CORPORATE GOVERNANCE (10 items)

- [ ] **Board resolutions authorizing transaction** (final board meeting minutes signed)
- [ ] **Shareholder approval obtained** (if required; board + stakeholders vote)
- [ ] **Cap table finalized** (all shares, options, warrants, SAFEs accounted for)
- [ ] **Stock ledger updated** (all shareholders listed, share counts verified)
- [ ] **Option pool exercises processed** (any last-minute exercises completed)
- [ ] **Warrant/SAFE conversions handled** (all converted to common before close)
- [ ] **Board observer rights terminated** (if any investors have board seats, clarify post-close role)
- [ ] **Founder vesting cliff cleared (if applicable)** (any time-based vesting completed)
- [ ] **Stock option plan amendment filed** (if buyer requires changes post-close)
- [ ] **Certificate of good standing obtained** (current corporate status)

---

## SECTION 6: HUMAN RESOURCES (12 items)

- [ ] **Employee census list provided** (names, titles, salaries, start dates, benefits)
- [ ] **All employment agreements current** (no pending updates, offer letters for new hires)
- [ ] **Benefits continuation plan communicated** (health insurance, 401k, PTO post-close)
- [ ] **Severance packages (if any) negotiated** (severance for non-retained employees identified)
- [ ] **Key person insurance (if applicable)** (policies assigned/cancelled per buyer intent)
- [ ] **Work permits/visa status confirmed** (for international employees, if any)
- [ ] **Background check confirmations on file** (all employees passed, or noted exceptions)
- [ ] **Employee handbook reviewed** (buyer to adopt or merge with their policy)
- [ ] **Payroll provider transition plan** (buyer's payroll system, timing of cutover)
- [ ] **Equity documents gathered** (all option agreements, restricted stock, SAFEs)
- [ ] **WARN Act notice (if applicable)** (if laying off >50 employees, 60-day notice)
- [ ] **Employee communications drafted** (announcement template, Q&A, retention message)

---

## SECTION 7: DATA SECURITY & COMPLIANCE (10 items)

- [ ] **Data protection audit completed** (GDPR, CCPA, SOC 2, if required)
- [ ] **Security incident history disclosed** (any past breaches, to buyer)
- [ ] **Customer data inventory mapped** (PII handled, encryption status, backup recovery)
- [ ] **Compliance certifications current** (SOC 2, ISO, FedRAMP, HL7, etc., if required)
- [ ] **Audit findings addressed** (no open compliance gaps)
- [ ] **Vendor security assessments done** (critical vendors' security posture confirmed)
- [ ] **Data residency requirements met** (GDPR, data sovereignty, regulatory location requirements)
- [ ] **Insurance/cyber liability current** (coverage adequate through close; transfer to buyer post-close)
- [ ] **Disaster recovery plan documented** (RTO/RPO, tested, buyer to assume)
- [ ] **Privacy policy & terms of service updated** (reflect new owner, if required)

---

## SECTION 8: BANK & CASH (8 items)

- [ ] **Wire instructions confirmed** (seller bank account, routing, SWIFT; triple-checked)
- [ ] **Escrow account set up** (third-party trustee, amount, release schedule)
- [ ] **Closing statement reconciled** (buyer + seller agree on final purchase price)
- [ ] **Holdback/retain amount calculated** (typically 10–20%; security for reps/warranties)
- [ ] **Tax escrow (if applicable)** (separate amount for future tax liabilities)
- [ ] **Earnout funding mechanism confirmed** (buyer's obligation to pay earnout, timeline)
- [ ] **Seller financing (if any)** (note terms, subordination, security)
- [ ] **Bank account transition plan** (when accounts close, new buyer accounts open)

---

## SECTION 9: OPERATIONAL READINESS (10 items)

- [ ] **IT systems access documentation** (buyer gets admin credentials, access to all systems)
- [ ] **Password manager migrated** (1Password, LastPass, or equivalent; all credentials shared)
- [ ] **Cloud infrastructure transferred** (AWS, GCP, Azure ownership/billing transferred)
- [ ] **Backups verified** (last backup tested, restore verified with buyer)
- [ ] **DNS/domain delegation ready** (buyer can take over immediately post-close)
- [ ] **Vendor account access provided** (GitHub, Stripe, Twilio, SendGrid, etc.)
- [ ] **Documentation compiled** (all technical docs in one place for buyer)
- [ ] **Customer onboarding documents updated** (reflect new owner, new support contact)
- [ ] **Runbook/handoff notes prepared** (how to operate the business day 1)
- [ ] **Status page/uptime monitoring set up** (buyer inherits monitoring, alerting)

---

## SECTION 10: FINAL WALKTHROUGH (10 items)

- [ ] **SPA schedule review** (all schedules reviewed line-by-line, no discrepancies)
- [ ] **Closing date confirmed** (all parties agree on final closing date/time)
- [ ] **Closing location & logistics** (in-person, remote, Zoom? Notary arrangements?)
- [ ] **Document signing order confirmed** (which documents signed first, sequence?)
- [ ] **Wire timing confirmed** (when exactly does buyer send wire?)
- [ ] **Document execution ceremony** (who signs, who witnesses, notarization needed?)
- [ ] **Hardcopy originals prepared** (if wet signatures required, documents printed, ready)
- [ ] **Post-close announcement timing** (press release timing, customer comms, employee all-hands)
- [ ] **Founder's personal files cleared** (laptop, phone data wiped if required)
- [ ] **Office keys/access badges returned** (physical items, if applicable)

---

## SECTION 11: STAKEHOLDER COMMUNICATIONS (8 items)

- [ ] **Advisor notification (lawyer, accountant)** (confirm final terms, tax implications)
- [ ] **Key customers notified (confidentially)** (if required before public announcement)
- [ ] **Key vendors notified (confidentially)** (if business continuity critical)
- [ ] **Board/investor update sent** (final terms, close date, next steps)
- [ ] **Employee communication ready** (drafted, approved, timed for post-close)
- [ ] **Press release drafted & approved** (buyer quote, quote from founder, embargo timing)
- [ ] **Social media post (optional)** (founder LinkedIn announcement)
- [ ] **Customer success team briefing** (how to handle customer questions, messaging)

---

## SECTION 12: POST-CLOSE TRANSITION (6 items)

- [ ] **Day 1 integration plan** (first 24 hours; all-hands meeting, comms, access setup)
- [ ] **Week 1 roadmap confirmed** (what gets done in first week with buyer)
- [ ] **Month 1 goals set** (key wins to deliver in first 30 days)
- [ ] **30-60-90 day integration plan** (longer roadmap, milestone dates)
- [ ] **Earnout KPI dashboard live** (weekly tracking begins)
- [ ] **CEO/founder transition role confirmed** (title, reporting line, responsibilities)

---
