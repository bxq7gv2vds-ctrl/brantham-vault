# 100-Day Integration Playbook

## Days 0-5: Announcement & Stabilization

- [ ] Joint press release / announcement call
- [ ] Message to customers (no surprises, continuity promised)
- [ ] Message to employees (vision, job security for 6 mo retroactively from close)
- [ ] Steering committee weekly kickoff
- [ ] Checklists for IP assignment, bank account transitions
- [ ] Assign integration lead (PMO)

---

## Days 6-30: Foundation

### Financial
- [ ] Consolidate accounting (GL structure, chart of accounts)
- [ ] Reconcile pre-close records
- [ ] Establish weekly revenue dashboard

### People
- [ ] Confirm all key person agreements in place
- [ ] Align comp/bonus/benefits (any gaps)
- [ ] Create org chart (new + acquired)
- [ ] Kickoff culture integration workshops

### Operations
- [ ] Map critical processes (sales, order-to-cash, payroll)
- [ ] Identify system cutover plan (Salesforce, ERP, etc.)
- [ ] Data migration readiness (test, backup)

### Commercial
- [ ] Customer success calls (top 20 accounts)
- [ ] Win/loss interviews with non-renewed accounts
- [ ] Cross-sell opportunity assessment

---

## Days 31-60: Execution

### IT/Systems
- [ ] Parallel run systems (old + new) where applicable
- [ ] Data migration pilot (test environment first)
- [ ] Security audit completion
- [ ] Full data cutover (if not phased)

### Synergies
- [ ] Cost-reduction quick-wins (e.g., vendor consolidation)
- [ ] Revenue cross-sell pilot (5-10 accounts)
- [ ] Shared services transition planning (HR, finance, IT)

### Integration Risks
- [ ] Attrition monitoring weekly
- [ ] Customer churn analysis (root cause)
- [ ] Escalate any red flags to steering committee

---

## Days 61-100: Stabilization & Optimization

### Steady State
- [ ] All systems live & stable
- [ ] New org structure in place
- [ ] Financial reporting consolidated
- [ ] Synergy burn-down tracking

### Optimization
- [ ] Performance tuning (cost structure)
- [ ] Upsell/cross-sell at scale
- [ ] Product roadmap alignment
- [ ] Brand consolidation decisions

### Handoff
- [ ] Integration lead → business as usual
- [ ] Document lessons learned
- [ ] Steering committee → quarterly reviews

---

## Key Deliverables By Milestone

| Day | Deliverable | Owner |
|-----|---|---|
| 5 | Comms package (external + internal) | CEO/IR |
| 30 | Process mapping complete | COO |
| 45 | System migration plan finalized | CTO |
| 60 | Synergy YTD realization report | CFO |
| 100 | Integration close-out report | PMO |
