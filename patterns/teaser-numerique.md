# Teaser Template - Secteur Digital

Pour les deals **services numériques, SaaS, digital marketing** dans les procédures collectives.

## Structure teaser

**HEADER**
Acquisition - Agence digitale [Spécialisation]
[Ville] - CA [X] k€ - Portefeuille clients stabilisé

**CORPS**
✅ **Actif numérique** : [base clients] clients récurrents avec LTV [X]€
✅ **Recurring revenue** : [X]% de revenus récurrents (contrats [durée] mois)
✅ **Équipe technique** : [nombre] développeurs full-stack
✅ **Stack technologique** : [technologies] - scalabilité prouvée
✅ **Urgence procédurale** : Due diligence technique avant [date]

**SYNTHESE**
- Modèle : [SaaS/Agence/Marketplace]
- Taux de rétention : [X]%
- CAARR : [CA] k€ (croissance [X]%)
- EBITDA : [X] k€ ([X]%)
- Prix : [prix] k€ (multiple [X]x CA)
- Contact : AJ [Nom AJ] - [email]

**CALL TO ACTION**
Investisseurs tech ou repreneurs sectorielles souhaitent analyse portefeuille clients.

**LEGAL**
*Confidentiel - Données clients RGPD protégées*
*Source : Annonces AJ, vérification BODACC*

---

## Variables à remplir

- [Ville] : Hub digital/remote
- [Spécialisation] : développement, marketing, SaaS
- [base clients] : nombre + type de clients
- [X]€ : LTV moyen par client
- [X]% : % revenus récurrents
- [durée] : durée moyenne contrats
- [technologies] : stack technique (React, Python, etc.)
- [date] : deadline due diligence
- [prix] : multiple CA ou revenus récurrents
- [Nom AJ] : Administrateur judiciaire

## Notes d'utilisation

- Mettre en avant la rétention clients
- Préciser la stack technique actuelle
- Souligner la scalabilité du modèle
- Adapter aux acquéreurs tech
