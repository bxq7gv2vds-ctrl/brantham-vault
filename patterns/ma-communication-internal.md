# Framework de Communication Interne M&A

## 🎯 Stratégie de Communication par Phase

### **Phase 1: Sélection & Approche (Jours 1-30)**
```markdown
**Objectif**: Maintenir la stabilité et productivité existante
**Audience Concernée**: Senior team uniquement
**Cadence**: Weekly updates

**Messages Clés**:
- "Nous explorons des opportunités stratégiques"
- "Aucun impact immédiat sur vos rôles"
- "Focus sur nos objectifs 2024"

**Communication Channels**:
- 📧 Email aux execs: 1x/semaine
- 🤝 Réunions individuelles: HR discussions
- 📊 Dashboard interne: Status updates

**Red Flags**: [ ] Employee questions [ ] Customer rumors [ ] Market speculation
```

### **Phase 2: Due Diligence Active (Jours 31-60)**
```markdown
**Objectif**: Préparer le terrain pour l'annonce
**Audience Concernée**: Department heads + key contributors
**Cadence**: Bi-weekly updates

**Messages Clés**:
- "Nous avançons dans notre évaluation"
- "Focus sur l'intégration future"
- "Rien à signaler pour le moment"

**Communication Channels**:
- 📋 Town hall departmental: Bi-weekly
- 📧 Briefing aux managers: Weekly
- 🎯 Objectifs d'équipe: Alignés avec l'objectif

**Points à surveiller**: [ ] Questions tactiques [ ] Anxiété des équipes [ ] Rumeurs externes
```

### **Phase 3: Annonce & Transition (Jours 61-90)**
```markdown
**Objectif**: Communiquer clairement la décision et les prochaines étapes
**Audience Concernée**: Toute l'organisation
**Cadence**: Daily updates initiales, then weekly

**Messages Clés**:
- "Nous avons finalisé l'acquisition de [Company]"
- "Vision intégrée: [Objectif commun]"
- "Timeline d'intégration: 90 jours"

**Communication Channels**:
- 🎥 Town hall全员: Day 1
- 📲 App mobile push notifications: Critical updates
- 💬 Slack/Discord channels: Daily Q&A
- 📅 Calendrier partagé: Milestones

**Échéances Clés**:
- Day 1: Annonce formelle + vision
- Day 3: Team meetings
- Day 7: Integration planning sessions
- Day 30: Progress review
```

### **Phase 4: Intégration (Jours 91-180)**
```markdown
**Objectif**: Maintenir l'engagement et la productivité
**Audience Concernée**: Toutes les équipes
**Cadence**: Weekly updates, daily critical updates

**Messages Clés**:
- "Progrès d'intégration: [Scorecard]"
- "Réalisations: [Quick wins]"
- "Challenges et solutions: [Transparency]"

**Communication Channels**:
- 📊 Dashboard d'intégration: Daily metrics
- 📝 Newsletter mensuelle: Success stories
- 🎥 Leadership updates: Bi-weekly video
- 💼 Sessions de feedback: Weekly

**Focus**: [ ] Process integration [ ] Culture alignment [ ] Customer communication
```

### **Phase 5: Stabilisation (Jours 181-365)**
```markdown
**Objectif**: Normaliser les opérations et anticiper le futur
**Audience Concernée**: Toute l'organisation
**Cadence**: Bi-weekly updates

**Messages Clés**:
- "Résultats intégré: [Performance vs targets]"
- "Nouvelle stratégie: [2025 objectives]"
- "Personnes et rôles: [Clarity]"

**Communication Channels**:
- 📈 Reporting trimestriel: Performance metrics
- 🎯 Town hall战略: Vision 2025
- 📋 Organigramme mis à jour: Structure claire
- 💡 Innovation sessions: Future planning
```

## 🎭 Stake-Specific Messaging Framework

### **Executive Team**
```markdown
**Frequency**: Daily during critical phases
**Format**: Executive briefings + one-on-one sessions
**Key Topics**:
- Financial integration status
- Strategic alignment
- Leadership decisions
- Crisis management protocols

**Messaging Style**: Direct, data-driven, solution-focused
**Success Metrics**: [ ] Clear decisions [ ] Unified leadership [ ] Strategic alignment
```

### **Middle Management**
```markdown
**Frequency**: Bi-weekly team briefings + weekly updates
**Format**: Departmental meetings + written briefings
**Key Topics**:
- Integration progress
- Resource allocation
- Team structure
- Customer impacts

**Messaging Style**: Practical, actionable, transparent
**Success Metrics**: [ ] Team stability [ ] Clear reporting lines [ ] Process continuity
```

### **Individual Contributors**
```markdown
**Frequency**: Weekly updates + as-needed Q&A
**Format**: Town halls + department meetings + internal comms
**Key Topics**:
- Role clarity
- Day-to-day operations
- Career development
- Culture updates

**Messaging Style**: Reassuring, empathetic, inclusive
**Success Metrics**: [ ] Low turnover [ ] High engagement [ ] Productivity maintained
```

### **External Stakeholders (Customers/Partners)**
```markdown
**Frequency**: Quarterly updates + crisis comms
**Format**: Account manager updates + public announcements
**Key Topics**:
- Service continuity
- Product roadmap
- Contact information
- Value proposition

**Messaging Style**: Confident, reassuring, benefit-focused
**Success Metrics**: [ ] Retention rate [ ] Customer satisfaction [ ] Renewal rates
```

## 📝 Templates de Communication

### **Email d'Annonce (Phase 3)**
```markdown
**Subject**: Important Update: Strategic Acquisition Announcement

Dear Team,

I'm writing to share important news: we have completed the acquisition of [Company Name].

**What this means:**
- Combined vision: [Shared strategic objective]
- Immediate impact: No changes to your role, reporting, or daily work
- Timeline: Integration will proceed over the next 90 days
- Next steps: Team meetings scheduled for [Date]

**Our commitment:**
- Transparent communication throughout the process
- Respect for both organizations' cultures and values
- Focus on delivering exceptional value to our customers
- Career development opportunities for all team members

Questions? Please reach out to your manager or HR Business Partner.

Best regards,
[Name]
CEO
```

## 🚨 Crisis Communication Protocols

### **Red Scenarios (Activate immediately)**
```markdown
**Scenario 1: Key Resignations**
- Immediate: 1:1 with departing employees
- 24h: Internal announcement of resignations
- 48h: Communication plan for replacements
- Weekly: Progress updates on hiring

**Scenario 2: Customer Backlash**
- Immediate: Briefing for customer-facing teams
- 24h: Customer communication template
- 72h: Proactive outreach to key clients
- Ongoing: Daily customer satisfaction tracking

**Scenario 3: Regulatory Issues**
- Immediate: Legal briefing for leadership
- 24h: Internal awareness communication
- 48h: External compliance statement
- Weekly: Regulatory updates
```

### **Communication Rules of Engagement**
```markdown
**Golden Rules:**
1. **No speculation**: Only confirmed facts
2. **Unified message**: Single source of truth
3. **Transparency**: What we know, what we don't
4. **Empathy**: Acknowledge concerns
5. **Consistency**: Same message across all channels

**Crisis Protocol:**
- **Level 1**: Daily comms team sync
- **Level 2**: Executive crisis meetings
- **Level 3**: External PR activation
- **Level 4**: Legal team engagement
```
