# Script Prospection - Fonds d'Investissement

**Cible** : Fonds d'investissement et family offices  
**Objectif** : Présenter pipeline deals distressed avec opportunités d'investissement

## Message LinkedIn

**Sujet** : Pipeline M&A distressed - Opportunités d'investissement

**Corps du message**Madame, Monsieur,

Chez Brantham Partners, nous développons un pipeline de deals M&A distressed ciblés, avec une spécialisation sur les PME en difficulté (CA 500k€ - 5M€).

**Notre valeur ajoutée** :
✅ Analyse précoce via notre scoring AI
✅ Processus d'acquisition accéléré (2-3 mois vs 6-12 mois)  
✅ Focus sur les secteurs à forte valeur : [secteur 1], [secteur 2], [secteur 3]
✅ Deal flow qualifié : [X] deals actifs avec scoring moyen de [Y]

**Opportunités actuelles** :
- Deal premium : scoring [Z] - Secteur [secteur] - CA [CA] k€
- Portefeuille : [nombre] deals sectoriels avec [X] k€ de valeur totale
- Focus géographique : [régions prioritaires]

**Pour collaborer avec nous** :
1. **Early access** aux deals scoring [80+] avant publication
2. **Due diligence** supportée par notre analyse sectorielle  
3. **Closing pipeline** accéléré grâce à notre réseau repreneurs

**Notre méthodologie** :
- Scoring prédictif multi-critères (financier, procédural, sectoriel)
- Intelligence BODACC + Pappers intégrée
- Processus légal optimisé pour les procédures collectives

**Prochaines étapes** :
1. Webinar mensuel présentation pipeline (15 min)
2. Accès deal room avec deals scoring [70+]  
3. Co-investissement sélectionné

Intéressé par une discussion next week ? Disponible [créneau] ?

Cordialement,
[Votre nom]
Brantham Partners - M&A AI Pipeline
