# Teaser Template - Secteur Industrie

Pour les deals **industriels, maintenance, distribution** dans les procédures collectives.

## Structure teaser

**HEADER**
Acquisition - Société industrielle [Spécialisation]
[Ville] - CA [X] k€ - Potentiel de rationalisation

**CORPS**
✅ **Actifs stratégiques** : [équipements/clients/fournisseurs clés]
✅ **Chiffre d'affaires** : [CA] k€ HT avec marge de [X]%
✅ **Potentiel de croissance** : Marché [marché cible] en croissance
✅ **Forte valeur ajoutée** : [compétence technique] propriétaire
✅ **Urgence procédurale** : Présélection offres avant [date]

**SYNTHESE**
- Secteur : [industrie/sous-traitance/maintenance]
- Effectif : [nombre] techniciens/ouvriers
- Investissements : [X] k€ (équipements)
- Prix : [prix] k€ (EBITDA [X]x)
- Contact : AJ [Nom AJ] - [email]

**CALL TO ACTION**
Repreneurs industriels ou fonds spécialisés souhaitent analyse technique avant [date].

**LEGAL**
*Confidentiel - Industrielle propriété intellectuelle*
*Source : Annonces AJ, vérification BODACC*

---

## Variables à remplir

- [Ville] : Localisation exacte + région
- [Spécialisation] : maintenance, distribution, sous-traitance
- [CA] : Chiffre d'affaires avec détail clientèle
- [X]% : Marge brute/nette
- [marché cible] : marché adressable
- [compétence technique] : savoir-faire unique
- [date] : deadline offers
- [prix] : multiple d'EBITDA
- [Nom AJ] : Administrateur judiciaire

## Notes d'utilisation

- Mettre en avant les actifs physiques
- Souligner la base clients récurrente
- Préciser les compétences techniques uniques
- Adapter aux investisseurs industriels
