# Teaser Template - Secteur Commerce

Pour les deals de **fonds de commerce, retail, supermarchés** dans les procédudes collectives.

## Structure teaser

**HEADER**
Opportunité d'acquisition - Fonds de commerce [Secteur]
[Ville] - CA estimé [X] k€ - Potentiel de restructuration

**CORPS**
✅ **Opportunité unique** : Fonds de commerce établi en [Ville], positionnement [positionnement]
✅ **Chiffre d'affaires** : [CA] k€ HT (données vérifiées BODACC)
✅ **Potentiel de restructuration** : Synergies avec réseaux existants
✅ **Localisation stratégique** : [détails emplacement]
✅ **Urgence procédurale** : Deadline [date] pour présentation offres

**SYNTHESE**
- Secteur : [commerce détail/SPécialisé/Gros]
- Surface : [surface] m²
- Effectif : [nombre] salariés
- Prix : [prix] k€ (à débattre)
- Contact : AJ [Nom AJ] - [email]

**CALL TO ACTION**
Repreneurs/investisseurs avec expérience sectorielle souhaitent présentation détaille avant le [date].

**LEGAL**
*Confidentiel - Ne pas diffuser sans autorisation*
*Source : Annonces AJ, vérification BODACC*

---

## Variables à remplir

- [Ville] : Localisation exacte
- [Secteur] : détail du commerce (boulangerie, joaillerie, supermarché)
- [CA] : Chiffre d'affaires annuel estimé
- [positionnement] : position sur le marché
- [date] : deadline procédure
- [surface] : surface commerciale
- [prix] : fourchette indicative
- [Nom AJ] : Administrateur judiciaire

## Notes d'utilisation

- Adapter le ton selon le type de commerce (grand public vs B2B)
- Mettre en avant la localisation pour les retail
- Insister sur le modèle économique existant
- Préciser si bail commercial inclus
