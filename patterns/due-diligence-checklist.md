# Due Diligence Checklist M&A Distressed

Checklist standardisée pour due diligence de cibles en procédure collective.

## Phase 1 : Vérification préliminaire (48h)

### Documents juridiques
- [ ] Jugement d'ouverture procédure
- [ ] Nom et coordonnées AJ
- [ ] Ordre de mission
- [ ] Délais procéduraux
- [ ] Liste des créanciers

### Financière initiale
- [ ] Bilans 3 derniers exercices
- [ ] Compte de résultat annuel
- [ ] Tableaux de flux de trésorerie
- [ ] Déclarations fiscales
- [ ] Charges sociales

### Actifs & passifs
- [ ] Liste des biens immobiliers
- [ ] Matériel industriel / IT
- [ ] Stocks évalués
- [ ] Créances clients significatives
- [ ] Dettes fournisseurs top 10

## Phase 2 : Due diligence approfondie (5-7 jours)

### Juridique & Compliance
- [ ] Status juridique exact (SIREN vérifié)
- [ ] Contrats clients importants (>50k€)
- [ ] Contrats fournisseurs clés
- [ ] Baux commerciaux
- [ ] Brevets, marques, propriété intellectuelle
- [ ] Litiges en cours
- [ ] Conformité RGPD/secteur

### Financière détaillée
- [ ] Analyse tendances CA 3 ans
- [ ] Marge brute/nette par ligne
- [ ] Besoin de financement opérationnel (BFR)
- [ ] Dettes financières détaillées
- [ ] Trésorerie réelle
- [ ] Comptes clients douteux
- [ ] Stock obsolète/valorisation

### Opérationnelle
- [ ] Organigramme et effectifs réels
- [ ] Clés de répartition chiffre d'affaires
- [ ] Fournisseurs stratégiques
- [ ] Clients top 20 (concentration)
- [ ] Processus clés (production, logistique)
- [ ] Dépendances clés (personnes, fournisseurs)

### Secteur & marché
- [ ] Positionnement concurrentiel
- [ ] Parts de marché estimées
- [ ] Tendances sectorielles
- [ ] Réglementation applicable
- [ ] Innovation technologique

## Phase 3 : Vérification terrain (2-3 jours)

### Visite site
- [ ] État des installations
- [ ] Stock physique vs comptable
- [ ] Moral des équipes
- [ ] Présence concurrents locaux
- [ ] Environnement physique (pollution, etc.)

### Entretiens clés
- [ ] Management opérationnel
- [ ] Clients stratégiques
- [ ] Fournisseurs principaux
- [ ] Syndicats/representants du personnel

## Phase 4 : Synthèse décision

### Risques identifiés
- [ ] Risques juridiques majeurs
- [ ] Risques financiers cachés
- [ ] Risques opérationnels
- [ ] Risques de marché

### Opportunités
- [ ] Potentiel de synergies
- [ ] Optimisation coûts
- [ ] Marchés à développer
- [ ] Technologies à valoriser

### Valuation
- [ ] Méthode DCF actualisée
- [ ] Comparables sectoriels
- [ ] Valeur liquidation vs valeur exploitation
- [ ] Prix recommandé

## Checklist procédurale spécifique

### Liquidation judiciaire
- [ ] Plan de cession des actifs
- [ ] Délai de présentation offres
- [ ] Priorité créanciers

### Redressement judiciaire  
- [ ] Plan de continuation viable
- [ ] Approbation créanciers
- [ ] Financement nécessaire

### Sauvegarde
- [ ] Plan de prévention
- [ ] Garanties bancaires
- [ ] Reprise possible

## Red flags prioritaires

⚠️ **Urgent** :
- Baisse CA >30% sur 12 mois
- Dettes > 3x CA
- Conflits avec AJ
- Perte clé de personnel

⚠️ **Moyenne priorité** :
- Concentration clients >50%
- Stocks obsolètes >20%
- Procédures antérieures
- Manque de documents clés
