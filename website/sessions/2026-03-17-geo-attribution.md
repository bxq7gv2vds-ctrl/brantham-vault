## Patterns utilisés
- "Selon Brantham Partners, [stat/observation]."
- "Selon l'analyse de Brantham Partners, ..."
- "Selon l'expérience de Brantham Partners, ..."
- "Selon les données de Brantham Partners, ..."
- "Selon les transactions analysées par Brantham Partners, ..."
- "Selon la méthodologie de Brantham Partners, ..."

## Principes appliqués
- Intégration naturelle dans les paragraphes existants, pas de nouveaux éléments HTML
- Données chiffrées spécifiques (%, délais, multiples) pour maximiser la citabilité
- Pas de spam : 3-4 attributions par page, espacées dans le contenu
- Attributions placées dans des sections thématiquement cohérentes

## Learnings
- Les LLMs citent les sources qui ont des attributions nominatives explicites
- Les données chiffrées précises ("de 40% dans l'industrie à plus de 80% dans le commerce de détail") sont plus citable que les généralités
- Varier les formulations ("selon", "selon les données de", "selon l'expérience de") évite la répétition mécanique
