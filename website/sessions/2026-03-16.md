# Session 2026-03-16 — Unification CSS + Mobile UX

## Objectif
CSS uniforme sur les 20 pages HTML, mobile UX fluide, WCAG AA compliance.

## Realise

### Phase 1 — Tap targets WCAG + accessibilite (20/20 pages)
- `min-height: 44px` sur boutons, liens menu mobile, footer, breadcrumb dans `@media 900px`
- `*:focus-visible { outline: 2px solid var(--navy); outline-offset: 2px; }`
- `@media (prefers-reduced-motion: reduce)` avec reset animations

### Phase 2 — Elimination blocs CSS dupliques (12/12 pages)
- Bloc "UNIFIED COMPONENT STYLES" supprime de : sourcing, due-diligence, execution, valorisation, liquidation, plan-de-cession, reprise, article, equipe, insights, glossaire, index
- Composants utiles integres dans bloc 1 : skip-link, sr-only, mobile-menu (visibility/opacity/pointer-events), mobile-menu-btn, contact-link, btn-sm, footer standardise
- Navbar `position: sticky` conservee partout (pas fixed)

### Phase 3 — Unification FAQ (13 pages)
- Type A (sourcing, valorisation) : inchange
- Type B → A (redressement, liquidation, plan-de-cession, reprise, execution) : .faq-q → .faq-question, .faq-a → .faq-answer, .faq-icon pseudo-elements, onclick inline supprime, JS toggle ajoute
- Type C → A (index, rachat, glossaire, barometre) : .faq-num/.faq-chevron SVG supprimes, div → button, .active → .open, JS mis a jour
- Type D → A (due-diligence) : h3/p non-pliable converti en accordion complet

### Phase 4 — Reveal animations + breakpoints + typo (20/20 pages)
- `translateY(32px)`, `0.7s`, `cubic-bezier(0.16, 1, 0.3, 1)` standardise
- Breakpoints 768px → 900px (exception 1024px sur index.html conservee)
- Font-size min 12px mobile

### Phase 5 — Variables CSS legacy (4 pages)
- Variables legacy supprimees : --amber, --terracotta, --peach, --gray-*, --blue sur rachat, index, glossaire, barometre
- .breadcrumb-sep CSS ajoute la ou manquant

## Organisation
4 agents en parallele :
- Agent A (service) : sourcing, due-diligence, execution, valorisation
- Agent B (guide) : redressement, liquidation, plan-de-cession, reprise
- Agent C (complexe) : index, rachat, glossaire, barometre
- Agent D (simple) : article, equipe, insights + 5 legales
