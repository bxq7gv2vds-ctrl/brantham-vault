# Session 13 mars 2026 — Mega Audit SEO + AI Visibility + UX + Content Creation

## Contexte
Objectif : faire de brantham.fr LE site de reference pour le rachat d'entreprise en difficulte en France. Apparaitre en #1 sur Google ET etre recommande par toutes les AI (Claude, ChatGPT, Perplexity, Gemini).

## Fixes UX pre-audit
- Title alignment "Notre approche" (section 05) : `align-items: flex-end` → `flex-start` en mobile
- Hero proportions : multiples iterations → `align-content: start`, `min-height: 0`, illustration 520x540
- Article descriptions : ajout `.article-desc` sous chaque titre d'article

## Phase 1 — Audits (3 agents paralleles)

### T01 — SEO Technique (score: 68/100)
- CRITICAL : 20+ liens footer vers des pages 404
- HIGH : WebSite + BreadcrumbList schemas manquants, canonical mismatch article.html, #apropos lien mort, Google Fonts render-blocking, zero images alt text, mentions legales incompletes
- MEDIUM : title 78 chars (>60), meta desc 185 chars (>155), stats sans H2, robots.txt minimal

### T02 — AI Visibility (score: 1.5/5)
- Organization schema minimaliste (manque knowsAbout, hasOfferCatalog)
- Zero contenu attribuable ("Brantham Partners + verbe")
- Pas de llms.txt
- Pas de tableaux HTML (AI parse pas les CSS grids)
- Pas de glossaire, pas de barometre, pas d'expert nomme
- Plan d'action 20 points sur 4 semaines

### T03 — UX/Performance (score: 78/100)
- HIGH : contraste couleurs insuffisant (--t3, --t4 FAIL WCAG AA)
- MEDIUM : scroll bar anime width (layout), 2 scroll listeners sans rAF, mobile menu display:none non-animable, pas de skip link, FAQ non-focusable clavier, will-change absent, backdrop-filter mobile couteux

## Phase 2 — Implementation fixes (3 agents paralleles)

### T04 — SEO fixes (11 modifs)
- Title 78→55 chars, meta desc 185→148 chars
- 15 liens 404 footer → ancres internes
- #apropos → #contact (navbar + mobile)
- Schemas WebSite + BreadcrumbList ajoutes
- H2 sr-only section stats
- robots.txt : Disallow /swarm/ et sections-proposals.html
- Google Fonts async (onload pattern) + DM Mono 3→2 poids

### T05 — AI Visibility (6 modifs)
- llms.txt cree a la racine
- Organization schema enrichi (knowsAbout, hasOfferCatalog, foundingDate, areaServed, numberOfEmployees)
- HowTo schema ajoute (7 etapes rachat en LJ)
- Table HTML sr-only avec stats (parsable AI)
- "Nous structurons" → "Brantham Partners structure"
- 4 nouvelles FAQ (10 total) dans HTML + JSON-LD FAQPage

### T06 — UX/Performance (11 modifs)
- Contraste : --t3 #8A8A86→#6A6A66, --t4 #B8B8B4→#8A8A86, --gray-400→#6A6A66, --gray-300→#9A9A96
- Scroll progress bar : width → transform:scaleX() + will-change
- 2 scroll listeners fusionnes en 1 avec rAF throttle
- Mobile menu : display:none → visibility/pointer-events (transition smooth)
- Skip link ajoute apres <body>
- FAQ : role="button", tabindex="0", aria-expanded, keydown Enter/Space
- will-change sur .marquee-track et .deal-float--*
- contain: layout style sur .marquee-container
- backdrop-filter: none sur mobile 900px
- barGrow keyframes : height → scaleY()
- Boutons min-height 44px sur 480px
- Doublon .methode-float-n supprime
